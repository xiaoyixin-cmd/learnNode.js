# 5.3 垃圾回收算法：Scavenge（新生代）+ Mark-Sweep-Compact（老生代）

> 深入理解 V8 的分代垃圾回收机制

---

## 📋 核心概念

V8 使用**分代垃圾回收**策略，针对不同年龄的对象使用不同的 GC 算法。

**核心问题：**
- V8 如何管理内存？
- 新生代和老生代有什么区别？
- Scavenge 算法如何工作？
- Mark-Sweep-Compact 是什么？

**分代假说：**
```
1. 大部分对象生命周期很短
2. 存活时间长的对象会继续存活
```

## 💻 Scavenge 算法（新生代）

### 1. Cheney 算法

**原理：** 将新生代分为两个相等的半空间（Semi-Space）。

```
┌─────────────────────────────────────┐
│         New Space (2-8 MB)          │
├─────────────────┬───────────────────┤
│   From Space    │    To Space       │
│   (活动)        │    (空闲)         │
└─────────────────┴───────────────────┘

GC 时：
1. 遍历 From Space 的存活对象
2. 复制到 To Space
3. 清空 From Space
4. 交换 From 和 To
```

---

### 2. Scavenger 源码

```cpp
// deps/v8/src/heap/scavenger.cc

/**
 * Scavenger（新生代 GC）
 */
class Scavenger {
 public:
  void ScavengeObject(HeapObject* object, HeapObject** target) {
    // 🔥 检查对象是否在新生代
    if (!InNewSpace(object)) {
      return;
    }

    // 🔥 检查是否已复制（转发地址）
    MapWord first_word = object->map_word();
    if (first_word.IsForwardingAddress()) {
      // 已复制，返回新地址
      *target = first_word.ToForwardingAddress();
      return;
    }

    // 🔥 复制对象到 To Space
    int object_size = object->Size();
    AllocationResult allocation = to_space_->AllocateRaw(object_size);
    
    HeapObject* target_object = allocation.ToObjectChecked();

    // 🔥 内存复制
    CopyBlock(target_object->address(), object->address(), object_size);

    // 🔥 设置转发地址
    object->set_map_word(MapWord::FromForwardingAddress(target_object));

    *target = target_object;

    // 🔥 更新引用
    MigrateObject(target_object, object_size);
  }

  void MigrateObject(HeapObject* dst, int size) {
    // 🔥 遍历对象的指针字段
    Map* map = dst->map();
    
    BodyDescriptor::IterateBody(
        map,
        dst,
        size,
        [this](HeapObject* host, ObjectSlot slot) {
          Object* object = *slot;
          if (object->IsHeapObject()) {
            HeapObject* heap_object = HeapObject::cast(object);
            // 🔥 递归复制引用的对象
            ScavengeObject(heap_object, slot);
          }
        });
  }
};
```

### 4. 晋升（Promotion）

```cpp
// deps/v8/src/heap/scavenger.cc

/**
 * 对象晋升到老生代
 */
void Scavenger::PromoteObject(HeapObject* object) {
  // 🔥 晋升条件
  // 1. 对象经历过一次 Scavenge（Age >= 1）
  // 2. To Space 使用率 > 25%

  int object_size = object->Size();

  // 🔥 在老生代分配空间
  AllocationResult allocation = old_space_->AllocateRaw(object_size);
  
  HeapObject* target = allocation.ToObjectChecked();

  // 🔥 复制对象
  CopyBlock(target->address(), object->address(), object_size);

  // 🔥 设置转发地址
  object->set_map_word(MapWord::FromForwardingAddress(target));

  // 🔥 记录晋升
  heap_->IncrementPromotedObjectsSize(object_size);
}
```

**晋升策略：**
```
对象年龄 = 经历的 Scavenge 次数

Age 0: 新对象
Age 1: 经历 1 次 Scavenge → 晋升候选
Age >= 1 或 To Space > 25% → 晋升到老生代
```

## 💻 Mark-Sweep-Compact 算法（老生代）

### 1. 三个阶段

```
阶段1：标记（Mark）
  遍历对象图，标记存活对象

阶段2：清除（Sweep）
  回收未标记的对象

阶段3：整理（Compact）
  压缩内存，减少碎片
```

---

### 2. 标记阶段（Mark）

```cpp
// deps/v8/src/heap/mark-compact.cc

/**
 * 标记存活对象
 */
class MarkCompactCollector {
 public:
  void MarkLiveObjects() {
    // 🔥 标记根对象
    MarkRoots();

    // 🔥 处理标记队列（深度优先）
    ProcessMarkingDeque();
  }

  void MarkRoots() {
    // 🔥 标记全局对象
    MarkObject(isolate_->global_object());

    // 🔥 标记栈上的对象
    StackVisitor stack_visitor(this);
    isolate_->Iterate(&stack_visitor);

    // 🔥 标记句柄中的对象
    isolate_->global_handles()->IterateStrongRoots(&stack_visitor);
  }

  void MarkObject(HeapObject* object) {
    // 🔥 检查是否已标记
    if (marking_state_->IsBlack(object)) {
      return;
    }

    // 🔥 标记为黑色
    marking_state_->WhiteToBlack(object);

    // 🔥 添加到标记队列
    marking_deque_.Push(object);
  }

  void ProcessMarkingDeque() {
    while (!marking_deque_.IsEmpty()) {
      HeapObject* object = marking_deque_.Pop();

      // 🔥 遍历对象的引用
      MarkingVisitor visitor(this);
      object->Iterate(&visitor);
    }
  }
};

/**
 * 标记访问器
 */
class MarkingVisitor {
 public:
  void VisitPointers(HeapObject* host, ObjectSlot start, ObjectSlot end) {
    for (ObjectSlot slot = start; slot < end; ++slot) {
      Object* object = *slot;
      if (object->IsHeapObject()) {
        // 🔥 标记引用的对象
        collector_->MarkObject(HeapObject::cast(object));
      }
    }
  }
};
```

### 3. 清除阶段（Sweep）

```cpp
// deps/v8/src/heap/mark-compact.cc

/**
 * 清除未标记的对象
 */
void MarkCompactCollector::Sweep() {
  // 🔥 遍历所有页
  PagedSpaceIterator spaces(heap());
  
  for (PagedSpace* space = spaces.Next(); space != nullptr; space = spaces.Next()) {
    // 🔥 清除页中的死对象
    SweepSpace(space);
  }
}

void MarkCompactCollector::SweepSpace(PagedSpace* space) {
  for (Page* page : *space) {
    // 🔥 遍历页中的对象
    Address current = page->area_start();
    Address limit = page->area_end();

    while (current < limit) {
      HeapObject* object = HeapObject::FromAddress(current);

      if (marking_state_->IsBlack(object)) {
        // 🔥 存活对象：清除标记
        marking_state_->BlackToWhite(object);
        current += object->Size();
      } else {
        // 🔥 死对象：添加到空闲列表
        int size = object->Size();
        space->Free(current, size);
        current += size;
      }
    }
  }
}
```

### 4. 整理阶段（Compact）

```cpp
// deps/v8/src/heap/mark-compact.cc

/**
 * 压缩内存（减少碎片）
 */
void MarkCompactCollector::EvacuatePages() {
  // 🔥 选择碎片化严重的页
  std::vector<Page*> evacuation_candidates;
  
  for (Page* page : *old_space_) {
    if (ShouldEvacuate(page)) {
      evacuation_candidates.push_back(page);
    }
  }

  // 🔥 疏散对象到其他页
  for (Page* page : evacuation_candidates) {
    EvacuatePage(page);
  }
}

void MarkCompactCollector::EvacuatePage(Page* page) {
  // 🔥 遍历页中的存活对象
  Address current = page->area_start();
  Address limit = page->area_end();

  while (current < limit) {
    HeapObject* object = HeapObject::FromAddress(current);

    if (marking_state_->IsBlack(object)) {
      // 🔥 移动对象到新位置
      int size = object->Size();
      AllocationResult allocation = old_space_->AllocateRaw(size);
      
      HeapObject* target = allocation.ToObjectChecked();

      // 🔥 复制对象
      CopyBlock(target->address(), current, size);

      // 🔥 设置转发地址
      object->set_map_word(MapWord::FromForwardingAddress(target));

      current += size;
    } else {
      current += object->Size();
    }
  }

  // 🔥 更新所有引用
  UpdatePointersAfterEvacuation();

  // 🔥 释放页
  old_space_->ReleasePage(page);
}
```

**特点：**

- 不参与 Scavenge
- 不进行 Compact（不移动）
- 只进行 Mark-Sweep

### 2. 对象池

```javascript
class ObjectPool {
  constructor(factory, size = 100) {
    this.factory = factory;
    this.pool = [];
    
    // 预分配对象
    for (let i = 0; i < size; i++) {
      this.pool.push(factory());
    }
  }

  acquire() {
    return this.pool.pop() || this.factory();
  }

  release(obj) {
    // 重置对象状态
    if (obj.reset) obj.reset();
    this.pool.push(obj);
  }
}

// 使用
const pool = new ObjectPool(() => ({ x: 0, y: 0 }));

function process() {
  const point = pool.acquire();  // 🔥 复用对象
  point.x = 10;
  point.y = 20;
  // 处理 point
  pool.release(point);  // 🔥 归还
}
```

## 📝 总结

### 核心要点

1. **分代 GC**：新生代（Scavenge）+ 老生代（Mark-Sweep-Compact）
2. **Scavenge**：复制算法，快速回收短命对象
3. **Mark-Sweep**：标记清除，回收老年对象
4. **Compact**：内存整理，减少碎片

### 实现关键

1. **新生代**：From/To Space，Cheney 算法
2. **晋升**：Age >= 1 或 To Space > 25%
3. **老生代**：Mark（标记）→ Sweep（清除）→ Compact（整理）
4. **大对象**：独立空间，不移动，只标记清除

### 应用价值

1. **性能优化**：减少临时对象，使用对象池
2. **内存管理**：避免泄漏，监控 GC
3. **调试技巧**：使用 --trace-gc 分析
4. **架构设计**：考虑对象生命周期
