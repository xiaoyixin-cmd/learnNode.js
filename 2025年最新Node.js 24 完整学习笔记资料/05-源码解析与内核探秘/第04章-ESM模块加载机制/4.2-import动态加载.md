# 4.2 import() 动态加载源码路径：ModuleWrap → ModuleJob

> 深入理解动态导入的完整执行流程与V8协作机制

---

## 📋 核心概念

`import()` 是 ESM 的**动态导入**表达式，返回 Promise，允许按需加载模块。

**核心问题：**
- `import()` 如何在V8中实现？
- ModuleWrap 与 ModuleJob 的关系？
- 动态导入的缓存机制？
- import.meta 如何生成？

**执行流程：**
```
import('module')
  ↓
V8: HostImportModuleDynamically
  ↓
Node.js: ModuleWrap::ImportModuleDynamically (C++)
  ↓
ESMLoader.import() (JavaScript)
  ↓
ModuleJob.run()
  ↓
V8: Module.evaluate()
```

## 💻 V8 层实现

### 1. HostImportModuleDynamically 回调

```cpp
// src/node_contextify.cc
v8::MaybeLocal<v8::Promise> HostImportModuleDynamically(
    v8::Local<v8::Context> context,
    v8::Local<v8::Data> host_defined_options,
    v8::Local<v8::Value> resource_name,
    v8::Local<v8::String> specifier,
    v8::Local<v8::FixedArray> import_assertions) {
  
  Environment* env = Environment::GetCurrent(context);
  v8::Isolate* isolate = env->isolate();

  // 🔥 创建 Promise
  v8::Local<v8::Promise::Resolver> resolver;
  if (!v8::Promise::Resolver::New(context).ToLocal(&resolver)) {
    return v8::MaybeLocal<v8::Promise>();
  }

  // 🔥 获取 referrer (调用方 URL)
  v8::Local<v8::String> referrer;
  if (!resource_name->ToString(context).ToLocal(&referrer)) {
    return v8::MaybeLocal<v8::Promise>();
  }

  // 🔥 转换为 Node.js 字符串
  node::Utf8Value specifier_utf8(isolate, specifier);
  node::Utf8Value referrer_utf8(isolate, referrer);

  // 🔥 调用 ModuleWrap::ImportModuleDynamically
  ModuleWrap::ImportModuleDynamically(
    env,
    resolver,
    *specifier_utf8,
    *referrer_utf8,
    import_assertions
  );

  return resolver->GetPromise();
}
```

## 💻 Node.js C++ 层

### 1. ModuleWrap 类

```cpp
// src/module_wrap.cc
class ModuleWrap : public BaseObject {
 public:
  // 🔥 动态导入入口
  static void ImportModuleDynamically(
      Environment* env,
      Local<Promise::Resolver> resolver,
      const char* specifier,
      const char* referrer,
      Local<FixedArray> import_assertions) {
    
    Isolate* isolate = env->isolate();
    HandleScope handle_scope(isolate);

    // 🔥 创建 import_attributes 对象
    Local<Object> attributes = Object::New(isolate);
    
    for (int i = 0; i < import_assertions->Length(); i += 3) {
      Local<String> key = import_assertions->Get(context, i).As<String>();
      Local<String> value = import_assertions->Get(context, i + 1).As<String>();
      attributes->Set(context, key, value).Check();
    }

    // 🔥 调用 JavaScript 层
    Local<Function> import_callback = env->host_import_module_dynamically_callback();
    
    Local<Value> args[] = {
      String::NewFromUtf8(isolate, specifier).ToLocalChecked(),
      String::NewFromUtf8(isolate, referrer).ToLocalChecked(),
      attributes
    };

    // 🔥 调用 ESMLoader.import()
    Local<Value> result;
    if (import_callback->Call(context, Undefined(isolate), 3, args).ToLocal(&result)) {
      // 🔥 result 是 Promise
      Local<Promise> promise = result.As<Promise>();
      
      // 🔥 链式 Promise
      promise->Then(context, 
        // onFulfilled
        FunctionTemplate::New(isolate, [](const FunctionCallbackInfo<Value>& args) {
          Local<Promise::Resolver> resolver = args.Data().As<Promise::Resolver>();
          resolver->Resolve(args.GetIsolate()->GetCurrentContext(), args[0]).Check();
        }, resolver)->GetFunction(context).ToLocalChecked(),
        
        // onRejected
        FunctionTemplate::New(isolate, [](const FunctionCallbackInfo<Value>& args) {
          Local<Promise::Resolver> resolver = args.Data().As<Promise::Resolver>();
          resolver->Reject(args.GetIsolate()->GetCurrentContext(), args[0]).Check();
        }, resolver)->GetFunction(context).ToLocalChecked()
      );
    }
  }

  // 🔥 模块实例化
  static MaybeLocal<Module> Instantiate(
      Environment* env,
      Local<String> source,
      Local<String> url,
      Local<Object> import_meta_callback) {
    
    Isolate* isolate = env->isolate();
    Local<Context> context = env->context();

    // 🔥 编译模块源码
    ScriptOrigin origin(
      url,
      0,                      // line offset
      0,                      // column offset
      false,                  // is shared cross-origin
      -1,                     // script id
      Local<Value>(),         // source map URL
      false,                  // is opaque
      false,                  // is WASM
      true,                   // is module
      import_meta_callback    // host-defined options
    );

    ScriptCompiler::Source script_source(source, origin);

    Local<Module> module;
    if (!ScriptCompiler::CompileModule(isolate, &script_source).ToLocal(&module)) {
      return MaybeLocal<Module>();
    }

    return module;
  }

 private:
  Local<Module> module_;
  bool linked_ = false;
};
```

### 2. import.meta 对象生成

```cpp
// src/module_wrap.cc
void ModuleWrap::HostInitializeImportMetaObjectCallback(
    Local<Context> context,
    Local<Module> module,
    Local<Object> meta) {
  
  Environment* env = Environment::GetCurrent(context);
  Isolate* isolate = env->isolate();

  // 🔥 获取模块 URL
  ModuleWrap* wrap = GetFromModule(env, module);
  if (wrap == nullptr) return;

  Local<String> url = wrap->url();

  // 🔥 设置 import.meta.url
  meta->CreateDataProperty(
    context,
    env->url_string(),
    url
  ).Check();

  // 🔥 设置 import.meta.resolve (实验性)
  if (env->options()->experimental_import_meta_resolve) {
    Local<Function> resolve_fn = FunctionTemplate::New(
      isolate,
      ImportMetaResolve,
      url
    )->GetFunction(context).ToLocalChecked();

    meta->CreateDataProperty(
      context,
      env->resolve_string(),
      resolve_fn
    ).Check();
  }

  // 🔥 设置 import.meta.dirname / import.meta.filename (Node.js v20.6+)
  if (url->StringEquals(OneByteString(isolate, "file://"))) {
    URL file_url(url);
    
    meta->CreateDataProperty(
      context,
      env->dirname_string(),
      String::NewFromUtf8(isolate, file_url.path().c_str()).ToLocalChecked()
    ).Check();

    meta->CreateDataProperty(
      context,
      env->filename_string(),
      url
    ).Check();
  }
}

// 🔥 import.meta.resolve() 实现
void ModuleWrap::ImportMetaResolve(const FunctionCallbackInfo<Value>& args) {
  Environment* env = Environment::GetCurrent(args);
  Isolate* isolate = env->isolate();

  // 🔥 获取 specifier
  if (args.Length() < 1 || !args[0]->IsString()) {
    return args.GetReturnValue().Set(Undefined(isolate));
  }

  Local<String> specifier = args[0].As<String>();
  Local<String> parent_url = args.Data().As<String>();

  // 🔥 调用 ESMLoader.resolve()
  Local<Value> resolved = env->esm_loader()->Resolve(specifier, parent_url);

  args.GetReturnValue().Set(resolved);
}
```

### 1. ESMLoader.import() 实现

```javascript
// lib/internal/modules/esm/loader.js
class ESMLoader {
  async import(specifier, parentURL, importAssertions) {
    // 🔥 解析模块路径
    const { url, format } = await this.resolve(specifier, parentURL, importAssertions);

    // 🔥 检查模块缓存
    let job = this.moduleMap.get(url);

    if (job === undefined) {
      // 🔥 创建新的 ModuleJob
      job = this.getModuleJob(url, parentURL, importAssertions);
      this.moduleMap.set(url, job);
    }

    // 🔥 运行模块
    const { module } = await job.run();

    // 🔥 返回模块命名空间
    return module.getNamespace();
  }

  getModuleJob(url, parentURL, importAssertions) {
    // 🔥 创建 ModuleJob
    const job = new ModuleJob(
      this,
      url,
      importAssertions,
      this.loadModule(url, parentURL, importAssertions)
    );

    return job;
  }

  async loadModule(url, parentURL, importAssertions) {
    // 🔥 加载模块源码
    const { source, format } = await this.load(url, { 
      parentURL, 
      importAssertions 
    });

    // 🔥 转换为 ModuleWrap
    const translator = this.translators.get(format);
    return translator(url, source, importAssertions);
  }
}
```

### 2. ModuleJob 类

```javascript
// lib/internal/modules/esm/module_job.js
class ModuleJob {
  constructor(loader, url, importAssertions, modulePromise) {
    this.loader = loader;
    this.url = url;
    this.importAssertions = importAssertions;

    // 🔥 模块加载 Promise
    this.modulePromise = modulePromise;

    // 🔥 模块实例
    this.module = undefined;

    // 🔥 是否已链接
    this.linked = false;

    // 🔥 是否已运行
    this.instantiated = undefined;
  }

  async run() {
    // 🔥 等待模块加载完成
    if (this.module === undefined) {
      this.module = await this.modulePromise;
    }

    // 🔥 链接模块
    if (!this.linked) {
      await this.link();
    }

    // 🔥 实例化并评估
    if (this.instantiated === undefined) {
      this.instantiated = this.instantiate();
    }

    await this.instantiated;

    return { module: this.module };
  }

  async link() {
    if (this.linked) return;

    // 🔥 链接所有依赖
    const promises = this.module.link(async (specifier, importAssertions) => {
      // 🔥 解析依赖
      const { url } = await this.loader.resolve(specifier, this.url, importAssertions);

      // 🔥 获取或创建依赖的 ModuleJob
      let job = this.loader.moduleMap.get(url);
      
      if (job === undefined) {
        job = this.loader.getModuleJob(url, this.url, importAssertions);
        this.loader.moduleMap.set(url, job);
      }

      // 🔥 等待依赖加载
      const { module } = await job.run();

      return module;
    });

    await Promise.all(promises);

    this.linked = true;
  }

  async instantiate() {
    // 🔥 评估模块
    return this.module.evaluate();
  }
}
```

```javascript
// lib/internal/modules/esm/module_wrap.js
const { ModuleWrap } = internalBinding('module_wrap');

class ESMModuleWrap {
  constructor(url, context, source) {
    // 🔥 创建 C++ ModuleWrap
    this.wrap = new ModuleWrap(url, context, source);
  }

  async link(linker) {
    // 🔥 链接依赖
    const requests = this.wrap.getStaticDependencies();

    const jobs = requests.map(async (specifier) => {
      const module = await linker(specifier);
      return module;
    });

    const modules = await Promise.all(jobs);

    // 🔥 设置依赖
    for (let i = 0; i < modules.length; i++) {
      this.wrap.setImport(i, modules[i].wrap);
    }
  }

  async evaluate() {
    // 🔥 评估模块（执行代码）
    return this.wrap.evaluate();
  }

  getNamespace() {
    // 🔥 获取模块命名空间
    return this.wrap.namespace();
  }

  getStaticDependencies() {
    // 🔥 获取静态依赖列表
    return this.wrap.getStaticDependencies();
  }
}
```

## 💻 模块缓存机制

### 1. ESMLoader 缓存

```javascript
// lib/internal/modules/esm/loader.js
class ESMLoader {
  constructor() {
    // 🔥 URL → ModuleJob 缓存
    this.moduleMap = new SafeMap();

    // 🔥 URL → { url, format } 缓存
    this.resolveCache = new SafeMap();
  }

  async resolve(specifier, parentURL, importAssertions) {
    // 🔥 生成缓存 key
    const cacheKey = JSON.stringify({ specifier, parentURL, importAssertions });

    // 🔥 检查缓存
    if (this.resolveCache.has(cacheKey)) {
      return this.resolveCache.get(cacheKey);
    }

    // 🔥 执行解析
    const result = await this._resolve(specifier, parentURL, importAssertions);

    // 🔥 缓存结果
    this.resolveCache.set(cacheKey, result);

    return result;
  }

  getModuleJob(url, parentURL, importAssertions) {
    // 🔥 检查缓存
    let job = this.moduleMap.get(url);

    if (job !== undefined) {
      return job;
    }

    // 🔥 创建新 job
    job = new ModuleJob(
      this,
      url,
      importAssertions,
      this.loadModule(url, parentURL, importAssertions)
    );

    // 🔥 加入缓存
    this.moduleMap.set(url, job);

    return job;
  }
}
```

```javascript
// 清空缓存（用于热重载）
function clearModuleCache(url) {
  const loader = getESMLoader();

  // 🔥 删除模块缓存
  loader.moduleMap.delete(url);

  // 🔥 删除解析缓存
  for (const [key, value] of loader.resolveCache.entries()) {
    if (value.url === url) {
      loader.resolveCache.delete(key);
    }
  }
}
```

### 3. 延迟加载

```javascript
// lazy-loading.js
class LazyModule {
  constructor(specifier) {
    this.specifier = specifier;
    this.promise = null;
    this.module = null;
  }

  async load() {
    if (this.module) {
      return this.module;
    }

    if (this.promise) {
      return this.promise;
    }

    // 🔥 开始加载
    this.promise = import(this.specifier).then(m => {
      this.module = m;
      this.promise = null;
      return m;
    });

    return this.promise;
  }

  async get(exportName) {
    const module = await this.load();
    return module[exportName];
  }
}

// 使用
const lazyUtils = new LazyModule('./utils.js');

// 🔥 第一次访问时才加载
const result = await lazyUtils.get('doSomething');
```

```javascript
// error-handling.js
async function safeImport(specifier) {
  try {
    return await import(specifier);
  } catch (error) {
    if (error.code === 'ERR_MODULE_NOT_FOUND') {
      console.error(`Module not found: ${specifier}`);
    } else if (error.code === 'ERR_INVALID_MODULE_SPECIFIER') {
      console.error(`Invalid module specifier: ${specifier}`);
    } else {
      console.error(`Failed to import ${specifier}:`, error);
    }
    
    throw error;
  }
}
```

```javascript
// retry-import.js
async function importWithRetry(specifier, maxRetries = 3, delay = 1000) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await import(specifier);
    } catch (error) {
      if (i === maxRetries - 1) {
        throw error;
      }

      console.log(`Import failed, retrying (${i + 1}/${maxRetries})...`);
      await new Promise(resolve => setTimeout(resolve, delay));
    }
  }
}
```

## 📝 总结

### 核心要点

1. **V8协作**：HostImportModuleDynamically → ModuleWrap
2. **执行流程**：resolve → load → link → evaluate
3. **缓存机制**：moduleMap 缓存 ModuleJob
4. **import.meta**：C++层生成，包含url、resolve等

### 实现关键

1. **ModuleWrap**：C++ Module 包装器
2. **ModuleJob**：模块加载任务
3. **ESMLoader**：模块加载器
4. **Promise链**：异步加载流程

### 应用价值

1. **按需加载**：减少初始加载时间
2. **条件导入**：根据条件加载不同模块
3. **代码分割**：优化打包体积
4. **性能优化**：预加载、懒加载
