## 📋 核心概念

Top-Level Await 允许在模块顶层**直接使用 await**，无需包裹在 async 函数中。

**核心问题：**
- Top-Level Await 如何工作？
- 模块图的异步评估顺序？
- 如何处理错误传播？
- 性能影响有多大？

**基本用法：**
```javascript
// module.mjs
const data = await fetch('/api/data');
const config = await import('./config.json', { assert: { type: 'json' } });

export default { data, config };
```

## 💻 V8 层实现

### 1. Module 异步评估

```cpp
// deps/v8/src/objects/module.cc

// 🔥 Module 状态
enum class Status {
  kUnlinked,    // 未链接
  kLinking,     // 链接中
  kLinked,      // 已链接
  kEvaluating,  // 评估中
  kEvaluated,   // 已评估
  kErrored      // 错误
};

// 🔥 评估模块（可能异步）
MaybeHandle<Object> Module::Evaluate(
    Isolate* isolate,
    Handle<Module> module) {
  
  // 🔥 检查模块状态
  if (module->status() == Status::kEvaluating) {
    // 🔥 正在评估，返回 Promise
    return module->top_level_capability()->promise();
  }

  if (module->status() == Status::kEvaluated) {
    // 🔥 已评估，返回结果
    return isolate->factory()->undefined_value();
  }

  // 🔥 创建 Promise
  Handle<JSPromise> promise = isolate->factory()->NewJSPromise();
  module->set_top_level_capability(promise);

  // 🔥 开始评估
  module->set_status(Status::kEvaluating);

  // 🔥 执行模块代码
  MaybeHandle<Object> result = ExecuteModule(isolate, module);

  if (result.is_null()) {
    // 🔥 评估失败
    module->set_status(Status::kErrored);
    return result;
  }

  // 🔥 检查是否有 Top-Level Await
  if (module->has_toplevel_await()) {
    // 🔥 异步评估：返回 Promise
    return promise;
  } else {
    // 🔥 同步评估：立即完成
    module->set_status(Status::kEvaluated);
    return result;
  }
}

// 🔥 执行模块代码
MaybeHandle<Object> ExecuteModule(
    Isolate* isolate,
    Handle<Module> module) {
  
  // 🔥 获取模块函数
  Handle<JSFunction> function = module->code();

  // 🔥 执行
  MaybeHandle<Object> result = Execution::Call(
    isolate,
    function,
    isolate->factory()->undefined_value(),
    0,
    nullptr
  );

  return result;
}
```

## 💻 Node.js JavaScript 层

### 1. ModuleJob 异步评估

```javascript
// lib/internal/modules/esm/module_job.js

class ModuleJob {
  constructor(loader, url, importAssertions, modulePromise) {
    this.loader = loader;
    this.url = url;
    this.modulePromise = modulePromise;
    this.module = undefined;
    this.linked = false;
    
    // 🔥 评估 Promise
    this.instantiated = undefined;
  }

  async run() {
    // 🔥 等待模块加载
    if (this.module === undefined) {
      this.module = await this.modulePromise;
    }

    // 🔥 链接模块
    if (!this.linked) {
      await this.link();
    }

    // 🔥 实例化并评估
    if (this.instantiated === undefined) {
      this.instantiated = this.instantiate();
    }

    // 🔥 等待评估完成（可能是异步的）
    await this.instantiated;

    return { module: this.module };
  }

  async instantiate() {
    // 🔥 评估模块（可能包含 Top-Level Await）
    return this.module.evaluate();
  }

  async link() {
    if (this.linked) return;

    // 🔥 链接所有依赖
    const linkerPromises = [];

    const dependencies = this.module.getStaticDependencies();
    
    for (const specifier of dependencies) {
      // 🔥 解析依赖
      const { url } = await this.loader.resolve(specifier, this.url);

      // 🔥 获取依赖的 ModuleJob
      let depJob = this.loader.moduleMap.get(url);
      
      if (depJob === undefined) {
        depJob = this.loader.getModuleJob(url, this.url);
        this.loader.moduleMap.set(url, depJob);
      }

      // 🔥 等待依赖的模块加载
      const depModule = await depJob.modulePromise;

      linkerPromises.push(depModule);
    }

    // 🔥 等待所有依赖加载完成
    const modules = await Promise.all(linkerPromises);

    // 🔥 链接依赖
    for (let i = 0; i < modules.length; i++) {
      this.module.link(i, modules[i]);
    }

    this.linked = true;
  }
}
```

### 2. 异步模块图评估

```javascript
// lib/internal/modules/esm/loader.js

class ESMLoader {
  async import(specifier, parentURL, importAssertions) {
    // 🔥 解析模块
    const { url } = await this.resolve(specifier, parentURL);

    // 🔥 获取 ModuleJob
    let job = this.moduleMap.get(url);
    
    if (job === undefined) {
      job = this.getModuleJob(url, parentURL, importAssertions);
      this.moduleMap.set(url, job);
    }

    // 🔥 运行模块（等待异步评估）
    const { module } = await job.run();

    // 🔥 返回模块命名空间
    return module.getNamespace();
  }

  /**
   * 评估模块图
   * 处理 Top-Level Await 的依赖关系
   */
  async evaluateModuleGraph(job) {
    const visited = new Set();
    const evaluating = new Map();

    // 🔥 深度优先评估
    async function evaluate(currentJob) {
      // 🔥 避免重复评估
      if (visited.has(currentJob.url)) {
        return evaluating.get(currentJob.url);
      }

      visited.add(currentJob.url);

      // 🔥 等待模块加载
      const { module } = await currentJob.modulePromise;

      // 🔥 先评估所有依赖
      const dependencies = module.getStaticDependencies();
      const depPromises = [];

      for (const specifier of dependencies) {
        const { url } = await this.resolve(specifier, currentJob.url);
        const depJob = this.moduleMap.get(url);
        
        if (depJob) {
          depPromises.push(evaluate(depJob));
        }
      }

      // 🔥 等待所有依赖评估完成
      await Promise.all(depPromises);

      // 🔥 评估当前模块
      const evaluationPromise = module.evaluate();
      evaluating.set(currentJob.url, evaluationPromise);

      // 🔥 等待评估完成（可能是异步的）
      await evaluationPromise;

      return evaluationPromise;
    }

    return evaluate(job);
  }
}
```

## 💻 执行流程详解

### 1. 同步模块评估

```javascript
// sync-module.mjs
console.log('Start');

const value = 42;

console.log('End');

export { value };
```

**执行流程：**
```
1. 解析模块
2. 实例化模块
3. 评估模块（同步）
4. 标记为已评估
```

---

### 2. 异步模块评估

```javascript
// async-module.mjs
console.log('Start');

// 🔥 Top-Level Await
const data = await fetch('/api/data');

console.log('Data loaded');

export { data };
```

**执行流程：**
```
1. 解析模块
2. 实例化模块
3. 开始评估
4. 遇到 await，挂起模块
5. 创建 Promise
6. 继续处理其他模块
7. await 完成后，恢复评估
8. 标记为已评估
9. Resolve Promise
```

---

### 3. 依赖链异步评估

```javascript
// a.mjs
console.log('A: start');

const data = await fetch('/api/data');

console.log('A: data loaded');

export { data };

// b.mjs
console.log('B: start');

import { data } from './a.mjs';  // 🔥 等待 A 的 await

console.log('B: got data from A');

const processed = await processData(data);

console.log('B: data processed');

export { processed };

// main.mjs
import { processed } from './b.mjs';  // 🔥 等待 B 的 await

console.log('Main: got processed data');
```

**执行顺序：**
```
1. A: start
2. A 挂起（await fetch）
3. （不会执行 B，因为 B 依赖 A）
4. （fetch 完成）
5. A: data loaded
6. B: start
7. B: got data from A
8. B 挂起（await processData）
9. （processData 完成）
10. B: data processed
11. Main: got processed data
```

## 📝 总结

### 核心要点

1. **异步评估**：模块评估可能返回Promise
2. **依赖顺序**：按依赖图深度优先评估
3. **错误传播**：异步错误通过Promise传播
4. **性能影响**：Top-Level Await会阻塞依赖者

### 实现关键

1. **V8 Module**：top_level_capability Promise
2. **ModuleJob**：instantiate()返回Promise
3. **ESMLoader**：evaluateModuleGraph管理评估
4. **Promise链**：管理异步依赖关系

### 应用价值

1. **简化代码**：无需async函数包裹
2. **配置加载**：模块级配置初始化
3. **依赖管理**：确保依赖按序初始化
4. **错误处理**：统一的异步错误处理
