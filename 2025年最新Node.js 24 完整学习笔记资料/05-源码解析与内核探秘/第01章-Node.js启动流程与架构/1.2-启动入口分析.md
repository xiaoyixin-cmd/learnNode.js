## 📋 知识点概述

### 核心概念

Node.js 的启动过程是一个从 C++ 到 JavaScript 的复杂转换过程，涉及多个关键步骤：

**启动链路：**
1. **操作系统启动** → `node_main.cc::main()`
2. **初始化 V8 Platform** → 多线程支持
3. **创建 Isolate** → V8 隔离实例
4. **创建 Context** → JavaScript 执行环境
5. **加载 Environment** → Node.js 运行环境
6. **执行 Bootstrap** → 引导 JavaScript 代码
7. **启动事件循环** → 进入 libuv 循环

**解决的问题：**
- 如何从操作系统进程启动到 JavaScript 代码执行？
- V8 引擎如何初始化和配置？
- Node.js 环境如何创建和管理？
- 如何处理命令行参数和环境变量？

## 💻 源码分析：main() 入口

### 1. node_main.cc - 跨平台入口

```cpp
// src/node_main.cc
#include "node.h"
#include "node_internals.h"
#include "node_options.h"

#ifdef _WIN32
#include <windows.h>
#include <shellapi.h>
#include <VersionHelpers.h>

// Windows 入口（支持 Unicode）
int wmain(int argc, wchar_t* wargv[]) {
  // 1. 设置控制台代码页为 UTF-8
  SetConsoleCP(CP_UTF8);
  SetConsoleOutputCP(CP_UTF8);
  
  // 2. 转换 UTF-16 参数到 UTF-8
  char** argv = new char*[argc + 1];
  for (int i = 0; i < argc; i++) {
    // 计算 UTF-8 长度
    int size = WideCharToMultiByte(
      CP_UTF8, 0, wargv[i], -1, nullptr, 0, nullptr, nullptr
    );
    
    argv[i] = new char[size];
    
    // 转换
    WideCharToMultiByte(
      CP_UTF8, 0, wargv[i], -1, argv[i], size, nullptr, nullptr
    );
  }
  argv[argc] = nullptr;
  
  // 3. 调用 Node.js 主函数
  int exit_code = node::Start(argc, argv);
  
  // 4. 清理
  for (int i = 0; i < argc; i++) {
    delete[] argv[i];
  }
  delete[] argv;
  
  return exit_code;
}

#else

// Unix/Linux/macOS 入口
int main(int argc, char* argv[]) {
  // 直接调用 Node.js 主函数
  return node::Start(argc, argv);
}

#endif
```

**关键点：**
1. **跨平台支持**：Windows 使用 `wmain()`，Unix 使用 `main()`
2. **编码转换**：Windows 需要将 UTF-16 转为 UTF-8
3. **统一接口**：最终都调用 `node::Start()`

```cpp
// src/node.cc
namespace node {

int Start(int argc, char** argv) {
  // 1. 保存原始参数
  per_process::cli_options->cmdline = {argv, argv + argc};
  
  // 2. 解析 Node.js 参数
  std::vector<std::string> errors;
  std::vector<std::string> exec_args;
  std::vector<std::string> v8_args;
  
  per_process::cli_options.reset(
    new PerProcessOptions()
  );
  
  // 解析参数
  //   node [node-options] [script] [script-args]
  //   node inspect [script] [script-args]
  //   node [--v8-options]
  int exit_code = ParseArgs(
    &argc,
    argv,
    per_process::cli_options.get(),
    &errors
  );
  
  if (exit_code != 0) {
    return exit_code;
  }
  
  // 3. 处理特殊参数
  if (per_process::cli_options->print_version) {
    printf("%s\n", NODE_VERSION);
    return 0;
  }
  
  if (per_process::cli_options->print_help) {
    PrintHelp();
    return 0;
  }
  
  // 4. 设置 V8 Flags
  V8::SetFlagsFromString(
    per_process::cli_options->v8_flags.c_str(),
    per_process::cli_options->v8_flags.size()
  );
  
  // 5. 继续启动流程
  return StartNodeInstance(argc, argv);
}

} // namespace node
```

### 2. 创建 V8 Platform

```cpp
// src/node.cc
std::unique_ptr<MultiIsolatePlatform> CreatePlatform(
  int thread_pool_size,
  v8::TracingController* tracing_controller
) {
  if (thread_pool_size <= 0) {
    // 默认线程数 = CPU 核心数
    thread_pool_size = NumberOfWorkerThreads();
  }
  
  // 创建 Platform（支持多 Isolate）
  auto platform = std::make_unique<NodePlatform>(
    thread_pool_size,
    tracing_controller
  );
  
  // 初始化 V8 Platform
  V8::InitializePlatform(platform.get());
  
  return platform;
}

// NodePlatform 实现
class NodePlatform : public MultiIsolatePlatform {
 public:
  NodePlatform(int thread_pool_size, TracingController* tracing)
    : tracing_controller_(tracing) {
    // 创建工作线程池
    for (int i = 0; i < thread_pool_size; i++) {
      auto worker = std::make_unique<WorkerThread>(this);
      workers_.push_back(std::move(worker));
    }
  }
  
  // 提交任务到线程池
  void CallOnWorkerThread(std::unique_ptr<Task> task) override {
    workers_[next_worker_++ % workers_.size()]->PostTask(std::move(task));
  }
  
  // 延迟任务
  void CallDelayedOnWorkerThread(
    std::unique_ptr<Task> task,
    double delay_in_seconds
  ) override {
    // 使用 libuv 定时器
    uv_timer_t* timer = new uv_timer_t;
    timer->data = task.release();
    
    uv_timer_init(uv_default_loop(), timer);
    uv_timer_start(
      timer,
      [](uv_timer_t* handle) {
        auto task = static_cast<Task*>(handle->data);
        task->Run();
        delete task;
        uv_close(reinterpret_cast<uv_handle_t*>(handle), 
                [](uv_handle_t* h) { delete h; });
      },
      delay_in_seconds * 1000,
      0
    );
  }
  
 private:
  std::vector<std::unique_ptr<WorkerThread>> workers_;
  TracingController* tracing_controller_;
  std::atomic<size_t> next_worker_{0};
};
```

**Platform 作用：**
- 提供线程池（后台编译、GC等）
- 任务调度
- 性能追踪
- 跨平台抽象

```cpp
// src/node.cc
Isolate* NewIsolate(
  ArrayBufferAllocator* allocator,
  uv_loop_t* event_loop,
  MultiIsolatePlatform* platform
) {
  // 1. 配置 Isolate 参数
  Isolate::CreateParams params;
  params.array_buffer_allocator = allocator;
  
  // 2. 设置外部引用表（C++ 函数地址）
  params.external_references = node::external_references;
  
  // 3. 加载 Snapshot（如果存在）
  if (per_process::cli_options->node_snapshot) {
    const SnapshotData* snapshot_data = 
      SnapshotBuilder::GetEmbeddedSnapshotData();
    
    if (snapshot_data != nullptr) {
      params.snapshot_blob = snapshot_data;
    }
  }
  
  // 4. 设置代码事件处理器（性能分析）
  params.code_event_handler = nullptr;  // 可选
  
  // 5. 创建 Isolate
  Isolate* isolate = Isolate::New(params);
  
  // 6. 设置内存限制
  if (per_process::cli_options->max_old_space_size > 0) {
    isolate->SetHeapSizeLimit(
      per_process::cli_options->max_old_space_size * 1024 * 1024
    );
  }
  
  // 7. 设置回调
  isolate->SetPromiseRejectCallback(PromiseRejectCallback);
  isolate->SetFatalErrorHandler(OnFatalError);
  isolate->SetOOMErrorHandler(OOMErrorHandler);
  isolate->SetPrepareStackTraceCallback(PrepareStackTraceCallback);
  
  // 8. 添加 GC 回调
  isolate->AddGCPrologueCallback(GCPrologueCallback);
  isolate->AddGCEpilogueCallback(GCEpilogueCallback);
  
  return isolate;
}
```

- **内存分配器**：ArrayBufferAllocator

### 2. ArrayBufferAllocator 实现

```cpp
// src/node.cc
class NodeArrayBufferAllocator : public ArrayBuffer::Allocator {
 public:
  void* Allocate(size_t length) override {
    if (length > kMaxLength) return nullptr;
    
    // 使用 libuv 内存分配
    void* data = UncheckedMalloc(length);
    if (data == nullptr) {
      FatalErrorCallback(nullptr, "Out of memory");
    }
    
    // 初始化为 0
    memset(data, 0, length);
    return data;
  }
  
  void* AllocateUninitialized(size_t length) override {
    if (length > kMaxLength) return nullptr;
    
    // 不初始化（性能优化）
    void* data = UncheckedMalloc(length);
    if (data == nullptr) {
      FatalErrorCallback(nullptr, "Out of memory");
    }
    
    return data;
  }
  
  void Free(void* data, size_t length) override {
    // 释放内存
    free(data);
  }
  
 private:
  static constexpr size_t kMaxLength = 0x7fffffff;
};
```

```cpp
// src/api/environment.cc
MaybeLocal<Value> LoadEnvironment(
  Environment* env,
  const char* main_script_source
) {
  CHECK_NOT_NULL(env);
  
  Isolate* isolate = env->isolate();
  HandleScope handle_scope(isolate);
  
  // 1. 执行 Bootstrap
  Local<Value> result;
  if (!env->RunBootstrapping().ToLocal(&result)) {
    return MaybeLocal<Value>();
  }
  
  // 2. 执行用户代码
  if (main_script_source != nullptr) {
    return env->RunUserCode(main_script_source);
  }
  
  return result;
}
```

```cpp
// src/node.cc
MaybeLocal<Value> Environment::RunBootstrapping() {
  EscapableHandleScope scope(isolate_);
  
  // 1. 加载 internal/bootstrap/loaders.js
  Local<Value> loaders_bootstrapper = 
    ExecuteBootstrapper(
      isolate_,
      "internal/bootstrap/loaders",
      nullptr
    );
  
  if (loaders_bootstrapper.IsEmpty()) {
    return MaybeLocal<Value>();
  }
  
  // 2. 加载 internal/bootstrap/node.js
  Local<Value> node_bootstrapper = 
    ExecuteBootstrapper(
      isolate_,
      "internal/bootstrap/node",
      &loaders_bootstrapper
    );
  
  if (node_bootstrapper.IsEmpty()) {
    return MaybeLocal<Value>();
  }
  
  // 3. 初始化模块系统
  Local<Function> run_main_module = 
    GetBootstrapperFunction("internal/main/run_main_module");
  
  set_main_module_loader(run_main_module);
  
  return scope.Escape(node_bootstrapper);
}
```
