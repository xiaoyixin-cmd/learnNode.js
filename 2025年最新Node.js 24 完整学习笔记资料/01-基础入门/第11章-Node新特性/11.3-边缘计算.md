# 11.3 边缘计算与Serverless

> 探索Node.js在边缘计算和Serverless中的应用

---

## 🎯 什么是边缘计算？

### 核心概念

```
边缘计算 = 在靠近用户的地方运行代码

传统架构：
用户 → CDN → 服务器
      ↑
   静态资源

边缘计算：
用户 → 边缘节点（运行代码）→ 服务器
      ↑
   动态内容+静态资源

优势：
✅ 更低的延迟（物理距离近）
✅ 更好的性能
✅ 自动扩展
✅ 按使用付费
✅ 全球分发

使用场景：
• API路由
• A/B测试
• 个性化内容
• 请求转换
• 身份验证
• 图片优化
```

---

## ☁️ Serverless概述

### Serverless vs 传统服务器

```
传统服务器：
• 需要管理服务器
• 24/7运行（即使没请求）
• 固定成本
• 需要手动扩展

Serverless：
• 无需管理服务器
• 按需运行（有请求才执行）
• 按使用付费
• 自动扩展

流行平台：
• AWS Lambda
• Vercel Edge Functions
• Cloudflare Workers
• Netlify Functions
• Azure Functions
```

## 🌍 Cloudflare Workers

### 1. 基础使用

```javascript
// ===== worker.js =====

// Cloudflare Workers使用Web标准API

// 监听请求
addEventListener('fetch', event => {
  event.respondWith(handleRequest(event.request));
});

async function handleRequest(request) {
  return new Response('Hello from Cloudflare Workers!', {
    headers: {
      'content-type': 'text/plain',
    },
  });
}

// ===== 路由示例 =====

addEventListener('fetch', event => {
  event.respondWith(handleRequest(event.request));
});

async function handleRequest(request) {
  const { pathname } = new URL(request.url);
  
  // 路由
  if (pathname === '/') {
    return new Response('Home Page');
  }
  
  if (pathname === '/api/hello') {
    return new Response(JSON.stringify({ message: 'Hello!' }), {
      headers: { 'content-type': 'application/json' }
    });
  }
  
  if (pathname.startsWith('/user/')) {
    const userId = pathname.split('/')[2];
    return new Response(`User ID: ${userId}`);
  }
  
  return new Response('Not Found', { status: 404 });
}
```

### 2. 使用KV存储

```javascript
// ===== KV存储（键值对）=====

addEventListener('fetch', event => {
  event.respondWith(handleRequest(event.request));
});

async function handleRequest(request) {
  const { pathname } = new URL(request.url);
  
  // 从KV读取
  if (pathname === '/get') {
    const value = await MY_KV.get('key');
    return new Response(value || 'Not found');
  }
  
  // 写入KV
  if (pathname === '/set') {
    await MY_KV.put('key', 'value', {
      expirationTtl: 3600  // 1小时后过期
    });
    return new Response('Saved');
  }
  
  // 删除
  if (pathname === '/delete') {
    await MY_KV.delete('key');
    return new Response('Deleted');
  }
}

// ===== 实战：访问计数器 =====

async function handleVisit(request) {
  // 获取当前计数
  const countStr = await MY_KV.get('visit_count') || '0';
  const count = parseInt(countStr) + 1;
  
  // 更新计数
  await MY_KV.put('visit_count', count.toString());
  
  return new Response(`访问次数: ${count}`);
}
```

### 案例1：图片优化边缘函数

```javascript
// ===== image-optimizer.js =====

export const config = {
  runtime: 'edge',
};

export default async function handler(request) {
  const { searchParams } = new URL(request.url);
  const imageUrl = searchParams.get('url');
  const width = parseInt(searchParams.get('width')) || 800;
  const quality = parseInt(searchParams.get('quality')) || 80;
  
  if (!imageUrl) {
    return new Response('Missing url parameter', { status: 400 });
  }
  
  try {
    // 获取原始图片
    const response = await fetch(imageUrl);
    const arrayBuffer = await response.arrayBuffer();
    
    // 使用边缘函数优化图片
    // （实际实现需要图片处理库）
    const optimized = await optimizeImage(arrayBuffer, {
      width,
      quality
    });
    
    return new Response(optimized, {
      headers: {
        'Content-Type': 'image/webp',
        'Cache-Control': 'public, max-age=31536000',
      },
    });
  } catch (error) {
    return new Response('Image optimization failed', { status: 500 });
  }
}

// 使用：
// /api/image?url=https://example.com/image.jpg&width=400&quality=80
```

### 案例2：API网关

```javascript
// ===== api-gateway.js =====

export const config = {
  runtime: 'edge',
};

// API路由配置
const routes = {
  '/api/users': 'https://users-service.example.com',
  '/api/posts': 'https://posts-service.example.com',
  '/api/comments': 'https://comments-service.example.com',
};

export default async function handler(request) {
  const { pathname } = new URL(request.url);
  
  // 身份验证
  const token = request.headers.get('Authorization');
  if (!token) {
    return Response.json({ error: 'Unauthorized' }, { status: 401 });
  }
  
  // 查找目标服务
  const targetBase = routes[pathname.split('/').slice(0, 3).join('/')];
  if (!targetBase) {
    return Response.json({ error: 'Not Found' }, { status: 404 });
  }
  
  // 构建目标URL
  const targetUrl = `${targetBase}${pathname}`;
  
  try {
    // 转发请求
    const response = await fetch(targetUrl, {
      method: request.method,
      headers: request.headers,
      body: request.body,
    });
    
    return response;
  } catch (error) {
    return Response.json(
      { error: 'Service Unavailable' },
      { status: 503 }
    );
  }
}
```

### 案例3：地理位置重定向

```javascript
// ===== geo-redirect.js =====

export const config = {
  runtime: 'edge',
};

// 地区到域名映射
const geoMapping = {
  'US': 'https://us.example.com',
  'EU': 'https://eu.example.com',
  'CN': 'https://cn.example.com',
  'default': 'https://example.com'
};

export default function handler(request) {
  // 获取用户地理位置
  const country = request.geo?.country || 'default';
  
  // 查找对应域名
  const targetDomain = geoMapping[country] || geoMapping.default;
  
  // 重定向
  const url = new URL(request.url);
  return Response.redirect(`${targetDomain}${url.pathname}${url.search}`, 302);
}
```

```
Cloudflare Workers：
✅ 全球最大的边缘网络
✅ 性能极佳（<50ms启动）
✅ KV和Durable Objects
✅ 价格便宜
❌ 有资源限制

Vercel Edge Functions：
✅ 与Next.js深度集成
✅ 开发体验好
✅ 部署简单
❌ 主要面向前端

AWS Lambda@Edge：
✅ AWS生态集成
✅ 功能强大
❌ 冷启动较慢
❌ 配置复杂

选择标准：
• 已用Next.js → Vercel
• 追求性能 → Cloudflare
• AWS生态 → Lambda@Edge
```
