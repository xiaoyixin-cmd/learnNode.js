# 9.5 MQTT 与 IoT

> 使用 MQTT 协议在 Node.js 中构建物联网应用

## 📚 知识导读

### 什么是 MQTT？

**定义：**  
MQTT（Message Queuing Telemetry Transport）是一种轻量级的发布/订阅消息协议，专为物联网（IoT）设计。

**核心特点：**
- **轻量级**：协议头仅2字节
- **低带宽**：适合2G/3G网络
- **QoS 保证**：3种服务质量等级
- **双向通信**：发布/订阅模式

**应用场景：**
- 智能家居（温湿度监控、灯光控制）
- 工业 IoT（设备状态、数据采集）
- 车联网（位置追踪、远程控制）
- 智慧城市（环境监测、交通管理）

---

## 🎯 MQTT 核心概念

### 1. 发布/订阅模式

```
发布者（Publisher）→ [主题(Topic)] → Broker → [主题] → 订阅者（Subscriber）
```

**角色：**
- **Broker**：消息中间件（Mosquitto/EMQX）
- **Publisher**：发布消息（传感器）
- **Subscriber**：订阅消息（控制器）

---

### 2. QoS 服务质量

| QoS | 名称 | 说明 | 场景 |
|-----|------|------|------|
| **0** | At most once | 最多一次（不保证） | 日志、非关键数据 |
| **1** | At least once | 至少一次（可能重复） | 状态更新 |
| **2** | Exactly once | 恰好一次（最可靠） | 支付、控制指令 |

---

### 3. 主题（Topic）

```
home/living-room/temperature        # 客厅温度
home/bedroom/light/status           # 卧室灯状态
factory/machine-01/speed            # 机器速度
car/VIN123456/location              # 车辆位置
```

**通配符：**
- `+`：单层通配符（`home/+/temperature`）
- `#`：多层通配符（`home/#`）

## 🎯 MQTT.js 客户端

### 1. 安装与连接

```bash
npm install mqtt
```

```javascript
import mqtt from 'mqtt';

// 连接 Broker
const client = mqtt.connect('mqtt://broker.emqx.io:1883', {
  clientId: `nodejs_${Math.random().toString(16).slice(3)}`,
  clean: true,      // 清除会话
  connectTimeout: 4000,
  username: 'admin',
  password: 'public',
  reconnectPeriod: 1000,  // 重连间隔
});

// 连接成功
client.on('connect', () => {
  console.log('✅ Connected to MQTT Broker');
});

// 连接失败
client.on('error', (error) => {
  console.error('❌ Connection failed:', error);
});

// 断开连接
client.on('close', () => {
  console.log('🔌 Connection closed');
});
```

---

### 2. 订阅主题

```javascript
// 订阅单个主题
client.subscribe('home/living-room/temperature', { qos: 1 }, (err) => {
  if (!err) {
    console.log('📩 Subscribed to temperature topic');
  }
});

// 订阅多个主题
client.subscribe([
  'home/+/temperature',
  'home/bedroom/light/#'
], { qos: 1 });

// 接收消息
client.on('message', (topic, message) => {
  console.log(`📨 Received from ${topic}:`, message.toString());
  
  // 解析 JSON 消息
  try {
    const data = JSON.parse(message.toString());
    console.log('Temperature:', data.value, data.unit);
  } catch (error) {
    console.error('Invalid JSON:', error);
  }
});
```

---

### 3. 发布消息

```javascript
// 发布温度数据
const temperatureData = {
  value: 25.6,
  unit: '°C',
  timestamp: Date.now()
};

client.publish(
  'home/living-room/temperature',
  JSON.stringify(temperatureData),
  { qos: 1, retain: true },  // retain: 保留消息
  (err) => {
    if (!err) {
      console.log('✅ Temperature published');
    }
  }
);

// 发布控制指令
client.publish('home/bedroom/light/control', 'ON', { qos: 2 });
```

### 1. 温湿度监控系统

**模拟传感器（Publisher）：**
```javascript
import mqtt from 'mqtt';

const client = mqtt.connect('mqtt://broker.emqx.io:1883');

client.on('connect', () => {
  console.log('🌡️ Sensor connected');
  
  // 每5秒发送一次数据
  setInterval(() => {
    const data = {
      temperature: (20 + Math.random() * 10).toFixed(1),
      humidity: (40 + Math.random() * 20).toFixed(1),
      timestamp: new Date().toISOString()
    };
    
    client.publish(
      'sensor/dht22/data',
      JSON.stringify(data),
      { qos: 1 }
    );
    
    console.log('📤 Sensor data sent:', data);
  }, 5000);
});
```

**监控后端（Subscriber）：**
```javascript
import mqtt from 'mqtt';
import { createClient } from 'redis';

const mqttClient = mqtt.connect('mqtt://broker.emqx.io:1883');
const redisClient = createClient();
await redisClient.connect();

mqttClient.on('connect', () => {
  mqttClient.subscribe('sensor/+/data', { qos: 1 });
});

mqttClient.on('message', async (topic, message) => {
  const data = JSON.parse(message.toString());
  
  // 存储到 Redis
  await redisClient.hSet(`sensor:${topic}`, {
    temperature: data.temperature,
    humidity: data.humidity,
    timestamp: data.timestamp
  });
  
  // 检查报警条件
  if (parseFloat(data.temperature) > 30) {
    console.log('🚨 Temperature alert:', data.temperature);
    
    // 发送控制指令（开启空调）
    mqttClient.publish('device/ac/control', JSON.stringify({
      action: 'turn_on',
      temperature: 26
    }), { qos: 2 });
  }
});
```

### 2. 智能灯控系统

**Express API + MQTT：**
```javascript
import express from 'express';
import mqtt from 'mqtt';

const app = express();
app.use(express.json());

const mqttClient = mqtt.connect('mqtt://broker.emqx.io:1883');

// 存储灯状态
const lightStatus = {};

// 订阅灯状态反馈
mqttClient.on('connect', () => {
  mqttClient.subscribe('home/+/light/status', { qos: 1 });
});

mqttClient.on('message', (topic, message) => {
  const [, room, , ] = topic.split('/');
  lightStatus[room] = message.toString();
  console.log(`💡 ${room} light: ${lightStatus[room]}`);
});

// 控制灯光 API
app.post('/api/light/:room', (req, res) => {
  const { room } = req.params;
  const { action } = req.body;  // "ON" | "OFF"
  
  mqttClient.publish(
    `home/${room}/light/control`,
    action,
    { qos: 2 },
    (err) => {
      if (err) {
        return res.status(500).json({ error: 'Failed to control light' });
      }
      
      res.json({ success: true, room, action });
    }
  );
});

// 查询灯状态
app.get('/api/light/:room', (req, res) => {
  const { room } = req.params;
  res.json({ room, status: lightStatus[room] || 'UNKNOWN' });
});

app.listen(3000, () => {
  console.log('🚀 Smart Light API running on http://localhost:3000');
});
```

### 3. 设备在线状态监控

**Last Will & Testament（遗嘱消息）：**
```javascript
// 设备客户端
const deviceClient = mqtt.connect('mqtt://broker.emqx.io:1883', {
  will: {
    topic: 'device/esp32-001/status',
    payload: 'offline',
    qos: 1,
    retain: true
  }
});

deviceClient.on('connect', () => {
  // 发送在线状态
  deviceClient.publish('device/esp32-001/status', 'online', {
    qos: 1,
    retain: true
  });
});

// 监控客户端
const monitorClient = mqtt.connect('mqtt://broker.emqx.io:1883');

monitorClient.on('connect', () => {
  monitorClient.subscribe('device/+/status', { qos: 1 });
});

monitorClient.on('message', (topic, message) => {
  const deviceId = topic.split('/')[1];
  const status = message.toString();
  
  console.log(`📡 Device ${deviceId}: ${status}`);
  
  if (status === 'offline') {
    // 发送报警通知
    console.error(`🚨 Device ${deviceId} is offline!`);
  }
});
```

### 2. Node.js Aedes Broker

```bash
npm install aedes aedes-server-factory
```

```javascript
import aedes from 'aedes';
import { createServer } from 'aedes-server-factory';

const broker = aedes();
const port = 1883;

// 创建 TCP 服务器
const server = createServer(broker, { tcp: true });

// 客户端连接
broker.on('client', (client) => {
  console.log(`✅ Client connected: ${client.id}`);
});

// 客户端断开
broker.on('clientDisconnect', (client) => {
  console.log(`❌ Client disconnected: ${client.id}`);
});

// 消息发布
broker.on('publish', (packet, client) => {
  if (client) {
    console.log(`📨 ${client.id} published to ${packet.topic}`);
  }
});

// 客户端订阅
broker.on('subscribe', (subscriptions, client) => {
  subscriptions.forEach((sub) => {
    console.log(`📩 ${client.id} subscribed to ${sub.topic}`);
  });
});

server.listen(port, () => {
  console.log(`🚀 MQTT Broker running on port ${port}`);
});
```

### 2. Node.js WebSocket Bridge

```javascript
import mqtt from 'mqtt';
import express from 'express';
import { WebSocketServer } from 'ws';

const app = express();
const server = app.listen(3000);
const wss = new WebSocketServer({ server });

const mqttClient = mqtt.connect('mqtt://broker.emqx.io:1883');

// MQTT → WebSocket
mqttClient.on('connect', () => {
  mqttClient.subscribe('sensor/#', { qos: 1 });
});

mqttClient.on('message', (topic, message) => {
  // 广播给所有 WebSocket 客户端
  wss.clients.forEach((ws) => {
    if (ws.readyState === WebSocket.OPEN) {
      ws.send(JSON.stringify({ topic, message: message.toString() }));
    }
  });
});

// WebSocket → MQTT
wss.on('connection', (ws) => {
  ws.on('message', (data) => {
    const { topic, message } = JSON.parse(data.toString());
    mqttClient.publish(topic, message, { qos: 1 });
  });
});

console.log('WebSocket server running on http://localhost:3000');
```

## ⚠️ 三个高频错误

### 错误 1：忘记处理重连

```javascript
// ❌ 错误：连接断开后无法恢复
const client = mqtt.connect('mqtt://broker.emqx.io:1883');

// ✅ 正确：自动重连
const client = mqtt.connect('mqtt://broker.emqx.io:1883', {
  reconnectPeriod: 1000,  // 1秒重连
  connectTimeout: 30000,
});

client.on('reconnect', () => {
  console.log('🔄 Reconnecting...');
});
```

---

### 错误 2：消息体过大

```javascript
// ❌ 错误：发送大型 JSON（可能超过 Broker 限制）
const largeData = { images: [...Array(1000)] };
client.publish('topic', JSON.stringify(largeData));

// ✅ 正确：分片或压缩
import zlib from 'zlib';

const compressed = zlib.gzipSync(JSON.stringify(largeData));
client.publish('topic', compressed);

// 接收端解压
client.on('message', (topic, message) => {
  const decompressed = zlib.gunzipSync(message);
  const data = JSON.parse(decompressed.toString());
});
```

---

### 错误 3：QoS 使用不当

```javascript
// ❌ 错误：关键指令使用 QoS 0
client.publish('device/ac/turn_off', 'OFF', { qos: 0 });  // 可能丢失！

// ✅ 正确：关键消息使用 QoS 2
client.publish('device/ac/turn_off', 'OFF', { qos: 2 });
```

## 📚 FAQ

### 1. MQTT vs HTTP？

| 特性 | MQTT | HTTP |
|------|------|------|
| **协议头** | 2 字节 | ~800 字节 |
| **连接** | 长连接 | 短连接 |
| **实时性** | 高（推送） | 低（轮询） |
| **适用场景** | IoT、实时通信 | Web API |

### 2. 如何保证消息安全？

**方案：**
1. **TLS/SSL**：`mqtts://` 协议
2. **用户名密码**：基础认证
3. **Token 认证**：客户端证书
4. **消息加密**：AES-256 加密 payload

```javascript
// TLS 连接
const client = mqtt.connect('mqtts://broker.emqx.io:8883', {
  key: readFileSync('./client-key.pem'),
  cert: readFileSync('./client-cert.pem'),
  ca: readFileSync('./ca.pem'),
  rejectUnauthorized: true
});
```

### 3. Broker 选型？

| Broker | 特点 | 适用场景 |
|--------|------|----------|
| **EMQX** | 高性能、企业级 | 生产环境 |
| **Mosquitto** | 轻量、开源 | 测试、小型项目 |
| **Aedes** | Node.js 原生 | 嵌入式应用 |
| **HiveMQ** | 云原生 | 云平台 |
