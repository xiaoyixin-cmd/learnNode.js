## 📚 知识导读

### 什么是 WebAssembly？

**WebAssembly（WASM）** 是一种低级字节码格式，可在现代 Web 浏览器和 Node.js 中运行。

**特点：**
- **高性能**：接近原生代码速度（C/C++/Rust 性能）
- **跨平台**：浏览器、Node.js、边缘计算
- **安全**：沙箱执行环境
- **语言无关**：支持 C/C++、Rust、AssemblyScript 等

**性能对比：**
```
计算密集型任务（斐波那契数列 n=40）

JavaScript: 1200ms
WASM (Rust): 120ms  ← 10倍提升 ✅
C++ Native: 100ms
```

### 2. 创建 Rust WASM 项目

```bash

```

**Cargo.toml：**
```toml
[package]
name = "fibonacci-wasm"
version = "0.1.0"
edition = "2021"

[lib]
crate-type = ["cdylib"]

[dependencies]
wasm-bindgen = "0.2"

[profile.release]
opt-level = 3        # 最高优化级别
lto = true          # Link Time Optimization
codegen-units = 1   # 单个代码生成单元（更好的优化）
```

**src/lib.rs：**
```rust
use wasm_bindgen::prelude::*;

// 导出到 JavaScript
#[wasm_bindgen]
pub fn fibonacci(n: u32) -> u64 {
    if n <= 1 {
        return n as u64;
    }
    
    let mut a: u64 = 0;
    let mut b: u64 = 1;
    
    for _ in 2..=n {
        let temp = a + b;
        a = b;
        b = temp;
    }
    
    b
}

// 图像处理示例
#[wasm_bindgen]
pub fn grayscale(data: &mut [u8]) {
    for chunk in data.chunks_mut(4) {
        let r = chunk[0] as f32;
        let g = chunk[1] as f32;
        let b = chunk[2] as f32;
        
        // 灰度公式
        let gray = (0.299 * r + 0.587 * g + 0.114 * b) as u8;
        
        chunk[0] = gray;
        chunk[1] = gray;
        chunk[2] = gray;
        // chunk[3] = alpha (不变)
    }
}
```

### 4. Node.js 使用 WASM

```javascript
// index.js
import { fibonacci, grayscale } from './pkg/fibonacci_wasm.js';

// 斐波那契数列
console.time('WASM fibonacci');
const result = fibonacci(40);
console.timeEnd('WASM fibonacci');
// WASM fibonacci: 120ms

console.log(`fibonacci(40) = ${result}`);

// 图像灰度化
import fs from 'fs';
import { createCanvas, loadImage } from 'canvas';

async function processImage() {
  const image = await loadImage('input.jpg');
  const canvas = createCanvas(image.width, image.height);
  const ctx = canvas.getContext('2d');
  
  ctx.drawImage(image, 0, 0);
  const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
  
  console.time('WASM grayscale');
  grayscale(imageData.data);  // WASM 处理
  console.timeEnd('WASM grayscale');
  // WASM grayscale: 15ms（纯 JS: 150ms）
  
  ctx.putImageData(imageData, 0, 0);
  fs.writeFileSync('output.jpg', canvas.toBuffer('image/jpeg'));
}

processImage();
```

### 2. C++ 示例

**fibonacci.cpp：**
```cpp
#include <emscripten/emscripten.h>

extern "C" {
  EMSCRIPTEN_KEEPALIVE
  int fibonacci(int n) {
    if (n <= 1) return n;
    
    int a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
      int temp = a + b;
      a = b;
      b = temp;
    }
    return b;
  }
}
```

---

### 3. 编译

```bash
emcc fibonacci.cpp -o fibonacci.js \
  -s WASM=1 \
  -s EXPORTED_FUNCTIONS='["_fibonacci"]' \
  -s EXPORTED_RUNTIME_METHODS='["ccall","cwrap"]' \
  -O3
```

### 4. Node.js 调用

```javascript

  console.log(`fibonacci(40) = ${result}`);
```

```typescript
// 数组处理
export function sum(arr: Int32Array): i32 {
  let total: i32 = 0;
  for (let i = 0; i < arr.length; i++) {
```

## 🎯 WASI（WebAssembly System Interface）

### 1. WASI 简介

**WASI** 允许 WASM 访问系统资源（文件、网络、环境变量）

**支持：** Node.js 18+（实验性）

---

### 2. Rust WASI 示例

**Cargo.toml：**
```toml
[dependencies]
# 无需 wasm-bindgen
```

**src/main.rs：**
```rust
use std::fs;

fn main() {
    // 读取文件
    let content = fs::read_to_string("/app/input.txt")
        .expect("Failed to read file");
    
    println!("File content: {}", content);
    
    // 写入文件
    fs::write("/app/output.txt", "Hello from WASI!")
        .expect("Failed to write file");
}
```

**编译：**
```bash
rustup target add wasm32-wasi
cargo build --target wasm32-wasi --release
```

---

### 3. Node.js 运行 WASI

```javascript
import { readFile } from 'fs/promises';
import { WASI } from 'wasi';
import { argv, env } from 'process';

const wasi = new WASI({
  args: argv,
  env,
  preopens: {
    '/app': './data',  // 映射目录
  },
});

const wasm = await WebAssembly.compile(
  await readFile('./target/wasm32-wasi/release/my_app.wasm')
);

const instance = await WebAssembly.instantiate(wasm, wasi.getImportObject());

wasi.start(instance);
```

## 🎯 性能优化

### 1. SIMD（Single Instruction Multiple Data）

**Rust SIMD：**
```rust
use std::simd::f32x4;

#[wasm_bindgen]
pub fn dot_product_simd(a: &[f32], b: &[f32]) -> f32 {
    let mut sum = f32x4::splat(0.0);
    
    for i in (0..a.len()).step_by(4) {
        let va = f32x4::from_slice(&a[i..i+4]);
        let vb = f32x4::from_slice(&b[i..i+4]);
        sum += va * vb;
    }
    
    sum.horizontal_sum()
}
```

### 3. 内存管理

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub struct LargeData {
    data: Vec<u8>,
}

#[wasm_bindgen]
impl LargeData {
    #[wasm_bindgen(constructor)]
    pub fn new(size: usize) -> Self {
        LargeData {
            data: vec![0; size],

    }

    }
    
    pub fn len(&self) -> usize {

    }
}
```

## ⚠️ 三个高频错误

### 错误 1：忽略 WASM 初始化

```javascript
// ❌ 错误
import { fibonacci } from './pkg/fibonacci_wasm.js';
const result = fibonacci(40);  // 可能未初始化

// ✅ 正确
import init, { fibonacci } from './pkg/fibonacci_wasm.js';

await init();  // 等待初始化
const result = fibonacci(40);
```

---

### 错误 2：频繁跨语言调用

```javascript
// ❌ 错误：每次迭代都调用 WASM
for (let i = 0; i < 1000000; i++) {
  wasmModule.increment(i);  // 跨语言调用开销大
}

// ✅ 正确：批量处理
const arr = new Int32Array(1000000);
for (let i = 0; i < arr.length; i++) {
  arr[i] = i;
}
wasmModule.process_batch(arr);  // 一次调用
```

---

### 错误 3：内存泄漏

```javascript
// ❌ 错误：未释放 WASM 对象
function process() {
  const data = new WasmData(1024 * 1024);  // 1MB
  data.process();
  // 忘记释放 data.free()
}

// ✅ 正确
function process() {
  const data = new WasmData(1024 * 1024);
  try {
    data.process();
  } finally {
    data.free();  // 释放内存
  }
}
```

## 📚 FAQ

### 1. WASM 适合哪些场景？

**适合：**
- 计算密集型（加密、压缩、图像处理）
- 性能关键路径（游戏引擎、音视频编解码）
- 跨平台代码复用（C/C++ 库移植）

**不适合：**
- DOM 操作（JavaScript 更快）
- I/O 密集型（Node.js 异步更好）
- 简单逻辑（WASM 调用开销）

### 2. Rust vs C++ vs AssemblyScript？

| 语言 | 优势 | 劣势 |
|------|------|------|
| **Rust** | 内存安全、现代工具链、wasm-bindgen | 学习曲线陡峭 |
| **C++** | 成熟生态、性能极致 | 内存不安全、工具链复杂 |
| **AssemblyScript** | TypeScript 语法、易上手 | 生态较小、性能略低 |

### 3. WASM 文件如何缓存？

```javascript
import { readFile, writeFile } from 'fs/promises';

async function loadWASM() {
  const cacheFile = './cache/fibonacci.wasm';
  
  try {
    const cached = await readFile(cacheFile);
    return await WebAssembly.compile(cached);
  } catch {
    const response = await fetch('https://cdn.example.com/fibonacci.wasm');
    const bytes = await response.arrayBuffer();
    await writeFile(cacheFile, Buffer.from(bytes));
    return await WebAssembly.compile(bytes);
  }
}
```
