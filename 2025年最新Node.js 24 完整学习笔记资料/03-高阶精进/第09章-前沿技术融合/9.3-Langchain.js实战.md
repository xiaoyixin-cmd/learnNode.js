# 9.3 Langchain.js 实战

> 使用 Langchain.js 构建 LLM 驱动的智能应用

## 📚 知识导读

### 什么是 Langchain.js？

**定义：**  
Langchain.js 是一个用于构建 LLM（大语言模型）应用的框架，提供了：
- **链式调用**：组合多个 LLM 步骤
- **记忆管理**：上下文保持
- **数据连接**：向量数据库、文档加载器
- **Agent 代理**：自主决策和工具调用

**核心概念：**
- **Model**：LLM（OpenAI/Anthropic/本地模型）
- **Prompt Template**：提示词模板
- **Chain**：调用链
- **Agent**：智能代理
- **Memory**：对话历史
- **Vector Store**：向量存储

### 1. LLMChain

```javascript
import { LLMChain } from 'langchain/chains';
import { ChatOpenAI } from '@langchain/openai';
import { PromptTemplate } from '@langchain/core/prompts';

// 创建链

```

### 2. Sequential Chain（顺序链）

```javascript
import { SequentialChain } from 'langchain/chains';

// 链1：生成标题
const titleChain = new LLMChain({
  llm: model,
  prompt: PromptTemplate.fromTemplate('为{topic}写一个文章标题'),
  outputKey: 'title',
});

// 链2：根据标题写大纲
const outlineChain = new LLMChain({
  llm: model,
  prompt: PromptTemplate.fromTemplate(
    '为标题"{title}"写一个详细大纲'
  ),
  outputKey: 'outline',
});

// 链3：根据大纲写文章
const articleChain = new LLMChain({
  llm: model,
  prompt: PromptTemplate.fromTemplate(
    '根据大纲：\n{outline}\n\n写一篇完整文章'
  ),
  outputKey: 'article',
});

// 组合链
const overallChain = new SequentialChain({
  chains: [titleChain, outlineChain, articleChain],
  inputVariables: ['topic'],
  outputVariables: ['title', 'outline', 'article'],
});

// 执行
const result = await overallChain.call({
  topic: 'Node.js性能优化',
});

console.log('标题:', result.title);
console.log('大纲:', result.outline);
console.log('文章:', result.article);
```

## 🎯 Memory 记忆管理

### 1. BufferMemory（缓冲记忆）

```javascript
import { BufferMemory } from 'langchain/memory';
import { ConversationChain } from 'langchain/chains';

// 创建记忆
const memory = new BufferMemory();

// 创建对话链
const chain = new ConversationChain({

// 查看历史

// {
//   history: [
//     { role: 'user', content: '我叫张三' },
//     { role: 'assistant', content: '你好，张三！...' },
//     ...
//   ]
// }
```

### 2. BufferWindowMemory（窗口记忆）

## 🎯 Agent 智能代理

### 1. Tool 定义

```javascript
import { DynamicTool } from '@langchain/core/tools';
import axios from 'axios';

// 天气查询工具
const weatherTool = new DynamicTool({
  name: 'get_weather',
  description: '查询指定城市的天气，输入城市名称（如"北京"）',
  func: async (city) => {
    const response = await axios.get(
      `https://api.weatherapi.com/v1/current.json?key=${API_KEY}&q=${city}`
    );
    const { temp_c, condition } = response.data.current;
    return `${city}当前温度${temp_c}°C，天气${condition.text}`;
  },
});

// 计算器工具
const calculatorTool = new DynamicTool({
  name: 'calculator',
  description: '执行数学计算，输入表达式（如"2+3*4"）',
  func: async (expression) => {
    try {
      // 注意：生产环境需要安全的eval替代方案
      const result = eval(expression);
      return `计算结果：${result}`;
    } catch (error) {
      return '计算错误';
    }
  },
});

// 数据库查询工具
const dbQueryTool = new DynamicTool({
  name: 'query_database',
  description: '查询数据库用户信息，输入用户ID',
  func: async (userId) => {
    const user = await User.findById(userId);
    if (!user) return '用户不存在';
    return `用户：${user.name}，邮箱：${user.email}`;
  },
});
```

### 2. Agent 执行器

```javascript
import { AgentExecutor, createOpenAIFunctionsAgent } from 'langchain/agents';
import { ChatOpenAI } from '@langchain/openai';

// 创建 Agent
const model = new ChatOpenAI({ modelName: 'gpt-4-turbo', temperature: 0 });

const agent = await createOpenAIFunctionsAgent({
  llm: model,
  tools: [weatherTool, calculatorTool, dbQueryTool],
  prompt: ChatPromptTemplate.fromMessages([
    ['system', '你是一个智能助手，可以使用工具回答问题。'],
    ['placeholder', '{chat_history}'],
    ['human', '{input}'],
    ['placeholder', '{agent_scratchpad}'],
  ]),
});

// 执行器
const executor = new AgentExecutor({
  agent,
  tools: [weatherTool, calculatorTool, dbQueryTool],
  verbose: true,  // 显示执行过程
});

// 测试
const result = await executor.invoke({
  input: '北京今天天气怎么样？如果温度低于10度，计算10-温度的差值。',
});

console.log(result.output);
// 1. 调用 get_weather("北京") → "北京当前温度8°C，天气晴"
// 2. 调用 calculator("10-8") → "计算结果：2"
// 3. 返回："北京今天天气晴，温度8°C，比10度低2度。"
```

### 3. 自定义 Agent

```javascript
import { BaseAgent } from 'langchain/agents';

class CustomAgent extends BaseAgent {
  async plan(steps, inputs) {
    // 自定义决策逻辑
    const { input } = inputs;
    
    if (input.includes('天气')) {
      return {
        tool: 'get_weather',
        toolInput: this.extractCity(input),
      };
    } else if (input.includes('计算')) {
      return {
        tool: 'calculator',
        toolInput: this.extractExpression(input),
      };
    } else {
      return { returnValues: { output: '我不知道如何处理这个问题' } };
    }
  }
  
  extractCity(text) {
    const match = text.match(/(\w+)天气/);
    return match ? match[1] : '北京';
  }
  
  extractExpression(text) {
    const match = text.match(/计算(.+)/);
    return match ? match[1].trim() : '';
  }
}

const customAgent = new CustomAgent({
  llm: model,
  tools: [weatherTool, calculatorTool],
});

const executor = new AgentExecutor({ agent: customAgent, tools });
const result = await executor.invoke({ input: '上海天气如何？' });
```

## 🎯 向量数据库 & RAG

### 1. 文档加载

```javascript
import { PDFLoader } from 'langchain/document_loaders/fs/pdf';
import { TextLoader } from 'langchain/document_loaders/fs/text';
import { RecursiveCharacterTextSplitter } from 'langchain/text_splitter';

// 加载 PDF
const loader = new PDFLoader('./node-guide.pdf');
const docs = await loader.load();

// 分割文本（避免超过 token 限制）
const splitter = new RecursiveCharacterTextSplitter({
  chunkSize: 1000,
  chunkOverlap: 200,
});

const splitDocs = await splitter.splitDocuments(docs);
console.log(`分割成${splitDocs.length}个文档块`);
```

### 2. 向量存储（Pinecone）

```javascript
import { Pinecone } from '@pinecone-database/pinecone';
import { OpenAIEmbeddings } from '@langchain/openai';
import { PineconeStore } from '@langchain/pinecone';

// 初始化 Pinecone
const pinecone = new Pinecone({
  apiKey: process.env.PINECONE_API_KEY,
});
const index = pinecone.index('node-docs');

// 创建嵌入模型
const embeddings = new OpenAIEmbeddings({
  openAIApiKey: process.env.OPENAI_API_KEY,
});

// 存储文档
await PineconeStore.fromDocuments(splitDocs, embeddings, {
  pineconeIndex: index,
  namespace: 'node-guide',
});

console.log('文档已存储到向量数据库');
```

### 2. Express SSE 流式响应

```javascript
import express from 'express';
import { ChatOpenAI } from '@langchain/openai';

const app = express();

app.get('/stream', async (req, res) => {
  res.setHeader('Content-Type', 'text/event-stream');
  res.setHeader('Cache-Control', 'no-cache');
  res.setHeader('Connection', 'keep-alive');

    streaming: true,

      },
    ],

  res.write('data: [DONE]\n\n');
  res.end();
});

app.listen(3000);
// 访问: http://localhost:3000/stream?question=什么是事件循环
```

## ⚠️ 三个高频错误

### 错误 1：忘记处理 Token 超限

```javascript
// ❌ 错误：长文本直接输入
const result = await model.invoke(veryLongText);  // 可能超过4096 tokens

// ✅ 正确：分割 + 总结
const splitter = new RecursiveCharacterTextSplitter({ chunkSize: 2000 });
const chunks = await splitter.splitText(veryLongText);

const summaries = await Promise.all(
  chunks.map(chunk => model.invoke(`总结：${chunk}`))
);

const finalSummary = await model.invoke(
  `综合以下摘要：\n${summaries.join('\n')}`
);
```

## 📚 FAQ

### 1. 如何降低 OpenAI API 成本？

**策略：**
1. 使用 `gpt-3.5-turbo`（便宜10倍）
2. 缓存常见问题答案
3. 实现 RAG 减少上下文长度
4. 使用 Function Calling 减少 Token

### 2. 如何保护 API Key？

```javascript
// ❌ 错误：硬编码
const model = new ChatOpenAI({
  openAIApiKey: 'sk-xxxxx',
});

// ✅ 正确：环境变量
const model = new ChatOpenAI({
  openAIApiKey: process.env.OPENAI_API_KEY,
});

// ✅ 更安全：代理服务器
const model = new ChatOpenAI({
  configuration: {
    basePath: 'https://my-proxy.com/v1',
  },
});
```

### 3. 本地模型 vs 云端 API？

| 场景 | 推荐方案 |
|------|----------|
| 高隐私要求 | Ollama（本地LLaMA） |
| 高性能要求 | OpenAI GPT-4 |
