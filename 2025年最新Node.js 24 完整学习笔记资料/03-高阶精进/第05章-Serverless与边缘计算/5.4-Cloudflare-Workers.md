## 📚 知识导读

### 什么是 Cloudflare Workers？

**Cloudflare Workers** 是运行在 Cloudflare 全球边缘网络（300+ 数据中心）上的无服务器脚本，使用 **V8 Isolate** 技术，冷启动时间 < 5ms，是全球最快的边缘计算平台之一。

**核心优势：**

```

├────────────────────────────────────────────────────────┤
│                                                        │

│     - 使用 V8 Isolate（比容器快 100 倍）                │

│     - 支持 Web 标准 API                                │
```

## 💻 Workers KV 实战

### 什么是 Workers KV？

**Workers KV** 是 Cloudflare 提供的全球分布式键值存储，数据自动复制到所有边缘节点。

**特点：**
- **最终一致性**：写入后 60 秒内全球同步
- **高性能读取**：< 10ms
- **大容量**：单个命名空间最多 100GB
- **适用场景**：配置数据、静态资产、缓存、CDN 内容

---

### 1. 创建 KV 命名空间

```bash
# 创建生产环境 KV
wrangler kv:namespace create "MY_KV"

# 创建预览环境 KV（用于开发）
wrangler kv:namespace create "MY_KV" --preview
```

**配置 `wrangler.toml`：**

```toml
name = "my-worker"
main = "src/index.js"
compatibility_date = "2025-01-01"

[[kv_namespaces]]
binding = "MY_KV"
id = "your-namespace-id"
preview_id = "your-preview-namespace-id"
```

---

### 2. KV 基本操作

```javascript
export default {
  async fetch(request, env, ctx) {
    const url = new URL(request.url);

    // 1. 写入数据
    if (url.pathname === '/set') {
      await env.MY_KV.put('username', 'alice');
      await env.MY_KV.put('user:123', JSON.stringify({ id: 123, name: 'Alice' }));

      // 设置过期时间（3600 秒后过期）
      await env.MY_KV.put('session:abc', 'session-data', { expirationTtl: 3600 });

      return new Response('Data saved');
    }

    // 2. 读取数据
    if (url.pathname === '/get') {
      const username = await env.MY_KV.get('username');
      const user = await env.MY_KV.get('user:123', { type: 'json' });

      return new Response(JSON.stringify({ username, user }), {
        headers: { 'Content-Type': 'application/json' },
      });
    }

    // 3. 删除数据
    if (url.pathname === '/delete') {
      await env.MY_KV.delete('username');
      return new Response('Data deleted');
    }

    // 4. 列出所有键（分页）
    if (url.pathname === '/list') {
      const { keys } = await env.MY_KV.list({ prefix: 'user:', limit: 10 });
      return new Response(JSON.stringify(keys), {
        headers: { 'Content-Type': 'application/json' },
      });
    }

    return new Response('Not Found', { status: 404 });
  },
};
```

### 3. 实战案例：URL 短链接服务

```javascript
// src/index.js
import { Hono } from 'hono';
import { nanoid } from 'nanoid';

const app = new Hono();

// 创建短链接
app.post('/shorten', async (c) => {
  const { url } = await c.req.json();

  // 验证 URL
  try {
    new URL(url);
  } catch {
    return c.json({ error: 'Invalid URL' }, 400);
  }

  // 生成短码
  const shortCode = nanoid(8);

  // 保存到 KV
  await c.env.MY_KV.put(`url:${shortCode}`, url);

  return c.json({
    shortCode,
    shortUrl: `https://short.example.com/${shortCode}`,
  });
});

// 重定向到原始 URL
app.get('/:shortCode', async (c) => {
  const shortCode = c.req.param('shortCode');

  // 从 KV 读取原始 URL
  const url = await c.env.MY_KV.get(`url:${shortCode}`);

  if (!url) {
    return c.json({ error: 'Short URL not found' }, 404);
  }

  // 重定向
  return c.redirect(url, 301);
});

export default app;
```

## 💻 Durable Objects 实战

### 什么是 Durable Objects？

**Durable Objects** 是 Cloudflare Workers 的有状态存储层，提供：
- **强一致性**：单实例保证数据一致性
- **WebSocket 支持**：实时双向通信
- **自动持久化**：状态自动保存到磁盘
- **全球分布**：根据 ID 自动选择最近的数据中心

**适用场景：**
- 聊天室
- 协作编辑（如 Google Docs）
- 实时游戏
- 购物车状态

---

### 1. 创建 Durable Object

```javascript
// src/index.js
export class Counter {
  constructor(state, env) {
    this.state = state;
    this.env = env;
  }

  async fetch(request) {
    const url = new URL(request.url);

    // 初始化计数器
    let count = (await this.state.storage.get('count')) || 0;

    if (url.pathname === '/increment') {
      count++;
      await this.state.storage.put('count', count);
      return new Response(String(count));
    }

    if (url.pathname === '/get') {
      return new Response(String(count));
    }

    return new Response('Not Found', { status: 404 });
  }
}

// Worker 入口
export default {
  async fetch(request, env, ctx) {
    // 创建或获取 Durable Object 实例
    const id = env.COUNTER.idFromName('global-counter');
    const stub = env.COUNTER.get(id);

    // 将请求转发给 Durable Object
    return stub.fetch(request);
  },
};
```

**配置 `wrangler.toml`：**

```toml

main = "src/index.js"
compatibility_date = "2025-01-01"

[[durable_objects.bindings]]
name = "COUNTER"
class_name = "Counter"
script_name = "my-worker"

[[migrations]]
tag = "v1"
new_classes = ["Counter"]
```

### 2. 实战案例：实时聊天室

```javascript
// src/chatroom.js
export class ChatRoom {
  constructor(state, env) {
    this.state = state;
    this.env = env;
    this.sessions = new Set();  // 存储所有 WebSocket 连接
  }

  async fetch(request) {
    const url = new URL(request.url);

    // WebSocket 升级
    if (request.headers.get('Upgrade') === 'websocket') {
      const pair = new WebSocketPair();
      const [client, server] = Object.values(pair);

      await this.handleSession(server);

      return new Response(null, {
        status: 101,
        webSocket: client,
      });
    }

    // HTTP API：获取聊天历史
    if (url.pathname === '/history') {
      const messages = (await this.state.storage.get('messages')) || [];
      return new Response(JSON.stringify(messages), {
        headers: { 'Content-Type': 'application/json' },
      });
    }

    return new Response('Not Found', { status: 404 });
  }

  async handleSession(webSocket) {
    webSocket.accept();

    // 添加到会话集合
    this.sessions.add(webSocket);

    // 监听消息
    webSocket.addEventListener('message', async (event) => {
      try {
        const message = JSON.parse(event.data);

        // 保存消息到存储
        const messages = (await this.state.storage.get('messages')) || [];
        messages.push({
          ...message,
          timestamp: Date.now(),
        });
        await this.state.storage.put('messages', messages);

        // 广播给所有客户端
        this.broadcast(JSON.stringify(message));
      } catch (error) {
        console.error('Message error:', error);
      }
    });

    // 监听关闭
    webSocket.addEventListener('close', () => {
      this.sessions.delete(webSocket);
    });

    // 发送欢迎消息
    webSocket.send(JSON.stringify({ type: 'welcome', text: 'Welcome to the chat!' }));
  }

  broadcast(message) {
    for (const session of this.sessions) {
      try {
        session.send(message);
      } catch (error) {
        // 连接已关闭，移除
        this.sessions.delete(session);
      }
    }
  }
}

// Worker 入口
export default {
  async fetch(request, env, ctx) {
    const url = new URL(request.url);

    // 从路径获取房间 ID
    const roomId = url.pathname.split('/')[2] || 'default';

    // 创建或获取房间实例
    const id = env.CHATROOM.idFromName(roomId);
    const room = env.CHATROOM.get(id);

    return room.fetch(request);
  },
};
```

```toml
[[migrations]]
tag = "v1"
new_classes = ["ChatRoom"]
```

## 💻 R2 Storage 实战

### 什么是 R2？

**R2** 是 Cloudflare 的对象存储服务，S3 兼容 API，**零出站费用**（相比 S3 节省 90%+ 成本）。

**适用场景：**
- 图片/视频存储
- 备份文件
- 静态网站托管

### 2. R2 基本操作

```javascript
export default {
  async fetch(request, env, ctx) {
    const url = new URL(request.url);

    // 1. 上传文件
    if (url.pathname === '/upload' && request.method === 'POST') {
      const formData = await request.formData();
      const file = formData.get('file');

      await env.MY_BUCKET.put(file.name, file.stream(), {
        httpMetadata: {
          contentType: file.type,
        },
      });

      return new Response('File uploaded');
    }

    // 2. 下载文件
    if (url.pathname.startsWith('/download/')) {
      const fileName = url.pathname.split('/download/')[1];
      const object = await env.MY_BUCKET.get(fileName);

      if (!object) {
        return new Response('File not found', { status: 404 });
      }

      return new Response(object.body, {
        headers: {
          'Content-Type': object.httpMetadata.contentType,
          'Content-Length': object.size,
        },
      });
    }

    // 3. 列出文件
    if (url.pathname === '/list') {
      const { objects } = await env.MY_BUCKET.list();
      return new Response(JSON.stringify(objects), {
        headers: { 'Content-Type': 'application/json' },
      });
    }

    // 4. 删除文件
    if (url.pathname.startsWith('/delete/')) {
      const fileName = url.pathname.split('/delete/')[1];
      await env.MY_BUCKET.delete(fileName);
      return new Response('File deleted');
    }

    return new Response('Not Found', { status: 404 });
  },
};
```

### 3. 实战案例：图片 CDN

```javascript
import { Hono } from 'hono';

const app = new Hono();

// 上传图片
app.post('/upload', async (c) => {
  const formData = await c.req.formData();
  const file = formData.get('image');

  if (!file || !file.type.startsWith('image/')) {
    return c.json({ error: 'Invalid image file' }, 400);

    success: true,
    url: `https://your-worker.workers.dev/images/${fileName}`,

app.get('/images/:fileName', async (c) => {
  const fileName = c.req.param('fileName');

  const object = await c.env.MY_BUCKET.get(fileName);

  if (!object) {
    return c.json({ error: 'Image not found' }, 404);
  }

  return new Response(object.body, {
    headers: {
      'Content-Type': object.httpMetadata.contentType,
      'Cache-Control': 'public, max-age=31536000',  // 缓存 1 年
      'ETag': object.etag,
    },
  });
});
```

## ⚠️ 三个高频错误

### 错误 1：超过 CPU 时间限制

**问题：** Workers 免费版 CPU 时间限制 10ms，付费版 50ms。

```javascript
// ❌ 错误：CPU 密集型计算
export default {
  async fetch(request, env, ctx) {
    let sum = 0;
    for (let i = 0; i < 1000000000; i++) {  // 超时！
      sum += i;
    }
    return new Response(String(sum));
  },
};
```

**正确做法：** 使用异步操作或分批处理。

```javascript
// ✅ 正确：异步操作
export default {
  async fetch(request, env, ctx) {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    return new Response(JSON.stringify(data), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
};
```

---

### 错误 2：KV 写入后立即读取不到

**问题：** KV 是最终一致性，写入后需要 60 秒才能全球同步。

```javascript
// ❌ 错误：立即读取
await env.MY_KV.put('key', 'value');
const value = await env.MY_KV.get('key');  // 可能读取不到
```

**正确做法：** 使用 Durable Objects 或在写入后缓存在内存。

```javascript
// ✅ 正确：使用缓存
const cache = new Map();

await env.MY_KV.put('key', 'value');
cache.set('key', 'value');  // 立即可用

const value = cache.get('key') || await env.MY_KV.get('key');
```

---

### 错误 3：忘记处理 WebSocket 错误

```javascript
// ❌ 错误：未处理错误
webSocket.send(message);  // 可能连接已关闭

// ✅ 正确：捕获错误
try {
  webSocket.send(message);
} catch (error) {
  console.error('WebSocket send error:', error);
  sessions.delete(webSocket);
}
```

## 📚 FAQ

### 1. Workers vs Lambda vs Vercel Edge Functions？

| 特性 | Cloudflare Workers | AWS Lambda | Vercel Edge |
|------|-------------------|------------|-------------|
| 冷启动 | <5ms | 100-500ms | <10ms |
| 执行时间 | 10ms/50ms (CPU) | 最长 15 分钟 | 最长 30 秒 |
| 定价 | 免费 10万请求/天 | 免费 100万请求/月 | 免费 10万请求/月 |
| 数据中心 | 300+ | 30+ | 300+ |

### 2. KV vs Durable Objects 如何选择？

| 特性 | Workers KV | Durable Objects |
|------|-----------|----------------|
| 一致性 | 最终一致性 | 强一致性 |
| 延迟 | <10ms | <50ms |
| 适用场景 | 配置、缓存 | 聊天室、协作 |
