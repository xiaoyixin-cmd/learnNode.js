## 📚 核心知识

### 1. gRPC 简介

**gRPC** 是 Google 开源的高性能 RPC（Remote Procedure Call）框架，基于 HTTP/2 协议和 Protocol Buffers 序列化。

**核心优势：**

```
┌─────────────────────────────────────────────────────────┐
│                    gRPC 架构图                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   客户端                            服务端               │
│   ┌──────────┐                    ┌──────────┐         │
│   │  Stub    │ ──── HTTP/2 ────>  │  Service │         │
│   │ (代理类)  │  Protobuf 编码      │   实现   │         │
│   └──────────┘                    └──────────┘         │
│        ↑                                ↑               │
│        │                                │               │
│   ┌────┴─────┐                    ┌────┴─────┐         │
│   │  .proto  │ ←──── 代码生成 ──── │  .proto  │         │
│   │   文件   │                    │   文件   │         │
│   └──────────┘                    └──────────┘         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**与 REST 对比：**

| 特性 | REST | gRPC |
|------|------|------|
| **协议** | HTTP/1.1 | HTTP/2 |
| **数据格式** | JSON（文本） | Protobuf（二进制） |
| **性能** | 较慢（文本解析） | 快（二进制+多路复用） |
| **类型安全** | 弱（运行时检查） | 强（编译时生成代码） |
| **流式传输** | 不支持 | 支持（4种模式） |
| **浏览器支持** | 原生支持 | 需要 gRPC-Web |
| **调试** | 简单（curl、浏览器） | 困难（需专用工具） |

### 2. Protocol Buffers 3.0

**Protocol Buffers（protobuf）** 是 Google 的数据序列化格式，比 JSON 更高效。

**核心概念：**

```protobuf
// user.proto
syntax = "proto3";  // 使用 proto3 语法

package user;       // 包名（避免命名冲突）

// 消息定义（类似于 TypeScript 的 interface）
message User {
  string id = 1;           // 字段编号（不能修改）
  string name = 2;
  string email = 3;
  int32 age = 4;
  repeated string tags = 5;  // 数组
  Address address = 6;       // 嵌套消息
}

message Address {
  string city = 1;
  string street = 2;
}

// 服务定义（类似于 API 接口）
service UserService {
  rpc GetUser (GetUserRequest) returns (User);
  rpc ListUsers (ListUsersRequest) returns (stream User);  // 服务端流
}

message GetUserRequest {
  string id = 1;
}

message ListUsersRequest {
  int32 page = 1;
  int32 pageSize = 2;
}
```

**字段类型对照表：**

| Protobuf 类型 | Node.js 类型 | 说明 |
|--------------|-------------|------|
| `string` | `string` | UTF-8 字符串 |
| `int32`, `int64` | `number`, `string` | 整数 |
| `float`, `double` | `number` | 浮点数 |
| `bool` | `boolean` | 布尔值 |
| `bytes` | `Buffer` | 二进制数据 |
| `repeated T` | `Array<T>` | 数组 |
| `map<K, V>` | `Map<K, V>` | 字典 |

### 3. gRPC 四种通信模式

```protobuf
service ChatService {
  // 1. Unary（一元调用）：一个请求，一个响应
  rpc GetMessage (MessageRequest) returns (Message);
  
  // 2. Server Streaming（服务端流）：一个请求，多个响应
  rpc StreamMessages (StreamRequest) returns (stream Message);
  
  // 3. Client Streaming（客户端流）：多个请求，一个响应
  rpc UploadMessages (stream Message) returns (UploadResult);
  
  // 4. Bidirectional Streaming（双向流）：多个请求，多个响应
  rpc Chat (stream Message) returns (stream Message);
}
```

**使用场景：**

| 模式 | 适用场景 | 示例 |
|------|---------|------|
| **Unary** | 普通 CRUD 操作 | 查询用户信息 |
| **Server Streaming** | 实时推送 | 股票行情、日志流 |
| **Client Streaming** | 批量上传 | 文件上传、批量导入 |
| **Bidirectional** | 实时双向通信 | 聊天室、游戏对战 |

### 实战1：Unary 模式 - 用户管理服务

#### 步骤1：定义 .proto 文件

```protobuf
// protos/user.proto
syntax = "proto3";

package user;

service UserService {
  rpc CreateUser (CreateUserRequest) returns (User);
  rpc GetUser (GetUserRequest) returns (User);
  rpc UpdateUser (UpdateUserRequest) returns (User);
  rpc DeleteUser (DeleteUserRequest) returns (DeleteUserResponse);
}

message User {
  string id = 1;
  string name = 2;
  string email = 3;
  int32 age = 4;
  int64 created_at = 5;  // Unix 时间戳（毫秒）
}

message CreateUserRequest {
  string name = 1;
  string email = 2;
  int32 age = 3;
}

message GetUserRequest {
  string id = 1;
}

message UpdateUserRequest {
  string id = 1;
  string name = 2;
  string email = 3;
  int32 age = 4;
}

message DeleteUserRequest {
  string id = 1;
}

message DeleteUserResponse {
  bool success = 1;
  string message = 2;
}
```

#### 步骤2：实现服务端

```javascript
// src/server/user-service.js
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/user.proto');

// 加载 proto 文件
const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,         // 保持字段名大小写
  longs: String,          // 将 int64 转为 string（避免精度丢失）
  enums: String,          // 枚举转为 string
  defaults: true,         // 设置默认值
  oneofs: true,           // 支持 oneof
});

const userProto = grpc.loadPackageDefinition(packageDefinition).user;

// 模拟数据库
const users = new Map();
let nextId = 1;

// 实现服务方法
const userService = {
  // 创建用户
  CreateUser: (call, callback) => {
    const { name, email, age } = call.request;
    
    // 参数校验
    if (!name || !email) {
      return callback({
        code: grpc.status.INVALID_ARGUMENT,
        message: 'Name and email are required',
      });
    }
    
    // 检查邮箱是否已存在
    for (const user of users.values()) {
      if (user.email === email) {
        return callback({
          code: grpc.status.ALREADY_EXISTS,
          message: 'Email already exists',
        });
      }
    }
    
    // 创建用户
    const user = {
      id: String(nextId++),
      name,
      email,
      age: age || 0,
      created_at: Date.now(),
    };
    
    users.set(user.id, user);
    
    console.log(`✅ User created: ${user.id} - ${user.name}`);
    
    callback(null, user);
  },
  
  // 查询用户
  GetUser: (call, callback) => {
    const { id } = call.request;
    
    const user = users.get(id);
    
    if (!user) {
      return callback({
        code: grpc.status.NOT_FOUND,
        message: `User ${id} not found`,
      });
    }
    
    callback(null, user);
  },
  
  // 更新用户
  UpdateUser: (call, callback) => {
    const { id, name, email, age } = call.request;
    
    const user = users.get(id);
    
    if (!user) {
      return callback({
        code: grpc.status.NOT_FOUND,
        message: `User ${id} not found`,
      });
    }
    
    // 更新字段
    if (name) user.name = name;
    if (email) user.email = email;
    if (age !== undefined) user.age = age;
    
    console.log(`✅ User updated: ${user.id} - ${user.name}`);
    
    callback(null, user);
  },
  
  // 删除用户
  DeleteUser: (call, callback) => {
    const { id } = call.request;
    
    const deleted = users.delete(id);
    
    if (!deleted) {
      return callback({
        code: grpc.status.NOT_FOUND,
        message: `User ${id} not found`,
      });
    }
    
    console.log(`✅ User deleted: ${id}`);
    
    callback(null, {
      success: true,
      message: 'User deleted successfully',
    });
  },
};

// 启动服务器
function startServer() {
  const server = new grpc.Server();
  
  // 注册服务
  server.addService(userProto.UserService.service, userService);
  
  // 绑定端口（不安全模式，生产环境需要使用 TLS）
  server.bindAsync(
    '0.0.0.0:50051',
    grpc.ServerCredentials.createInsecure(),
    (err, port) => {
      if (err) {
        console.error('Failed to bind server:', err);
        return;
      }
      
      console.log(`🚀 gRPC Server running on port ${port}`);
      server.start();
    }
  );
}

startServer();
```

```javascript
// src/client/user-client.js
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/user.proto');

// 加载 proto 文件
const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const userProto = grpc.loadPackageDefinition(packageDefinition).user;

// 创建客户端
const client = new userProto.UserService(
  'localhost:50051',
  grpc.credentials.createInsecure()
);

// 封装为 Promise
function createUser(name, email, age) {
  return new Promise((resolve, reject) => {
    client.CreateUser({ name, email, age }, (err, response) => {
      if (err) reject(err);
      else resolve(response);
    });
  });
}

function getUser(id) {
  return new Promise((resolve, reject) => {
    client.GetUser({ id }, (err, response) => {
      if (err) reject(err);
      else resolve(response);
    });
  });
}

function updateUser(id, updates) {
  return new Promise((resolve, reject) => {
    client.UpdateUser({ id, ...updates }, (err, response) => {
      if (err) reject(err);
      else resolve(response);
    });
  });
}

function deleteUser(id) {
  return new Promise((resolve, reject) => {
    client.DeleteUser({ id }, (err, response) => {
      if (err) reject(err);
      else resolve(response);
    });
  });
}

// 测试
async function main() {
  try {
    // 1. 创建用户
    console.log('📝 Creating user...');
    const user = await createUser('Alice', 'alice@example.com', 25);
    console.log('✅ User created:', user);
    
    // 2. 查询用户
    console.log('\n📖 Getting user...');
    const fetchedUser = await getUser(user.id);
    console.log('✅ User fetched:', fetchedUser);
    
    // 3. 更新用户
    console.log('\n✏️ Updating user...');
    const updatedUser = await updateUser(user.id, { name: 'Alice Smith', age: 26 });
    console.log('✅ User updated:', updatedUser);
    
    // 4. 删除用户
    console.log('\n🗑️ Deleting user...');
    const deleteResult = await deleteUser(user.id);
    console.log('✅ User deleted:', deleteResult);
    
    // 5. 尝试查询已删除的用户
    console.log('\n📖 Getting deleted user...');
    try {
      await getUser(user.id);
    } catch (err) {
      console.log('❌ Expected error:', err.message);
    }
  } catch (error) {
    console.error('❌ Error:', error.message);
  }
}

main();
```

### 实战2：Server Streaming - 实时日志流

#### 步骤1：定义 .proto 文件

```protobuf
// protos/log.proto
syntax = "proto3";

package log;

service LogService {
  // 服务端流：客户端发送一次请求，服务端持续推送日志
  rpc StreamLogs (StreamLogsRequest) returns (stream LogEntry);
}

message StreamLogsRequest {
  string service_name = 1;
  string level = 2;  // "INFO", "WARN", "ERROR"
  int64 since = 3;   // Unix 时间戳（毫秒）
}

message LogEntry {
  string id = 1;
  string service_name = 2;
  string level = 3;
  string message = 4;
  int64 timestamp = 5;
}
```

#### 步骤2：实现服务端

```javascript
// src/server/log-service.js
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/log.proto');

const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const logProto = grpc.loadPackageDefinition(packageDefinition).log;

// 模拟日志生成器
function* generateLogs(serviceName, level, since) {
  let id = 1;
  const levels = level ? [level] : ['INFO', 'WARN', 'ERROR'];
  
  while (true) {
    const log = {
      id: String(id++),
      service_name: serviceName,
      level: levels[Math.floor(Math.random() * levels.length)],
      message: `Sample log message #${id}`,
      timestamp: String(Date.now()),
    };
    
    // 过滤时间戳
    if (since && Number(log.timestamp) < since) {
      continue;
    }
    
    yield log;
    
    // 模拟延迟（500ms-2s）
    const delay = 500 + Math.random() * 1500;
    Atomics.wait(new Int32Array(new SharedArrayBuffer(4)), 0, 0, delay);
  }
}

// 实现服务方法
const logService = {
  StreamLogs: (call) => {
    const { service_name, level, since } = call.request;
    
    console.log(`📡 Client connected for ${service_name} logs (level: ${level || 'ALL'})`);
    
    const logGenerator = generateLogs(service_name, level, Number(since));
    
    // 定时推送日志
    const interval = setInterval(() => {
      if (call.cancelled) {
        console.log('❌ Client disconnected');
        clearInterval(interval);
        return;
      }
      
      const { value: log } = logGenerator.next();
      
      // 写入日志到流
      call.write(log);
      
      console.log(`📤 Sent log: [${log.level}] ${log.message}`);
    }, 1000);  // 每秒发送一条日志
    
    // 监听客户端取消
    call.on('cancelled', () => {
      console.log('❌ Client cancelled connection');
      clearInterval(interval);
    });
    
    // 注意：Server Streaming 不应调用 call.end()，除非主动断开
    // call.end();  // ❌ 不要调用
  },
};

// 启动服务器
function startServer() {
  const server = new grpc.Server();
  server.addService(logProto.LogService.service, logService);
  
  server.bindAsync(
    '0.0.0.0:50052',
    grpc.ServerCredentials.createInsecure(),
    (err, port) => {
      if (err) {
        console.error('Failed to bind server:', err);
        return;
      }
      
      console.log(`🚀 Log Service running on port ${port}`);
      server.start();
    }
  );
}

startServer();
```

```javascript
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/log.proto');

const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const logProto = grpc.loadPackageDefinition(packageDefinition).log;

const client = new logProto.LogService(
  'localhost:50052',
  grpc.credentials.createInsecure()
);

// 订阅日志流
function streamLogs(serviceName, level = '', since = 0) {
  const call = client.StreamLogs({ service_name: serviceName, level, since });
  
  // 监听数据事件
  call.on('data', (log) => {
    const color = log.level === 'ERROR' ? '\x1b[31m' : log.level === 'WARN' ? '\x1b[33m' : '\x1b[32m';
    console.log(`${color}[${log.level}]\x1b[0m ${log.message} (${new Date(Number(log.timestamp)).toISOString()})`);
  });
  
  // 监听结束事件
  call.on('end', () => {
    console.log('✅ Stream ended');
  });
  
  // 监听错误事件
  call.on('error', (err) => {
    console.error('❌ Stream error:', err.message);
  });
  
  // 10秒后取消订阅
  setTimeout(() => {
    console.log('\n⏹️ Cancelling stream...');
    call.cancel();
  }, 10000);
}

// 测试
streamLogs('user-service', 'ERROR');  // 只订阅 ERROR 级别日志
```

### 实战3：Client Streaming - 批量文件上传

#### 步骤1：定义 .proto 文件

```protobuf
// protos/file.proto
syntax = "proto3";

package file;

service FileService {
  // 客户端流：客户端分块上传，服务端返回汇总结果
  rpc UploadFile (stream FileChunk) returns (UploadResult);
}

message FileChunk {
  string filename = 1;
  int32 chunk_index = 2;
  bytes data = 3;  // 二进制数据
}

message UploadResult {
  bool success = 1;
  string file_id = 2;
  int64 total_size = 3;
  int32 total_chunks = 4;
  string message = 5;
}
```

#### 步骤2：实现服务端

```javascript
// src/server/file-service.js
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import fs from 'fs/promises';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/file.proto');

const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const fileProto = grpc.loadPackageDefinition(packageDefinition).file;

// 上传目录
const UPLOAD_DIR = path.join(__dirname, '../../uploads');
await fs.mkdir(UPLOAD_DIR, { recursive: true });

const fileService = {
  UploadFile: (call, callback) => {
    let filename = '';
    const chunks = [];
    let totalSize = 0;
    
    // 监听客户端发送的数据块
    call.on('data', (chunk) => {
      if (!filename) {
        filename = chunk.filename;
        console.log(`📥 Receiving file: ${filename}`);
      }
      
      chunks.push(chunk);
      totalSize += chunk.data.length;
      
      console.log(`📦 Chunk ${chunk.chunk_index} received (${chunk.data.length} bytes)`);
    });
    
    // 客户端发送完毕
    call.on('end', async () => {
      try {
        // 排序并合并所有块
        chunks.sort((a, b) => a.chunk_index - b.chunk_index);
        const fileBuffer = Buffer.concat(chunks.map(c => c.data));
        
        // 保存文件
        const fileId = `${Date.now()}-${filename}`;
        const filePath = path.join(UPLOAD_DIR, fileId);
        await fs.writeFile(filePath, fileBuffer);
        
        console.log(`✅ File saved: ${filePath} (${totalSize} bytes)`);
        
        // 返回结果
        callback(null, {
          success: true,
          file_id: fileId,
          total_size: totalSize,
          total_chunks: chunks.length,
          message: 'File uploaded successfully',
        });
      } catch (error) {
        console.error('❌ Upload failed:', error);
        callback({
          code: grpc.status.INTERNAL,
          message: error.message,
        });
      }
    });
    
    // 错误处理
    call.on('error', (err) => {
      console.error('❌ Stream error:', err);
    });
  },
};

// 启动服务器
function startServer() {
  const server = new grpc.Server();
  server.addService(fileProto.FileService.service, fileService);
  
  server.bindAsync(
    '0.0.0.0:50053',
    grpc.ServerCredentials.createInsecure(),
    (err, port) => {
      if (err) {
        console.error('Failed to bind server:', err);
        return;
      }
      
      console.log(`🚀 File Service running on port ${port}`);
      server.start();
    }
  );
}

startServer();
```

#### 步骤3：实现客户端

```javascript
// src/client/file-client.js

  defaults: true,
  oneofs: true,

  grpc.credentials.createInsecure()

// 上传文件（分块）
async function uploadFile(filePath, chunkSize = 64 * 1024) {  // 64KB per chunk
  return new Promise(async (resolve, reject) => {
    const call = client.UploadFile((err, response) => {
      if (err) reject(err);
      else resolve(response);
    });
    
    const filename = path.basename(filePath);
    const fileBuffer = await fs.readFile(filePath);
    
    console.log(`📤 Uploading ${filename} (${fileBuffer.length} bytes)`);
    
    // 分块发送
    let chunkIndex = 0;
    for (let offset = 0; offset < fileBuffer.length; offset += chunkSize) {
      const chunk = fileBuffer.slice(offset, offset + chunkSize);
      
      call.write({
        filename,
        chunk_index: chunkIndex++,
        data: chunk,
      });
      
      console.log(`📦 Sent chunk ${chunkIndex - 1} (${chunk.length} bytes)`);
      
      // 模拟网络延迟
      await new Promise(r => setTimeout(r, 100));
    }
    
    // 通知服务端发送完毕
    call.end();
  });
}
```

### 实战4：Bidirectional Streaming - 实时聊天室

#### 步骤1：定义 .proto 文件

```protobuf
// protos/chat.proto
syntax = "proto3";

package chat;

service ChatService {
  // 双向流：客户端和服务端同时收发消息
  rpc Chat (stream ChatMessage) returns (stream ChatMessage);
}

message ChatMessage {
  string user_id = 1;
  string username = 2;
  string message = 3;
  int64 timestamp = 4;
  MessageType type = 5;
}

enum MessageType {
  TEXT = 0;
  JOIN = 1;
  LEAVE = 2;
  SYSTEM = 3;
}
```

#### 步骤2：实现服务端

```javascript
// src/server/chat-service.js
import grpc from '@grpc/grpc-js';
import protoLoader from '@grpc/proto-loader';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const PROTO_PATH = path.join(__dirname, '../../protos/chat.proto');

const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const chatProto = grpc.loadPackageDefinition(packageDefinition).chat;

// 存储所有连接的客户端
const clients = new Map();

const chatService = {
  Chat: (call) => {
    let userId = null;
    let username = null;
    
    // 监听客户端消息
    call.on('data', (message) => {
      if (!userId) {
        userId = message.user_id;
        username = message.username;
        clients.set(userId, call);
        
        console.log(`✅ ${username} joined the chat`);
        
        // 广播加入消息
        broadcast({
          user_id: 'system',
          username: 'System',
          message: `${username} joined the chat`,
          timestamp: String(Date.now()),
          type: 'JOIN',
        }, userId);
      } else {
        console.log(`💬 ${username}: ${message.message}`);
        
        // 广播消息给所有客户端（除了发送者）
        broadcast(message, userId);
      }
    });
    
    // 客户端断开连接
    call.on('end', () => {
      if (userId) {
        clients.delete(userId);
        console.log(`❌ ${username} left the chat`);
        
        // 广播离开消息
        broadcast({
          user_id: 'system',
          username: 'System',
          message: `${username} left the chat`,
          timestamp: String(Date.now()),
          type: 'LEAVE',
        });
      }
      
      call.end();
    });
    
    // 错误处理
    call.on('error', (err) => {
      console.error('❌ Stream error:', err);
      if (userId) {
        clients.delete(userId);
      }
    });
  },
};

// 广播消息
function broadcast(message, excludeUserId = null) {
  for (const [userId, client] of clients.entries()) {
    if (userId !== excludeUserId) {
      client.write(message);
    }
  }
}

// 启动服务器
function startServer() {
  const server = new grpc.Server();
  server.addService(chatProto.ChatService.service, chatService);
  
  server.bindAsync(
    '0.0.0.0:50054',
    grpc.ServerCredentials.createInsecure(),
    (err, port) => {
      if (err) {
        console.error('Failed to bind server:', err);
        return;
      }
      
      console.log(`🚀 Chat Service running on port ${port}`);
      server.start();
    }
  );
}

startServer();
```

### 场景1：微服务间高性能通信

**需求：**
- 电商系统中，订单服务需要频繁调用库存服务、支付服务
- 要求低延迟（<10ms）、高吞吐（10000+ QPS）

```javascript
// 订单服务调用库存服务（gRPC）
const inventoryClient = new inventoryProto.InventoryService(
  'inventory-service:50051',
  grpc.credentials.createInsecure()
);

// 批量检查库存（并发调用）
async function checkStocks(productIds) {
  const promises = productIds.map(id =>
    new Promise((resolve, reject) => {
      inventoryClient.GetStock({ product_id: id }, { deadline: Date.now() + 3000 }, (err, res) => {
        if (err) reject(err);
        else resolve(res);
      });
    })
  );
  
  return Promise.all(promises);
}

// 性能对比：
// REST: ~50ms per request × 10 products = 500ms
// gRPC: ~5ms per request × 10 products (并发) = ~10ms
```

## ❌ 高频错误

### 错误1：未处理 gRPC 错误码

**错误代码：**
```javascript
// ❌ 错误：直接捕获异常，丢失错误详情
try {
  const user = await getUser('invalid-id');
} catch (error) {
  console.error('Error:', error.message);  // 无法判断具体错误类型
  res.status(500).json({ error: 'Internal error' });
}
```

**正确做法：**
```javascript
// ✅ 根据 gRPC 状态码处理
import grpc from '@grpc/grpc-js';

try {
  const user = await getUser('invalid-id');
} catch (error) {
  // 判断错误类型
  switch (error.code) {
    case grpc.status.NOT_FOUND:
      res.status(404).json({ error: 'User not found' });
      break;
    case grpc.status.INVALID_ARGUMENT:
      res.status(400).json({ error: 'Invalid user ID' });
      break;
    case grpc.status.DEADLINE_EXCEEDED:
      res.status(504).json({ error: 'Service timeout' });
      break;
    case grpc.status.UNAVAILABLE:
      res.status(503).json({ error: 'Service unavailable' });
      break;
    default:
      res.status(500).json({ error: 'Internal error' });
  }
}
```

**gRPC 状态码对照表：**

| gRPC 状态码 | HTTP 状态码 | 说明 |
|------------|------------|------|
| OK | 200 | 成功 |
| INVALID_ARGUMENT | 400 | 参数错误 |
| NOT_FOUND | 404 | 资源不存在 |
| ALREADY_EXISTS | 409 | 资源已存在 |
| PERMISSION_DENIED | 403 | 权限不足 |
| UNAUTHENTICATED | 401 | 未认证 |
| RESOURCE_EXHAUSTED | 429 | 超过限流 |
| DEADLINE_EXCEEDED | 504 | 超时 |
| UNAVAILABLE | 503 | 服务不可用 |
| INTERNAL | 500 | 内部错误 |

### 错误2：流式传输未正确结束

**错误代码：**
```javascript
// ❌ 错误：Server Streaming 提前调用 end()
StreamLogs: (call) => {
  const logs = generateLogs();
  
  for (const log of logs) {
    call.write(log);
  }
  
  call.end();  // ❌ 立即结束，客户端无法持续接收
},
```

**正确做法：**
```javascript
// ✅ 使用定时器持续推送，直到客户端取消
StreamLogs: (call) => {
  const interval = setInterval(() => {
    if (call.cancelled) {
      clearInterval(interval);
      return;
    }
    
    const log = generateLog();
    call.write(log);
  }, 1000);
  
  call.on('cancelled', () => {
    clearInterval(interval);
  });
  
  // 不要调用 call.end()，除非主动断开
},
```

### 错误3：Protobuf int64 精度丢失

**错误代码：**
```protobuf
message User {
  int64 id = 1;  // ❌ JavaScript Number 无法安全表示大于 2^53 的整数
}
```

**问题：**
```javascript
const user = { id: 9007199254740993 };  // 2^53 + 1

// 传输后变成
console.log(user.id);  // 9007199254740992（精度丢失）
```

**正确做法：**
```javascript
// ✅ 方案1：使用 string 类型
message User {
  string id = 1;  // 推荐用于大整数
}

// ✅ 方案2：加载 proto 时设置 longs: String
const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  longs: String,  // 将 int64 转为 string
});

// 现在 int64 会自动转为 string
const user = await getUser('123');
console.log(typeof user.id);  // 'string'
console.log(user.id);          // '9007199254740993'
```

## 🤔 FAQ

### Q1：gRPC 与 REST 如何选择？

**使用 gRPC：**
- ✅ 内部微服务通信（不需要浏览器支持）
- ✅ 高性能要求（QPS > 5000）
- ✅ 需要流式传输
- ✅ 多语言异构系统（自动生成代码）

**使用 REST：**
- ✅ 外部 API（第三方集成）
- ✅ 前端直接调用
- ✅ 需要 HTTP 缓存
- ✅ 团队对 REST 更熟悉

### Q2：gRPC 如何实现认证鉴权？

**方案1：Metadata 传递 Token**
```javascript
// 客户端：发送 Token
const metadata = new grpc.Metadata();
metadata.add('authorization', `Bearer ${token}`);

client.GetUser({ id: '123' }, metadata, (err, response) => {
  // ...
});

// 服务端：验证 Token
GetUser: (call, callback) => {
  const metadata = call.metadata;
  const token = metadata.get('authorization')[0];
  
  if (!verifyToken(token)) {
    return callback({
      code: grpc.status.UNAUTHENTICATED,
      message: 'Invalid token',
    });
  }
  
  // 正常处理
},
```

**方案2：拦截器统一处理**
```javascript
// 客户端拦截器（自动添加 Token）
function authInterceptor(options, nextCall) {
  return new grpc.InterceptingCall(nextCall(options), {
    start: (metadata, listener, next) => {
      metadata.add('authorization', `Bearer ${getToken()}`);
      next(metadata, listener);
    },
  });
}

const client = new userProto.UserService(
  'localhost:50051',
  grpc.credentials.createInsecure(),
  { interceptors: [authInterceptor] }
);
```
