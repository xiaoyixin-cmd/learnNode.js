## 📖 知识点概述

### 核心概念

**Redis**（Remote Dictionary Server）是一个开源的内存数据结构存储系统，可用作数据库、缓存和消息中间件。Redis 7.x（2024+）带来了更多新特性和性能提升。

### 解决的问题

- **高性能缓存**：内存存储，ns级响应
- **数据结构丰富**：String、Hash、List、Set、Sorted Set等
- **原子操作**：天然支持分布式锁
- **持久化**：RDB + AOF双保险
- **发布订阅**：实时消息推送

### 主要特点

1. **极高性能**：单机10万+QPS
2. **丰富数据结构**：5种基础+多种高级
3. **原子操作**：INCR、SET NX等
4. **持久化**：数据可以持久化到磁盘
5. **集群支持**：主从复制、哨兵、集群模式

## 💡 实践：五种数据结构

### 1. String（字符串）

**适用场景**：缓存、计数器、分布式锁

```javascript
// redis-string.js
import Redis from 'ioredis';

const redis = new Redis({
  host: 'localhost',
  port: 6379,
  db: 0,
  retryStrategy: (times) => Math.min(times * 50, 2000)
});

// ===== 基础操作 =====

// SET/GET
await redis.set('user:123:name', 'John');
const name = await redis.get('user:123:name');
console.log(name); // 'John'

// 💡 SET with options
await redis.set('session:abc', 'data', 'EX', 3600); // 1小时过期
await redis.setex('key', 60, 'value'); // 简写
await redis.setnx('lock:order:123', '1'); // 不存在时才设置

// ===== 计数器 =====

// INCR/DECR
await redis.set('page:views', 0);
await redis.incr('page:views'); // 1
await redis.incrby('page:views', 10); // 11
await redis.decr('page:views'); // 10

// 💡 应用：限流
async function rateLimit(userId) {
  const key = `rate:${userId}`;
  const count = await redis.incr(key);
  
  if (count === 1) {
    // 第一次访问，设置过期时间
    await redis.expire(key, 60); // 1分钟
  }
  
  if (count > 100) {
    throw new Error('Rate limit exceeded');
  }
  
  return count;
}

// ===== 批量操作 =====

// MSET/MGET
await redis.mset('key1', 'value1', 'key2', 'value2');
const values = await redis.mget('key1', 'key2');
console.log(values); // ['value1', 'value2']

// ===== TTL管理 =====

await redis.set('temp', 'data');
await redis.expire('temp', 300); // 5分钟后过期

const ttl = await redis.ttl('temp');
console.log(`还剩 ${ttl} 秒`);

await redis.persist('temp'); // 移除过期时间
```

### 2. Hash（哈希表）

**适用场景**：存储对象、用户信息

```javascript
// redis-hash.js

// ===== 基础操作 =====

// HSET/HGET
await redis.hset('user:123', 'name', 'John');
await redis.hset('user:123', 'email', 'john@example.com');
await redis.hset('user:123', 'age', 25);

const name = await redis.hget('user:123', 'name');
console.log(name); // 'John'

// HMSET/HMGET（批量）
await redis.hmset('user:456', {
  name: 'Jane',
  email: 'jane@example.com',
  age: 24
});

const user = await redis.hgetall('user:456');
console.log(user);
// { name: 'Jane', email: 'jane@example.com', age: '24' }

// ===== 应用：购物车 =====

class ShoppingCart {
  constructor(redis, userId) {
    this.redis = redis;
    this.key = `cart:${userId}`;
  }
  
  // 添加商品
  async addItem(productId, quantity) {
    await this.redis.hincrby(this.key, productId, quantity);
  }
  
  // 移除商品
  async removeItem(productId) {
    await this.redis.hdel(this.key, productId);
  }
  
  // 获取购物车
  async getCart() {
    const items = await this.redis.hgetall(this.key);
    
    return Object.entries(items).map(([productId, quantity]) => ({
      productId,
      quantity: parseInt(quantity)
    }));
  }
  
  // 清空购物车
  async clear() {
    await this.redis.del(this.key);
  }
  
  // 获取商品总数
  async getTotalItems() {
    const items = await this.redis.hvals(this.key);
    return items.reduce((sum, qty) => sum + parseInt(qty), 0);
  }
}

// 使用
const cart = new ShoppingCart(redis, 123);
await cart.addItem('product-1', 2);
await cart.addItem('product-2', 1);

const items = await cart.getCart();
console.log('购物车:', items);

const total = await cart.getTotalItems();
console.log('总商品数:', total);
```

### 3. List（列表）

**适用场景**：消息队列、时间线、最新动态

```javascript
// redis-list.js

// ===== 基础操作 =====

// LPUSH/RPUSH（左/右插入）
await redis.rpush('queue:jobs', 'job1', 'job2', 'job3');
await redis.lpush('timeline:user:123', 'post-10', 'post-9');

// LPOP/RPOP（左/右弹出）
const job = await redis.lpop('queue:jobs');
console.log(job); // 'job1'

// LRANGE（范围查询）
const jobs = await redis.lrange('queue:jobs', 0, -1); // 所有
console.log(jobs); // ['job2', 'job3']

// LLEN（长度）
const length = await redis.llen('queue:jobs');
console.log(length); // 2

// ===== 应用：消息队列 =====

class SimpleQueue {
  constructor(redis, queueName) {
    this.redis = redis;
    this.queueName = queueName;
  }
  
  // 入队
  async enqueue(message) {
    await this.redis.rpush(this.queueName, JSON.stringify(message));
  }
  
  // 出队（非阻塞）
  async dequeue() {
    const data = await this.redis.lpop(this.queueName);
    return data ? JSON.parse(data) : null;
  }
  
  // 出队（阻塞）
  async dequeueBlocking(timeout = 0) {
    const result = await this.redis.blpop(this.queueName, timeout);
    return result ? JSON.parse(result[1]) : null;
  }
  
  // 队列长度
  async size() {
    return await this.redis.llen(this.queueName);
  }
}

// 使用
const queue = new SimpleQueue(redis, 'tasks');

// 生产者
await queue.enqueue({ type: 'email', to: 'user@example.com' });
await queue.enqueue({ type: 'sms', phone: '13800138000' });

// 消费者
const task = await queue.dequeue();
console.log('处理任务:', task);

// ===== 应用：最新动态（限制长度）=====

async function addPost(userId, postId) {
  const key = `timeline:${userId}`;
  
  // 添加到列表头部
  await redis.lpush(key, postId);
  
  // 💡 只保留最新100条
  await redis.ltrim(key, 0, 99);
}

// 获取最新10条
async function getRecentPosts(userId, limit = 10) {
  const key = `timeline:${userId}`;
  return await redis.lrange(key, 0, limit - 1);
}
```

### 4. Set（集合）

**适用场景**：标签、好友关系、去重

```javascript
// redis-set.js

// ===== 基础操作 =====

// SADD/SREM（添加/删除）
await redis.sadd('tags:post:123', 'nodejs', 'redis', 'cache');
await redis.srem('tags:post:123', 'cache');

// SMEMBERS（获取所有成员）
const tags = await redis.smembers('tags:post:123');
console.log(tags); // ['nodejs', 'redis']

// SISMEMBER（是否存在）
const exists = await redis.sismember('tags:post:123', 'nodejs');
console.log(exists); // 1（存在）

// SCARD（元素数量）
const count = await redis.scard('tags:post:123');
console.log(count); // 2

// ===== 集合运算 =====

await redis.sadd('user:1:friends', 2, 3, 4, 5);
await redis.sadd('user:2:friends', 3, 4, 6, 7);

// 交集（共同好友）
const common = await redis.sinter('user:1:friends', 'user:2:friends');
console.log('共同好友:', common); // [3, 4]

// 并集（所有好友）
const allFriends = await redis.sunion('user:1:friends', 'user:2:friends');
console.log('所有好友:', allFriends); // [2, 3, 4, 5, 6, 7]

// 差集（用户1独有的好友）
const unique = await redis.sdiff('user:1:friends', 'user:2:friends');
console.log('独有好友:', unique); // [2, 5]

// ===== 应用：点赞去重 =====

class LikeSystem {
  constructor(redis) {
    this.redis = redis;
  }
  
  // 点赞
  async like(postId, userId) {
    const key = `post:${postId}:likes`;
    const added = await this.redis.sadd(key, userId);
    
    if (added === 1) {
      // 首次点赞，增加计数
      await this.redis.hincrby(`post:${postId}`, 'like_count', 1);
      return true;
    }
    
    return false; // 已经点过赞
  }
  
  // 取消点赞
  async unlike(postId, userId) {
    const key = `post:${postId}:likes`;
    const removed = await this.redis.srem(key, userId);
    
    if (removed === 1) {
      await this.redis.hincrby(`post:${postId}`, 'like_count', -1);
      return true;
    }
    
    return false;
  }
  
  // 是否点赞
  async hasLiked(postId, userId) {
    const key = `post:${postId}:likes`;
    return await this.redis.sismember(key, userId) === 1;
  }
  
  // 点赞用户列表
  async getLikedUsers(postId) {
    const key = `post:${postId}:likes`;
    return await this.redis.smembers(key);
  }
}

// 使用
const likeSystem = new LikeSystem(redis);
await likeSystem.like('post-123', 'user-1');
await likeSystem.like('post-123', 'user-2');

const hasLiked = await likeSystem.hasLiked('post-123', 'user-1');
console.log('用户1是否点赞:', hasLiked); // true
```

### 5. Sorted Set（有序集合）

**适用场景**：排行榜、延时队列、优先级队列

```javascript
// redis-zset.js

// ===== 基础操作 =====

// ZADD（添加，score是排序依据）
await redis.zadd('leaderboard', 100, 'player1');
await redis.zadd('leaderboard', 200, 'player2');
await redis.zadd('leaderboard', 150, 'player3');

// ZRANGE（按score升序）
const bottom = await redis.zrange('leaderboard', 0, -1);
console.log('升序:', bottom); // ['player1', 'player3', 'player2']

// ZREVRANGE（按score降序）
const top = await redis.zrevrange('leaderboard', 0, -1);
console.log('降序:', top); // ['player2', 'player3', 'player1']

// ZREVRANGE with WITHSCORES
const topWithScores = await redis.zrevrange('leaderboard', 0, 2, 'WITHSCORES');
console.log('前3名:', topWithScores);
// ['player2', '200', 'player3', '150', 'player1', '100']

// ZSCORE（获取score）
const score = await redis.zscore('leaderboard', 'player1');
console.log('player1分数:', score); // '100'

// ZRANK（排名，从0开始）
const rank = await redis.zrevrank('leaderboard', 'player1');
console.log('player1排名:', rank); // 2（第3名）

// ZINCRBY（增加score）
await redis.zincrby('leaderboard', 50, 'player1'); // 100 + 50 = 150

// ===== 应用：排行榜 =====

class Leaderboard {
  constructor(redis, leaderboardKey) {
    this.redis = redis;
    this.key = leaderboardKey;
  }
  
  // 更新分数
  async updateScore(playerId, score) {
    await this.redis.zadd(this.key, score, playerId);
  }
  
  // 增加分数
  async addScore(playerId, increment) {
    return await this.redis.zincrby(this.key, increment, playerId);
  }
  
  // 获取排名（从1开始）
  async getRank(playerId) {
    const rank = await this.redis.zrevrank(this.key, playerId);
    return rank !== null ? rank + 1 : null;
  }
  
  // 获取分数
  async getScore(playerId) {
    const score = await this.redis.zscore(this.key, playerId);
    return score !== null ? parseFloat(score) : null;
  }
  
  // 获取前N名
  async getTopPlayers(count = 10) {
    const results = await this.redis.zrevrange(
      this.key,
      0,
      count - 1,
      'WITHSCORES'
    );
    
    const players = [];
    for (let i = 0; i < results.length; i += 2) {
      players.push({
        playerId: results[i],
        score: parseFloat(results[i + 1]),
        rank: i / 2 + 1
      });
    }
    
    return players;
  }
  
  // 获取玩家周围排名
  async getPlayerWithNeighbors(playerId, range = 2) {
    const rank = await this.redis.zrevrank(this.key, playerId);
    if (rank === null) return [];
    
    const start = Math.max(0, rank - range);
    const end = rank + range;
    
    const results = await this.redis.zrevrange(
      this.key,
      start,
      end,
      'WITHSCORES'
    );
    
    const players = [];
    for (let i = 0; i < results.length; i += 2) {
      players.push({
        playerId: results[i],
        score: parseFloat(results[i + 1]),
        rank: start + i / 2 + 1
      });
    }
    
    return players;
  }
}

// 使用
const leaderboard = new Leaderboard(redis, 'game:scores');

// 更新分数
await leaderboard.updateScore('player1', 1000);
await leaderboard.updateScore('player2', 1500);
await leaderboard.addScore('player1', 200); // 1000 + 200

// 查询排名
const rank = await leaderboard.getRank('player1');
console.log('player1排名:', rank);

// 获取前10名
const top10 = await leaderboard.getTopPlayers(10);
console.log('前10名:', top10);

// 查看player1周围的排名
const neighbors = await leaderboard.getPlayerWithNeighbors('player1', 2);
console.log('player1周围:', neighbors);
```

## 🔒 分布式锁

### 实现原理

```javascript
// distributed-lock.js

class DistributedLock {
  constructor(redis) {
    this.redis = redis;
  }
  
  // 💡 获取锁
  async acquire(lockKey, timeout = 10000, value = Date.now().toString()) {
    // SET key value NX EX timeout
    // NX: 不存在时才设置
    // EX: 设置过期时间（秒）
    const result = await this.redis.set(
      lockKey,
      value,
      'NX',
      'PX',
      timeout
    );
    
    return result === 'OK' ? value : null;
  }
  
  // 💡 释放锁（Lua脚本保证原子性）
  async release(lockKey, value) {
    const script = `
      if redis.call("get", KEYS[1]) == ARGV[1] then
        return redis.call("del", KEYS[1])
      else
        return 0
      end
    `;
    
    return await this.redis.eval(script, 1, lockKey, value);
  }
  
  // 💡 自动续期（看门狗）
  startWatchdog(lockKey, value, interval = 3000) {
    return setInterval(async () => {
      const current = await this.redis.get(lockKey);
      if (current === value) {
        await this.redis.pexpire(lockKey, 10000);
        console.log('锁已续期');
      }
    }, interval);
  }
}

// 使用示例：防止订单重复提交
async function createOrder(orderId) {
  const lock = new DistributedLock(redis);
  const lockKey = `lock:order:${orderId}`;
  const lockValue = await lock.acquire(lockKey, 5000);
  
  if (!lockValue) {
    throw new Error('订单正在处理中，请勿重复提交');
  }
  
  try {
    // 处理订单...
    console.log('创建订单:', orderId);
    await new Promise(resolve => setTimeout(resolve, 2000));
    
    console.log('订单创建成功');
  } finally {
    // 释放锁
    await lock.release(lockKey, lockValue);
  }
}
```

## 📋 总结

### 核心要点

1. **5种数据结构**：String、Hash、List、Set、Sorted Set
2. **分布式锁**：SET NX EX + Lua脚本释放
3. **发布订阅**：实时消息推送
4. **Pipeline**：批量操作，减少网络开销

### 实现关键

1. **String**：SET/GET/INCR，用于缓存和计数
2. **Hash**：HSET/HGET，用于存储对象
3. **List**：LPUSH/RPOP，用于队列
4. **Set**：SADD/SISMEMBER，用于去重和集合运算
5. **Sorted Set**：ZADD/ZREVRANGE，用于排行榜

### 应用价值

- **极高性能**：10万+QPS
- **丰富功能**：缓存、锁、队列、排行榜
- **易于扩展**：集群、哨兵

### 面试重点

1. **Redis数据结构有哪些？**
   - String、Hash、List、Set、Sorted Set

2. **如何实现分布式锁？**
   - SET NX EX获取锁，Lua脚本原子释放

3. **Redis为什么这么快？**
   - 内存存储、单线程、高效数据结构、IO多路复用

4. **Redis持久化方式？**
   - RDB（快照）、AOF（日志）

### 注意事项

⚠️ **必须设置锁的过期时间**：
```javascript
// ✅ 正确
await redis.set('lock', '1', 'NX', 'EX', 10);

// ❌ 错误（可能死锁）
await redis.setnx('lock', '1');
```

⚠️ **释放锁要验证owner**：
```javascript
// 用Lua脚本保证原子性
if redis.call("get", KEYS[1]) == ARGV[1] then
  return redis.call("del", KEYS[1])
end
```
