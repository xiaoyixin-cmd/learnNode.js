# 1.3 路由系统手写实现

> 从零实现一个功能完整的路由系统，理解Web框架的核心机制

## 📖 知识点概述

### 核心概念

**路由系统**是Web应用的核心组件，负责将HTTP请求映射到对应的处理函数。它解决了"客户端请求/api/users/123时，应该调用哪个函数来处理"的问题。

### 解决的问题

- **请求分发**：根据URL和HTTP方法分发请求到不同处理器
- **参数提取**：从URL中提取动态参数（如/users/:id中的id）
- **路由嵌套**：支持路由分组和前缀（如/api/v1/users）
- **中间件机制**：在请求处理前后执行通用逻辑（日志、认证等）

### 主要特点

1. **路径匹配**：支持静态路径和动态参数
2. **HTTP方法区分**：GET、POST、PUT、DELETE等
3. **参数提取**：自动提取路径参数和查询参数
4. **中间件支持**：洋葱模型或管道模型
5. **性能优化**：使用Trie树或正则缓存提升匹配速度

### 典型应用场景

- **RESTful API**：实现资源的CRUD操作
- **SPA应用**：前后端分离的路由处理
- **微服务网关**：请求路由转发
- **学习框架原理**：理解Express/Koa的路由实现

## 🚀 使用指南

### 理解：路由系统做什么？

路由系统的核心任务：
1. **匹配**：判断请求URL是否匹配某个路由规则
2. **提取**：从URL中提取参数
3. **执行**：调用对应的处理函数
4. **传递**：执行中间件链

### 实践：从简单到复杂逐步实现

#### 第一版：基础路由（静态路径）

```javascript
// router-v1.js
import http from 'node:http';

class Router {
  constructor() {
    // 存储路由规则：{ method: { path: handler } }
    this.routes = {
      GET: new Map(),
      POST: new Map(),
      PUT: new Map(),
      DELETE: new Map()
    };
  }
  
  // 注册路由
  get(path, handler) {
    this.routes.GET.set(path, handler);
  }
  
  post(path, handler) {
    this.routes.POST.set(path, handler);
  }
  
  put(path, handler) {
    this.routes.PUT.set(path, handler);
  }
  
  delete(path, handler) {
    this.routes.DELETE.set(path, handler);
  }
  
  // 处理请求
  async handle(req, res) {
    const method = req.method;
    const url = new URL(req.url, `http://${req.headers.host}`);
    const path = url.pathname;
    
    // 查找对应的处理函数
    const handler = this.routes[method]?.get(path);
    
    if (handler) {
      try {
        await handler(req, res);
      } catch (error) {
        console.error('处理错误:', error);
        res.writeHead(500, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify({ error: 'Internal Server Error' }));
      }
    } else {
      // 404
      res.writeHead(404, { 'Content-Type': 'application/json' });
      res.end(JSON.stringify({ error: 'Not Found' }));
    }
  }
}

// 使用示例
const router = new Router();

router.get('/api/hello', (req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ message: 'Hello World' }));
});

router.get('/api/users', (req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ users: ['Alice', 'Bob'] }));
});

const server = http.createServer((req, res) => {
  router.handle(req, res);
});

server.listen(3000, () => {
  console.log('V1路由服务器运行在 http://localhost:3000');
  console.log('测试: curl http://localhost:3000/api/hello');
});
```

#### 第二版：支持动态参数

```javascript
// router-v2.js
import http from 'node:http';

class Router {
  constructor() {
    this.routes = [];
  }
  
  // 将路径模式转换为正则表达式
  // /users/:id/:action -> /users/([^/]+)/([^/]+)
  pathToRegex(path) {
    const paramNames = [];
    
    // 提取参数名
    const pattern = path.replace(/:([^/]+)/g, (match, paramName) => {
      paramNames.push(paramName);
      return '([^/]+)'; // 匹配任意非/字符
    });
    
    // 💡 技巧：^和$确保完全匹配
    const regex = new RegExp(`^${pattern}$`);
    
    return { regex, paramNames };
  }
  
  // 注册路由
  addRoute(method, path, handler) {
    const { regex, paramNames } = this.pathToRegex(path);
    this.routes.push({ method, path, regex, paramNames, handler });
  }
  
  get(path, handler) {
    this.addRoute('GET', path, handler);
  }
  
  post(path, handler) {
    this.addRoute('POST', path, handler);
  }
  
  put(path, handler) {
    this.addRoute('PUT', path, handler);
  }
  
  delete(path, handler) {
    this.addRoute('DELETE', path, handler);
  }
  
  // 匹配路由并提取参数
  match(method, pathname) {
    for (const route of this.routes) {
      if (route.method !== method) continue;
      
      const match = pathname.match(route.regex);
      if (match) {
        // 提取参数
        const params = {};
        route.paramNames.forEach((name, index) => {
          params[name] = match[index + 1];
        });
        
        return { handler: route.handler, params };
      }
    }
    
    return null;
  }
  
  // 处理请求
  async handle(req, res) {
    const method = req.method;
    const url = new URL(req.url, `http://${req.headers.host}`);
    const pathname = url.pathname;
    
    const result = this.match(method, pathname);
    
    if (result) {
      try {
        // 💡 技巧：将params挂载到req对象上
        req.params = result.params;
        req.query = Object.fromEntries(url.searchParams);
        
        await result.handler(req, res);
      } catch (error) {
        console.error('处理错误:', error);
        res.writeHead(500, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify({ error: 'Internal Server Error' }));
      }
    } else {
      res.writeHead(404, { 'Content-Type': 'application/json' });
      res.end(JSON.stringify({ error: 'Not Found' }));
    }
  }
}

// 使用示例
const router = new Router();

router.get('/api/users/:id', (req, res) => {
  const { id } = req.params;
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ user: { id, name: `User ${id}` } }));
});

router.get('/api/posts/:postId/comments/:commentId', (req, res) => {
  const { postId, commentId } = req.params;
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ postId, commentId }));
});

const server = http.createServer((req, res) => {
  router.handle(req, res);
});

server.listen(3000, () => {
  console.log('V2路由服务器运行在 http://localhost:3000');
  console.log('测试: curl http://localhost:3000/api/users/123');
  console.log('测试: curl http://localhost:3000/api/posts/1/comments/5');
});
```

#### 第三版：中间件支持（完整版）

```javascript
// router-v3.js
import http from 'node:http';

class Router {
  constructor() {
    this.routes = [];
    this.middlewares = []; // 全局中间件
  }
  
  pathToRegex(path) {
    const paramNames = [];
    const pattern = path.replace(/:([^/]+)/g, (match, paramName) => {
      paramNames.push(paramName);
      return '([^/]+)';
    });
    const regex = new RegExp(`^${pattern}$`);
    return { regex, paramNames };
  }
  
  // 注册全局中间件
  use(middleware) {
    this.middlewares.push(middleware);
  }
  
  addRoute(method, path, ...handlers) {
    const { regex, paramNames } = this.pathToRegex(path);
    // ⚠️ 注意：最后一个是处理函数，前面的都是中间件
    const routeMiddlewares = handlers.slice(0, -1);
    const handler = handlers[handlers.length - 1];
    
    this.routes.push({
      method,
      path,
      regex,
      paramNames,
      middlewares: routeMiddlewares,
      handler
    });
  }
  
  get(path, ...handlers) {
    this.addRoute('GET', path, ...handlers);
  }
  
  post(path, ...handlers) {
    this.addRoute('POST', path, ...handlers);
  }
  
  put(path, ...handlers) {
    this.addRoute('PUT', path, ...handlers);
  }
  
  delete(path, ...handlers) {
    this.addRoute('DELETE', path, ...handlers);
  }
  
  match(method, pathname) {
    for (const route of this.routes) {
      if (route.method !== method) continue;
      
      const match = pathname.match(route.regex);
      if (match) {
        const params = {};
        route.paramNames.forEach((name, index) => {
          params[name] = match[index + 1];
        });
        
        return {
          handler: route.handler,
          middlewares: route.middlewares,
          params
        };
      }
    }
    
    return null;
  }
  
  // 执行中间件链（洋葱模型）
  async runMiddlewares(middlewares, req, res, finalHandler) {
    let index = 0;
    
    const next = async () => {
      if (index < middlewares.length) {
        const middleware = middlewares[index++];
        await middleware(req, res, next);
      } else {
        // 所有中间件执行完毕，执行最终处理函数
        await finalHandler(req, res);
      }
    };
    
    await next();
  }
  
  async handle(req, res) {
    const method = req.method;
    const url = new URL(req.url, `http://${req.headers.host}`);
    const pathname = url.pathname;
    
    const result = this.match(method, pathname);
    
    if (result) {
      try {
        req.params = result.params;
        req.query = Object.fromEntries(url.searchParams);
        
        // 💡 技巧：合并全局中间件和路由中间件
        const allMiddlewares = [
          ...this.middlewares,
          ...result.middlewares
        ];
        
        await this.runMiddlewares(allMiddlewares, req, res, result.handler);
      } catch (error) {
        console.error('处理错误:', error);
        if (!res.headersSent) {
          res.writeHead(500, { 'Content-Type': 'application/json' });
          res.end(JSON.stringify({ error: 'Internal Server Error' }));
        }
      }
    } else {
      res.writeHead(404, { 'Content-Type': 'application/json' });
      res.end(JSON.stringify({ error: 'Not Found' }));
    }
  }
}

// ===== 使用示例 =====

const router = new Router();

// 全局中间件：日志
router.use(async (req, res, next) => {
  console.log(`[${new Date().toISOString()}] ${req.method} ${req.url}`);
  await next();
});

// 全局中间件：CORS
router.use(async (req, res, next) => {
  res.setHeader('Access-Control-Allow-Origin', '*');
  res.setHeader('Access-Control-Allow-Methods', 'GET, POST, PUT, DELETE');
  await next();
});

// 路由级中间件：认证
async function authMiddleware(req, res, next) {
  const token = req.headers.authorization;
  
  if (!token) {
    res.writeHead(401, { 'Content-Type': 'application/json' });
    res.end(JSON.stringify({ error: 'Unauthorized' }));
    return;
  }
  
  // 模拟token验证
  req.user = { id: 1, name: 'Alice' };
  await next();
}

// 使用中间件的路由
router.get('/api/profile', authMiddleware, (req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ user: req.user }));
});

router.get('/api/users/:id', (req, res) => {
  const { id } = req.params;
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ user: { id, name: `User ${id}` } }));
});

const server = http.createServer((req, res) => {
  router.handle(req, res);
});

server.listen(3000, () => {
  console.log('V3路由服务器运行在 http://localhost:3000');
  console.log('测试公开接口: curl http://localhost:3000/api/users/123');
  console.log('测试认证接口: curl -H "Authorization: Bearer token" http://localhost:3000/api/profile');
});
```

### 错误处理：3个高频问题

#### 问题1：中间件中忘记调用next()

**错误表现**：请求挂起，没有响应

**原因**：中间件没有调用next()，导致请求处理链中断

**错误代码**：
```javascript
// ❌ 错误示例
router.use(async (req, res, next) => {
  console.log('日志记录');
  // 忘记调用next()，请求挂起
});
```

**解决方案**：
```javascript
// ✅ 正确做法：必须调用next()
router.use(async (req, res, next) => {
  console.log('日志记录');
  await next(); // 必须调用
});
```

#### 问题2：正则匹配不精确导致路由冲突

**错误表现**：/api/users/new被/api/users/:id匹配

**原因**：正则表达式没有使用^和$，导致部分匹配

**错误代码**：
```javascript
// ❌ 错误示例
const regex = new RegExp(pattern); // 没有^和$
// 会导致 /api/users/new 被 /api/users/:id 匹配
```

**解决方案**：
```javascript
// ✅ 正确做法：使用^和$确保完全匹配
const regex = new RegExp(`^${pattern}$`);

// ✅ 或者：调整路由顺序（更具体的在前）
router.get('/api/users/new', handler1);  // 先注册
router.get('/api/users/:id', handler2);  // 后注册
```

#### 问题3：参数类型未转换

**错误表现**：数字参数是字符串，导致比较失败

**原因**：URL参数都是字符串，需要手动转换类型

**错误代码**：
```javascript
// ❌ 错误示例

  const { id } = req.params;
  if (id === 123) { // ❌ id是字符串'123'，永远不等于数字123
    // ...
  }
});
```

**解决方案**：
```javascript
// ✅ 方案1：手动转换
router.get('/api/users/:id', (req, res) => {
  const id = parseInt(req.params.id, 10);
  if (id === 123) { // ✅ 正确
    // ...
  }
});

// ✅ 方案2：路由器内置转换（高级功能）
class Router {
  // ...
  match(method, pathname) {
    // ... 提取参数后
    // 自动转换数字
    Object.keys(params).forEach(key => {
      if (/^\d+$/.test(params[key])) {
        params[key] = parseInt(params[key], 10);
      }
    });
    return { handler, params };
  }
}
```

## 🎯 应用场景

### 场景1：微服务API网关路由

#### 问题背景
公司有10+个微服务，需要一个API网关统一对外提供接口。网关需要：
- 根据路径前缀转发到不同服务
- 统一认证和限流
- 请求日志记录
- 支持版本控制（/api/v1、/api/v2）

#### 技术选型理由
手写路由系统的原因：
- **性能要求高**：QPS需达到10万+，框架开销不可接受
- **定制需求多**：需要特殊的路由匹配逻辑
- **学习成本**：团队可完全掌控代码

#### 实现代码

```javascript
// api-gateway-router.js
import http from 'node:http';
import httpProxy from 'http-proxy'; // npm install http-proxy

class GatewayRouter {
  constructor() {
    this.routes = [];
    this.proxy = httpProxy.createProxyServer({});
  }
  
  // 注册服务路由
  service(prefix, target, middlewares = []) {
    this.routes.push({
      prefix,
      target,
      middlewares,
      regex: new RegExp(`^${prefix}`)
    });
  }
  
  // 限流中间件（令牌桶）
  createRateLimiter(maxTokens, refillRate) {
    let tokens = maxTokens;
    setInterval(() => {
      tokens = Math.min(maxTokens, tokens + refillRate);
    }, 1000);
    
    return async (req, res, next) => {
      if (tokens > 0) {
        tokens--;
        await next();
      } else {
        res.writeHead(429, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify({ error: 'Too Many Requests' }));
      }
    };
  }
  
  // 认证中间件
  authMiddleware = async (req, res, next) => {
    const token = req.headers.authorization;
    
    if (!token || !token.startsWith('Bearer ')) {
      res.writeHead(401, { 'Content-Type': 'application/json' });
      res.end(JSON.stringify({ error: 'Unauthorized' }));
      return;
    }
    
    // 简化：实际应该验证JWT
    req.user = { id: 1, name: 'Alice' };
    await next();
  };
  
  // 日志中间件
  logMiddleware = async (req, res, next) => {
    const start = Date.now();
    const originalEnd = res.end;
    
    res.end = function(...args) {
      const duration = Date.now() - start;
      console.log(
        `[${new Date().toISOString()}] ` +
        `${req.method} ${req.url} - ` +
        `${res.statusCode} - ${duration}ms`
      );
      originalEnd.apply(res, args);
    };
    
    await next();
  };
  
  async handle(req, res) {
    const url = new URL(req.url, `http://${req.headers.host}`);
    const pathname = url.pathname;
    
    // 匹配服务
    for (const route of this.routes) {
      if (route.regex.test(pathname)) {
        try {
          // 执行中间件
          await this.runMiddlewares(route.middlewares, req, res, async () => {
            // 转发请求
            this.proxy.web(req, res, { target: route.target }, (error) => {
              console.error('代理错误:', error);
              if (!res.headersSent) {
                res.writeHead(502);
                res.end('Bad Gateway');
              }
            });
          });
          return;
        } catch (error) {
          console.error('网关错误:', error);
          if (!res.headersSent) {
            res.writeHead(500);
            res.end('Internal Server Error');
          }
          return;
        }
      }
    }
    
    // 未匹配到服务
    res.writeHead(404, { 'Content-Type': 'application/json' });
    res.end(JSON.stringify({ error: 'Service Not Found' }));
  }
  
  async runMiddlewares(middlewares, req, res, finalHandler) {
    let index = 0;
    const next = async () => {
      if (index < middlewares.length) {
        await middlewares[index++](req, res, next);
      } else {
        await finalHandler();
      }
    };
    await next();
  }
}

// 使用
const gateway = new GatewayRouter();

// 限流器：每秒100个请求
const limiter = gateway.createRateLimiter(100, 100);

// 注册服务路由
gateway.service(
  '/api/v1/users',
  'http://localhost:4001',
  [gateway.logMiddleware, gateway.authMiddleware, limiter]
);

gateway.service(
  '/api/v1/orders',
  'http://localhost:4002',
  [gateway.logMiddleware, gateway.authMiddleware, limiter]
);

gateway.service(
  '/api/v1/products',
  'http://localhost:4003',
  [gateway.logMiddleware, limiter] // 公开接口，不需要认证
);

const server = http.createServer((req, res) => {
  gateway.handle(req, res);
});

server.listen(3000, () => {
  console.log('API网关运行在 http://localhost:3000');
  console.log('用户服务: /api/v1/users -> http://localhost:4001');
  console.log('订单服务: /api/v1/orders -> http://localhost:4002');
  console.log('商品服务: /api/v1/products -> http://localhost:4003');
});
```

### 场景2：SPA单页应用路由

#### 问题背景

#### 实现代码

```javascript
// spa-router.js
import http from 'node:http';
import fs from 'node:fs';
import path from 'node:path';
import { fileURLToPath } from 'node:url';

const __filename = fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);
const PUBLIC_DIR = path.join(__dirname, 'dist');

class SPARouter {
  constructor() {
    this.apiRoutes = [];
  }
  
  // 注册API路由
  api(method, path, handler) {
    const regex = new RegExp(`^${path.replace(/:\w+/g, '([^/]+)')}$`);
    this.apiRoutes.push({ method, path, regex, handler });
  }
  
  // 匹配API路由
  matchAPI(method, pathname) {
    for (const route of this.apiRoutes) {
      if (route.method === method && route.regex.test(pathname)) {
        return route.handler;
      }
    }
    return null;
  }
  
  // 提供静态文件
  async serveStatic(pathname, res) {
    const filePath = path.join(PUBLIC_DIR, pathname);
    
    // 安全检查
    if (!filePath.startsWith(PUBLIC_DIR)) {
      res.writeHead(403);
      res.end('Forbidden');
      return;
    }
    
    try {
      const stats = await fs.promises.stat(filePath);
      
      if (stats.isFile()) {
        const stream = fs.createReadStream(filePath);
        const ext = path.extname(filePath);
        const mimeTypes = {
          '.html': 'text/html',
          '.js': 'application/javascript',
          '.css': 'text/css',
          '.png': 'image/png',
          '.jpg': 'image/jpeg'
        };
        
        res.writeHead(200, { 'Content-Type': mimeTypes[ext] || 'text/plain' });
        stream.pipe(res);
        return true;
      }
    } catch (error) {
      return false;
    }
  }
  
  // 返回index.html（用于前端路由）
  async serveIndex(res) {
    const indexPath = path.join(PUBLIC_DIR, 'index.html');
    
    try {
      const html = await fs.promises.readFile(indexPath, 'utf8');
      res.writeHead(200, { 'Content-Type': 'text/html' });
      res.end(html);
    } catch (error) {
      res.writeHead(500);
      res.end('Internal Server Error');
    }
  }
  
  async handle(req, res) {
    const url = new URL(req.url, `http://${req.headers.host}`);
    const pathname = url.pathname;
    const method = req.method;
    
    // 1. 检查是否是API请求
    if (pathname.startsWith('/api/')) {
      const handler = this.matchAPI(method, pathname);
      if (handler) {
        await handler(req, res);
        return;
      }
    }
    
    // 2. 尝试提供静态文件
    if (await this.serveStatic(pathname, res)) {
      return;
    }
    
    // 3. 所有其他请求返回index.html（前端路由处理）
    await this.serveIndex(res);
  }
}

// 使用
const router = new SPARouter();

// 注册API路由
router.api('GET', '/api/users', (req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ users: ['Alice', 'Bob'] }));
});

const server = http.createServer((req, res) => {
  router.handle(req, res);
});

server.listen(3000, () => {
  console.log('SPA服务器运行在 http://localhost:3000');
});
```

- **开发体验**：前端路由和后端路由完美配合
- **部署简单**：单个服务同时提供前端和API
- **性能**：静态文件缓存，加载速度提升50%

### 性能优化案例

#### 优化前：每次遍历所有路由

```javascript
// ❌ 性能问题：线性查找，O(n)复杂度
class Router {
  match(method, pathname) {
    for (const route of this.routes) {
      if (route.method === method && route.regex.test(pathname)) {
        return route;
      }
    }
    return null;
  }
}

// 问题分析：
// 1000个路由时，平均需要遍历500次
// QPS受限于路由数量
```

#### 优化后：使用Trie树优化

```javascript
// ✅ 性能优化：Trie树，O(m)复杂度（m为路径长度）
class TrieNode {
  constructor() {
    this.children = new Map();
    this.handler = null;
    this.isParam = false;
    this.paramName = null;
  }
}

class TrieRouter {
  constructor() {
    this.root = new TrieNode();
  }
  
  // 插入路由到Trie树
  insert(path, handler) {
    const segments = path.split('/').filter(Boolean);
    let node = this.root;
    
    for (const segment of segments) {
      if (segment.startsWith(':')) {
        // 参数节点
        if (!node.children.has(':param')) {
          const paramNode = new TrieNode();
          paramNode.isParam = true;
          paramNode.paramName = segment.slice(1);
          node.children.set(':param', paramNode);
        }
        node = node.children.get(':param');
      } else {
        // 静态节点
        if (!node.children.has(segment)) {
          node.children.set(segment, new TrieNode());
        }
        node = node.children.get(segment);
      }
    }
    
    node.handler = handler;
  }
  
  // 查找路由
  search(pathname) {
    const segments = pathname.split('/').filter(Boolean);
    let node = this.root;
    const params = {};
    
    for (const segment of segments) {
      // 优先匹配静态路由
      if (node.children.has(segment)) {
        node = node.children.get(segment);
      }
      // 其次匹配参数路由
      else if (node.children.has(':param')) {
        const paramNode = node.children.get(':param');
        params[paramNode.paramName] = segment;
        node = paramNode;
      }
      // 未匹配
      else {
        return null;
      }
    }
    
    return node.handler ? { handler: node.handler, params } : null;
  }
}

// 使用
const router = new TrieRouter();
router.insert('/api/users/:id', handler1);
router.insert('/api/users/:id/posts', handler2);

const result = router.search('/api/users/123/posts');
// { handler: handler2, params: { id: '123' } }
```

## 📋 总结

### 核心要点

1. **路由系统的本质**：URL模式匹配 + 处理函数映射
2. **动态参数**：使用正则表达式提取URL中的变量
3. **中间件机制**：洋葱模型，逐层执行

### 实现关键

1. **正则转换**：将/users/:id转为/users/([^/]+)
2. **参数提取**：从匹配结果中提取参数值
3. **中间件链**：使用递归或迭代执行中间件
4. **错误处理**：捕获所有异步错误
5. **性能优化**：使用Trie树或缓存

### 应用价值

- **理解框架**：知道Express/Koa路由如何工作
- **性能优化**：在极致性能场景下使用原生实现
- **定制需求**：完全控制路由匹配逻辑
- **面试加分**：展示对底层原理的深入理解

### 面试重点

**常见问题**：
1. **如何实现动态路由参数？**
   - 将:id转换为正则表达式([^/]+)，匹配后提取参数

2. **中间件的洋葱模型是什么？**
   - 请求进入时逐层执行next()前的代码，响应返回时逆序执行next()后的代码

3. **如何优化路由匹配性能？**
   - 使用Trie树数据结构，将O(n)降为O(m)

4. **路由冲突如何处理？**
   - 静态路由优先于动态路由，或按注册顺序匹配

### 注意事项

⚠️ **必须使用^和$确保完全匹配**：
```javascript
const regex = new RegExp(`^${pattern}$`); // 完全匹配
```

⚠️ **中间件必须调用next()**：
```javascript
async (req, res, next) => {
  // 处理逻辑
  await next(); // 必须调用
};
```
