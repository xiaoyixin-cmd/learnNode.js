# 客户端缓存策略

## 1. 概述与背景

### 1.1 什么是客户端缓存

客户端缓存是在浏览器端存储数据的技术,用于减少网络请求、提升应用性能和改善用户体验。在 Next.js 16 中,合理使用客户端缓存可以显著提升应用的响应速度。

客户端缓存的核心特点:

- **减少网络请求**: 避免重复请求相同数据
- **提升响应速度**: 从本地读取数据更快
- **离线支持**: 在网络不可用时仍能访问数据
- **降低服务器负载**: 减少服务器压力
- **改善用户体验**: 更快的页面加载和交互

### 1.2 为什么需要客户端缓存

在现代 Web 应用中,客户端缓存至关重要:

- **性能优化**: 减少网络延迟
- **用户体验**: 即时响应用户操作
- **成本节约**: 减少带宽消耗
- **离线功能**: 支持 PWA 应用
- **数据一致性**: 保持数据同步

### 1.3 缓存策略对比

**对比表格**:

| 缓存类型       | 容量       | 持久性       | 性能 | 适用场景 | 浏览器支持 |
| :------------- | :--------- | :----------- | :--- | :------- | :--------- |
| Memory Cache   | 小         | 页面刷新丢失 | 最快 | 临时数据 | 所有浏览器 |
| localStorage   | 5-10MB     | 永久         | 快   | 用户设置 | 所有浏览器 |
| sessionStorage | 5-10MB     | 会话结束丢失 | 快   | 会话数据 | 所有浏览器 |
| IndexedDB      | 大 (50MB+) | 永久         | 中等 | 大量数据 | 所有浏览器 |
| Cache API      | 大         | 永久         | 快   | 静态资源 | 现代浏览器 |

## 2. 核心概念

### 2.1 Memory Cache (内存缓存)

#### 基本实现

```typescript
"use client";

class MemoryCache<T> {
  private cache: Map<string, { data: T; timestamp: number }> = new Map();
  private ttl: number;

  constructor(ttl: number = 5 * 60 * 1000) {
    this.ttl = ttl;
  }

  set(key: string, data: T): void {
    this.cache.set(key, {
      data,
      timestamp: Date.now(),
    });
  }

  get(key: string): T | null {
    const item = this.cache.get(key);

    if (!item) return null;

    if (Date.now() - item.timestamp > this.ttl) {
      this.cache.delete(key);
      return null;
    }

    return item.data;
  }

  clear(): void {
    this.cache.clear();
  }

  delete(key: string): void {
    this.cache.delete(key);
  }
}

// 使用示例
const cache = new MemoryCache<any>(5 * 60 * 1000); // 5 分钟 TTL

export async function fetchWithCache(url: string) {
  const cached = cache.get(url);
  if (cached) return cached;

  const response = await fetch(url);
  const data = await response.json();

  cache.set(url, data);
  return data;
}
```

### 2.2 localStorage

#### 基本使用

```typescript
"use client";

interface CacheItem<T> {
  data: T;
  timestamp: number;
  ttl: number;
}

class LocalStorageCache {
  private prefix: string;

  constructor(prefix: string = "app_cache_") {
    this.prefix = prefix;
  }

  set<T>(key: string, data: T, ttl: number = 3600000): void {
    const item: CacheItem<T> = {
      data,
      timestamp: Date.now(),
      ttl,
    };

    try {
      localStorage.setItem(this.prefix + key, JSON.stringify(item));
    } catch (error) {
      console.error("localStorage set error:", error);
    }
  }

  get<T>(key: string): T | null {
    try {
      const itemStr = localStorage.getItem(this.prefix + key);
      if (!itemStr) return null;

      const item: CacheItem<T> = JSON.parse(itemStr);

      if (Date.now() - item.timestamp > item.ttl) {
        this.delete(key);
        return null;
      }

      return item.data;
    } catch (error) {
      console.error("localStorage get error:", error);
      return null;
    }
  }

  delete(key: string): void {
    localStorage.removeItem(this.prefix + key);
  }

  clear(): void {
    const keys = Object.keys(localStorage);
    keys.forEach((key) => {
      if (key.startsWith(this.prefix)) {
        localStorage.removeItem(key);
      }
    });
  }
}

// 使用示例
const cache = new LocalStorageCache();

export function saveUserPreferences(preferences: any) {
  cache.set("user_preferences", preferences, 7 * 24 * 60 * 60 * 1000); // 7 天
}

export function getUserPreferences() {
  return cache.get("user_preferences");
}
```

### 2.4 IndexedDB

#### 基本使用

```typescript
"use client";

class IndexedDBCache {
  private dbName: string;
  private storeName: string;
  private db: IDBDatabase | null = null;

  constructor(dbName: string = "app_cache", storeName: string = "cache") {
    this.dbName = dbName;
    this.storeName = storeName;
  }

  async init(): Promise<void> {
    return new Promise((resolve, reject) => {
      const request = indexedDB.open(this.dbName, 1);

      request.onerror = () => reject(request.error);
      request.onsuccess = () => {
        this.db = request.result;
        resolve();
      };

      request.onupgradeneeded = (event) => {
        const db = (event.target as IDBOpenDBRequest).result;
        if (!db.objectStoreNames.contains(this.storeName)) {
          db.createObjectStore(this.storeName, { keyPath: "key" });
        }
      };
    });
  }

  async set<T>(key: string, data: T, ttl?: number): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([this.storeName], "readwrite");
      const store = transaction.objectStore(this.storeName);

      const item = {
        key,
        data,
        timestamp: Date.now(),
        ttl,
      };

      const request = store.put(item);
      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async get<T>(key: string): Promise<T | null> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([this.storeName], "readonly");
      const store = transaction.objectStore(this.storeName);
      const request = store.get(key);

      request.onsuccess = () => {
        const item = request.result;

        if (!item) {
          resolve(null);
          return;
        }

        if (item.ttl && Date.now() - item.timestamp > item.ttl) {
          this.delete(key);
          resolve(null);
          return;
        }

        resolve(item.data);
      };

      request.onerror = () => reject(request.error);
    });
  }

  async delete(key: string): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([this.storeName], "readwrite");
      const store = transaction.objectStore(this.storeName);
      const request = store.delete(key);

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async clear(): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([this.storeName], "readwrite");
      const store = transaction.objectStore(this.storeName);
      const request = store.clear();

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }
}

// 使用示例

  await idbCache.set("posts", posts, 24 * 60 * 60 * 1000); // 24 小时
}

export async function getPostsFromCache() {
  return await idbCache.get("posts");
}
```

### 3.2 API 响应缓存

#### 多层缓存策略

```typescript
"use client";

class APICache {
  private memoryCache: Map<string, any> = new Map();
  private localStoragePrefix = "api_cache_";

  async get(url: string): Promise<any | null> {
    // 1. 检查内存缓存
    if (this.memoryCache.has(url)) {
      return this.memoryCache.get(url);
    }

    // 2. 检查 localStorage
    try {
      const cached = localStorage.getItem(this.localStoragePrefix + url);
      if (cached) {
        const { data, timestamp, ttl } = JSON.parse(cached);

        if (Date.now() - timestamp < ttl) {
          this.memoryCache.set(url, data);
          return data;
        }

        localStorage.removeItem(this.localStoragePrefix + url);
      }
    } catch (error) {
      console.error("Cache get error:", error);
    }

    return null;
  }

  async set(url: string, data: any, ttl: number = 300000): Promise<void> {
    // 1. 保存到内存缓存
    this.memoryCache.set(url, data);

    // 2. 保存到 localStorage
    try {
      const cacheItem = {
        data,
        timestamp: Date.now(),
        ttl,
      };

      localStorage.setItem(
        this.localStoragePrefix + url,
        JSON.stringify(cacheItem)
      );
    } catch (error) {
      console.error("Cache set error:", error);
    }
  }

  clear(): void {
    this.memoryCache.clear();

    const keys = Object.keys(localStorage);
    keys.forEach((key) => {
      if (key.startsWith(this.localStoragePrefix)) {
        localStorage.removeItem(key);
      }
    });
  }
}

export const apiCache = new APICache();

export async function fetchWithMultiLayerCache(url: string) {
  const cached = await apiCache.get(url);
  if (cached) return cached;

  const response = await fetch(url);
  const data = await response.json();

  await apiCache.set(url, data, 5 * 60 * 1000); // 5 分钟

  return data;
}
```

```typescript
          const store = db.createObjectStore(this.storeName, { keyPath: "id" });
          store.createIndex("createdAt", "createdAt", { unique: false });
        }
      };
    });
  }

  async savePosts(posts: Post[]): Promise<void> {
    if (!this.db) await this.init();

    const transaction = this.db!.transaction([this.storeName], "readwrite");
    const store = transaction.objectStore(this.storeName);

    for (const post of posts) {
      store.put(post);
    }

    return new Promise((resolve, reject) => {
      transaction.oncomplete = () => resolve();
      transaction.onerror = () => reject(transaction.error);
    });
  }

  async getPosts(): Promise<Post[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([this.storeName], "readonly");
      const store = transaction.objectStore(this.storeName);
      const request = store.getAll();

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }
}
```

### 4.1 缓存策略配置

#### 缓存策略类型

```typescript
type CacheStrategy =
  | "cache-first" // 优先使用缓存
  | "network-first" // 优先使用网络
  | "cache-only" // 仅使用缓存
  | "network-only" // 仅使用网络
  | "stale-while-revalidate"; // 返回缓存同时更新

interface CacheConfig {
  strategy: CacheStrategy;
  ttl?: number;
  maxAge?: number;
  maxItems?: number;
}
```

#### 缓存策略实现

```typescript
"use client";

class CacheStrategyManager {
  private cache: Map<string, any> = new Map();

  async fetch(url: string, config: CacheConfig): Promise<any> {
    switch (config.strategy) {
      case "cache-first":
        return this.cacheFirst(url, config);
      case "network-first":
        return this.networkFirst(url, config);
      case "cache-only":
        return this.cacheOnly(url);
      case "network-only":
        return this.networkOnly(url);
      case "stale-while-revalidate":
        return this.staleWhileRevalidate(url, config);
      default:
        return this.networkFirst(url, config);
    }
  }

  private async cacheFirst(url: string, config: CacheConfig): Promise<any> {
    const cached = this.cache.get(url);
    if (cached) return cached;

    const data = await this.networkOnly(url);
    this.cache.set(url, data);
    return data;
  }

  private async networkFirst(url: string, config: CacheConfig): Promise<any> {
    try {
      const data = await this.networkOnly(url);
      this.cache.set(url, data);
      return data;
    } catch (error) {
      const cached = this.cache.get(url);
      if (cached) return cached;
      throw error;
    }
  }

  private async cacheOnly(url: string): Promise<any> {
    const cached = this.cache.get(url);
    if (!cached) throw new Error("No cache available");
    return cached;
  }

  private async networkOnly(url: string): Promise<any> {
    const response = await fetch(url);
    return await response.json();
  }

  private async staleWhileRevalidate(
    url: string,
    config: CacheConfig
  ): Promise<any> {
    const cached = this.cache.get(url);

    // 后台更新
    this.networkOnly(url).then((data) => {
      this.cache.set(url, data);
    });

    // 立即返回缓存
    if (cached) return cached;

    // 如果没有缓存,等待网络请求
    return this.networkOnly(url);
  }
}

export const cacheManager = new CacheStrategyManager();
```

```typescript
import { useState, useEffect } from "react";

function useCache<T>(key: string, fetcher: () => Promise<T>) {
  const [data, setData] = useState<T | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    async function loadData() {
      // 检查缓存
      const cached = localStorage.getItem(key);
      if (cached) {
        setData(JSON.parse(cached));
        setLoading(false);
        return;
      }

      // 获取数据
      const result = await fetcher();
      localStorage.setItem(key, JSON.stringify(result));
      setData(result);
      setLoading(false);
    }

    loadData();
  }, [key]);

  return { data, loading };
}
```

### 5.2 进阶使用

#### 智能缓存 Hook

```typescript
"use client";

import { useState, useEffect, useCallback } from "react";

interface CacheOptions {
  ttl?: number;
  strategy?: "cache-first" | "network-first" | "stale-while-revalidate";
}

function useSmartCache<T>(
  key: string,
  fetcher: () => Promise<T>,
  options: CacheOptions = {}
) {
  const [data, setData] = useState<T | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<Error | null>(null);

  const { ttl = 300000, strategy = "cache-first" } = options;

  const loadData = useCallback(
    async (forceRefresh = false) => {
      try {
        setLoading(true);
        setError(null);

        // 检查缓存
        if (!forceRefresh) {
          const cached = localStorage.getItem(key);
          if (cached) {
            const { data: cachedData, timestamp } = JSON.parse(cached);

            if (Date.now() - timestamp < ttl) {
              setData(cachedData);
              setLoading(false);

              if (strategy === "stale-while-revalidate") {
                // 后台更新
                fetcher().then((freshData) => {
                  localStorage.setItem(
                    key,
                    JSON.stringify({
                      data: freshData,
                      timestamp: Date.now(),
                    })
                  );
                  setData(freshData);
                });
              }

              return;
            }
          }
        }

        // 获取新数据
        const freshData = await fetcher();
        localStorage.setItem(
          key,
          JSON.stringify({
            data: freshData,
            timestamp: Date.now(),
          })
        );
        setData(freshData);
      } catch (err) {
        setError(err as Error);

        // 网络失败时尝试使用缓存
        const cached = localStorage.getItem(key);
        if (cached) {
          const { data: cachedData } = JSON.parse(cached);
          setData(cachedData);
        }
      } finally {
        setLoading(false);
      }
    },
    [key, fetcher, ttl, strategy]
  );

  useEffect(() => {
    loadData();
  }, [loadData]);

  const refresh = useCallback(() => {
    return loadData(true);
  }, [loadData]);

  return { data, loading, error, refresh };
}

// 使用示例
export default function PostList() {
  const { data, loading, error, refresh } = useSmartCache(
    "posts",
    async () => {
      const res = await fetch("/api/posts");
      return res.json();
    },
    {
      ttl: 5 * 60 * 1000,
      strategy: "stale-while-revalidate",
    }
  );

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <button onClick={refresh}>Refresh</button>
      <ul>
        {data?.map((post: any) => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </div>
  );
}
```

```typescript
const CACHE_VERSION = "v1";

class VersionedCache {
  private getKey(key: string): string {

  get(key: string): any | null {

    const keys = Object.keys(localStorage);
    keys.forEach((key) => {
      if (!key.startsWith(CACHE_VERSION)) {
        localStorage.removeItem(key);
      }
    });
  }
}

export const versionedCache = new VersionedCache();
```

## 7. 常见问题

### 7.1 基础问题

#### 问题一: 应该选择哪种缓存方式?

**问题**: localStorage、sessionStorage、IndexedDB 应该选哪个?

**简短回答**: 小数据用 localStorage,会话数据用 sessionStorage,大数据用 IndexedDB。

**详细解释**:

localStorage 适合持久化的小数据,如用户设置。sessionStorage 适合会话期间的临时数据。IndexedDB 适合大量结构化数据。

**代码示例**:

```typescript
// localStorage - 用户设置
localStorage.setItem("theme", "dark");

// sessionStorage - 表单数据
sessionStorage.setItem("formData", JSON.stringify(formData));

// IndexedDB - 大量文章数据
await idbCache.set("posts", posts);
```

#### 问题二: 如何设置缓存过期时间?

**问题**: 如何让缓存在一定时间后失效?

**简短回答**: 保存时间戳,读取时检查是否过期。

**详细解释**:

在缓存数据时保存当前时间戳和 TTL,读取时检查是否超过 TTL。

**代码示例**:

```typescript
// 保存
const cacheItem = {
  data,
  timestamp: Date.now(),
  ttl: 5 * 60 * 1000, // 5 分钟
};
localStorage.setItem(key, JSON.stringify(cacheItem));

// 读取
const cached = JSON.parse(localStorage.getItem(key));
if (Date.now() - cached.timestamp > cached.ttl) {
  // 过期,删除缓存
  localStorage.removeItem(key);
}
```

#### 问题三: 如何清除所有缓存?

**问题**: 如何一次性清除所有缓存数据?

**简短回答**: 使用 clear() 方法或遍历删除。

**详细解释**:

localStorage 和 sessionStorage 有 clear() 方法。IndexedDB 需要遍历删除或删除整个数据库。

**代码示例**:

```typescript
// localStorage
localStorage.clear();

// sessionStorage
sessionStorage.clear();

// IndexedDB
await idbCache.clear();

// 清除特定前缀的缓存
const keys = Object.keys(localStorage);
keys.forEach((key) => {
  if (key.startsWith("app_")) {
    localStorage.removeItem(key);
  }
});
```

### 7.2 进阶问题

#### 问题四: 如何实现缓存优先策略?

**问题**: 如何优先使用缓存,缓存失效时才请求网络?

**简短回答**: 先检查缓存,没有或过期时再请求网络。

**详细解释**:

实现 cache-first 策略,先检查缓存是否存在且未过期,如果是则直接返回,否则请求网络并更新缓存。

**代码示例**:

```typescript
async function fetchWithCacheFirst(url: string) {
  const cached = cache.get(url);
  if (cached) return cached;

  const response = await fetch(url);
  const data = await response.json();

  cache.set(url, data);
  return data;
}
```

#### 问题五: 如何处理缓存容量限制?

**问题**: localStorage 满了怎么办?

**简短回答**: 实现 LRU 缓存淘汰策略。

**详细解释**:

当存储空间不足时,删除最久未使用的缓存项。可以记录每个缓存项的访问时间,空间不足时删除最旧的。

**代码示例**:

```typescript
class LRUCache {
  private cache: Map<string, any> = new Map();
  private maxSize: number;

  constructor(maxSize: number = 100) {
    this.maxSize = maxSize;
  }

  set(key: string, value: any): void {
    if (this.cache.has(key)) {
      this.cache.delete(key);
    }

    if (this.cache.size >= this.maxSize) {
      const firstKey = this.cache.keys().next().value;
      this.cache.delete(firstKey);
    }

    this.cache.set(key, value);
  }

  get(key: string): any | null {
    if (!this.cache.has(key)) return null;

    const value = this.cache.get(key);
    this.cache.delete(key);
    this.cache.set(key, value);

    return value;
  }
}
```
