# 离线数据处理策略

## 1. 概述与背景

### 1.1 什么是离线数据处理

离线数据处理是指在网络不可用或不稳定的情况下,应用仍能正常工作并处理数据的能力。在 Next.js 16 中,通过 Service Worker、IndexedDB 等技术可以实现完整的离线功能。

离线数据处理的核心特点:

- **离线可用**: 无网络时仍可使用
- **数据同步**: 网络恢复后自动同步
- **冲突解决**: 处理数据冲突
- **用户体验**: 无缝的离线体验
- **数据持久化**: 本地存储数据

### 1.2 为什么需要离线数据处理

在现代 Web 应用中,离线功能越来越重要:

- **网络不稳定**: 移动网络经常断开
- **用户体验**: 避免因网络问题中断工作
- **数据安全**: 本地保存避免数据丢失
- **性能提升**: 减少网络请求
- **PWA 支持**: 渐进式 Web 应用的基础

### 1.3 离线策略对比

**对比表格**:

| 策略            | 实现难度 | 数据一致性 | 适用场景 | 浏览器支持 |
| :-------------- | :------- | :--------- | :------- | :--------- |
| Cache First     | 简单     | 低         | 静态资源 | 所有浏览器 |
| Network First   | 简单     | 高         | 动态数据 | 所有浏览器 |
| Offline First   | 复杂     | 中等       | 离线应用 | 现代浏览器 |
| Background Sync | 复杂     | 高         | 数据同步 | 现代浏览器 |

### 2.2 IndexedDB 离线存储

#### 数据库初始化

```typescript
"use client";

class OfflineDB {
  private db: IDBDatabase | null = null;
  private dbName = "offline_app";
  private version = 1;

  async init(): Promise<void> {
    return new Promise((resolve, reject) => {
      const request = indexedDB.open(this.dbName, this.version);

      request.onerror = () => reject(request.error);
      request.onsuccess = () => {
        this.db = request.result;
        resolve();
      };

      request.onupgradeneeded = (event) => {
        const db = (event.target as IDBOpenDBRequest).result;

        // 创建对象存储
        if (!db.objectStoreNames.contains("posts")) {
          const postStore = db.createObjectStore("posts", { keyPath: "id" });
          postStore.createIndex("timestamp", "timestamp", { unique: false });
          postStore.createIndex("synced", "synced", { unique: false });
        }

        if (!db.objectStoreNames.contains("queue")) {
          db.createObjectStore("queue", { keyPath: "id", autoIncrement: true });
        }
      };
    });
  }

  async savePost(post: any): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["posts"], "readwrite");
      const store = transaction.objectStore("posts");
      const request = store.put({
        ...post,
        timestamp: Date.now(),
        synced: false,
      });

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async getPosts(): Promise<any[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["posts"], "readonly");
      const store = transaction.objectStore("posts");
      const request = store.getAll();

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async getUnsyncedPosts(): Promise<any[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["posts"], "readonly");
      const store = transaction.objectStore("posts");
      const index = store.index("synced");
      const request = index.getAll(false);

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async markAsSynced(id: string): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["posts"], "readwrite");
      const store = transaction.objectStore("posts");
      const getRequest = store.get(id);

      getRequest.onsuccess = () => {
        const post = getRequest.result;
        if (post) {
          post.synced = true;
          const putRequest = store.put(post);
          putRequest.onsuccess = () => resolve();
          putRequest.onerror = () => reject(putRequest.error);
        } else {
          resolve();
        }
      };

      getRequest.onerror = () => reject(getRequest.error);
    });
  }
}

export const offlineDB = new OfflineDB();
```

```typescript
import { useState, useEffect } from "react";
import { offlineDB } from "@/lib/offline-db";

interface Note {
  id: string;
  title: string;
  content: string;
  timestamp: number;
  synced: boolean;
}

export default function OfflineNotes() {
  const [notes, setNotes] = useState<Note[]>([]);
  const [title, setTitle] = useState("");
  const [content, setContent] = useState("");
  const [isOnline, setIsOnline] = useState(true);

  useEffect(() => {
    loadNotes();

    const handleOnline = () => {
      setIsOnline(true);
      syncNotes();
    };

    const handleOffline = () => {
      setIsOnline(false);
    };

    window.addEventListener("online", handleOnline);
    window.addEventListener("offline", handleOffline);

    return () => {
      window.removeEventListener("online", handleOnline);
      window.removeEventListener("offline", handleOffline);
    };
  }, []);

  const loadNotes = async () => {
    const loadedNotes = await offlineDB.getPosts();
    setNotes(loadedNotes);
  };

  const saveNote = async () => {
    const note: Note = {
      id: Date.now().toString(),
      title,
      content,
      timestamp: Date.now(),
      synced: false,
    };

    await offlineDB.savePost(note);
    setNotes((prev) => [note, ...prev]);
    setTitle("");
    setContent("");

    if (isOnline) {
      await syncNote(note);
    }
  };

  const syncNote = async (note: Note) => {
    try {
      const response = await fetch("/api/notes", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(note),
      });

      if (response.ok) {
        await offlineDB.markAsSynced(note.id);
        setNotes((prev) =>
          prev.map((n) => (n.id === note.id ? { ...n, synced: true } : n))
        );
      }
    } catch (error) {
      console.error("Sync failed:", error);
    }
  };

  const syncNotes = async () => {
    const unsyncedNotes = await offlineDB.getUnsyncedPosts();
    for (const note of unsyncedNotes) {
      await syncNote(note);
    }
  };

  return (
    <div>
      <div>
        <span>Status: {isOnline ? "Online" : "Offline"}</span>
      </div>
      <div>
        <input
          value={title}
          onChange={(e) => setTitle(e.target.value)}
          placeholder="Title"
        />
        <textarea
          value={content}
          onChange={(e) => setContent(e.target.value)}
          placeholder="Content"
        />
        <button onClick={saveNote}>Save Note</button>
      </div>
      <div>
        <h2>Notes</h2>
        {notes.map((note) => (
          <div key={note.id}>
            <h3>{note.title}</h3>
            <p>{note.content}</p>
            <span>{note.synced ? "Synced" : "Not synced"}</span>
          </div>
        ))}
      </div>
    </div>
  );
}
```

```typescript
"use client";

interface QueueItem {
  id?: number;
  url: string;
  method: string;
  data: any;
  timestamp: number;
  retries: number;
}

class OfflineQueue {
  private db: IDBDatabase | null = null;

  async init(): Promise<void> {
    return new Promise((resolve, reject) => {
      const request = indexedDB.open("offline_queue", 1);

      request.onerror = () => reject(request.error);
      request.onsuccess = () => {
        this.db = request.result;
        resolve();
      };

      request.onupgradeneeded = (event) => {
        const db = (event.target as IDBOpenDBRequest).result;
        if (!db.objectStoreNames.contains("queue")) {
          db.createObjectStore("queue", { keyPath: "id", autoIncrement: true });
        }
      };
    });
  }

  async add(item: Omit<QueueItem, "id">): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["queue"], "readwrite");
      const store = transaction.objectStore("queue");
      const request = store.add(item);

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async getAll(): Promise<QueueItem[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["queue"], "readonly");
      const store = transaction.objectStore("queue");
      const request = store.getAll();

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async remove(id: number): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["queue"], "readwrite");
      const store = transaction.objectStore("queue");
      const request = store.delete(id);

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async processQueue(): Promise<void> {
    const items = await this.getAll();

    for (const item of items) {
      try {
        const response = await fetch(item.url, {
          method: item.method,
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify(item.data),
        });

        if (response.ok) {
          await this.remove(item.id!);
        } else if (item.retries < 3) {
          // 更新重试次数
          item.retries++;
        } else {
          // 达到最大重试次数,移除
          await this.remove(item.id!);
        }
      } catch (error) {
        console.error("Queue processing failed:", error);
      }
    }
  }
}

export const offlineQueue = new OfflineQueue();

// 使用示例
export default function OfflineForm() {
  const [formData, setFormData] = useState({ name: "", email: "" });
  const [isOnline, setIsOnline] = useState(true);

  useEffect(() => {
    const handleOnline = async () => {
      setIsOnline(true);
      await offlineQueue.processQueue();
    };

    const handleOffline = () => {
      setIsOnline(false);
    };

    window.addEventListener("online", handleOnline);
    window.addEventListener("offline", handleOffline);

    return () => {
      window.removeEventListener("online", handleOnline);
      window.removeEventListener("offline", handleOffline);
    };
  }, []);

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();

    if (isOnline) {
      try {
        await fetch("/api/submit", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify(formData),
        });
      } catch (error) {
        await offlineQueue.add({
          url: "/api/submit",
          method: "POST",
          data: formData,
          timestamp: Date.now(),
          retries: 0,
        });
      }
    } else {
      await offlineQueue.add({
        url: "/api/submit",
        method: "POST",
        data: formData,
        timestamp: Date.now(),
        retries: 0,
      });
    }

    setFormData({ name: "", email: "" });
  };

  return (
    <form onSubmit={handleSubmit}>
      <p>Status: {isOnline ? "Online" : "Offline"}</p>
      <input
        value={formData.name}
        onChange={(e) => setFormData({ ...formData, name: e.target.value })}

      />

        onChange={(e) => setFormData({ ...formData, email: e.target.value })}

      />
```

```typescript
export default function OfflineImageUpload() {

      // 保存到 IndexedDB

      // 尝试上传
      if (navigator.onLine) {
        await uploadImage(image);
      }
    }

    const db = await openImageDB();
    const transaction = db.transaction(["images"], "readwrite");
    const store = transaction.objectStore("images");

    await store.put({
      id: image.id,
      file: image.file,
      uploaded: false,
    });
  };

  const uploadImage = async (image: ImageUpload) => {
    const formData = new FormData();
    formData.append("file", image.file);

    try {
      const response = await fetch("/api/upload", {
        method: "POST",
        body: formData,

      if (response.ok) {
        setImages((prev) =>
          prev.map((img) =>
            img.id === image.id ? { ...img, uploaded: true } : img
```

### 4.2 IndexedDB API

#### 数据库配置

```typescript
interface IndexedDBConfig {
  name: string;
  version: number;
  stores: {
    name: string;
    keyPath: string;
    autoIncrement?: boolean;
    indexes?: {
      name: string;
      keyPath: string;
      unique?: boolean;
    }[];
  }[];
}

class IndexedDBManager {
  private config: IndexedDBConfig;
  private db: IDBDatabase | null = null;

  constructor(config: IndexedDBConfig) {
    this.config = config;
  }

  async open(): Promise<IDBDatabase> {
    return new Promise((resolve, reject) => {
      const request = indexedDB.open(this.config.name, this.config.version);

      request.onerror = () => reject(request.error);
      request.onsuccess = () => {
        this.db = request.result;
        resolve(request.result);
      };

      request.onupgradeneeded = (event) => {
        const db = (event.target as IDBOpenDBRequest).result;

        this.config.stores.forEach((storeConfig) => {
          if (!db.objectStoreNames.contains(storeConfig.name)) {
            const store = db.createObjectStore(storeConfig.name, {
              keyPath: storeConfig.keyPath,
              autoIncrement: storeConfig.autoIncrement,
            });

            storeConfig.indexes?.forEach((index) => {
              store.createIndex(index.name, index.keyPath, {
                unique: index.unique,
              });
            });
          }
        });
      };
    });
  }

  async get<T>(storeName: string, key: any): Promise<T | undefined> {
    if (!this.db) await this.open();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([storeName], "readonly");
      const store = transaction.objectStore(storeName);
      const request = store.get(key);

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async put<T>(storeName: string, value: T): Promise<void> {
    if (!this.db) await this.open();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([storeName], "readwrite");
      const store = transaction.objectStore(storeName);
      const request = store.put(value);

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async delete(storeName: string, key: any): Promise<void> {
    if (!this.db) await this.open();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([storeName], "readwrite");
      const store = transaction.objectStore(storeName);
      const request = store.delete(key);

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async getAll<T>(storeName: string): Promise<T[]> {
    if (!this.db) await this.open();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction([storeName], "readonly");
      const store = transaction.objectStore(storeName);
      const request = store.getAll();

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }
}
```

```typescript
          const dataStore = db.createObjectStore("data", { keyPath: "id" });
          dataStore.createIndex("synced", "synced", { unique: false });
          dataStore.createIndex("timestamp", "timestamp", { unique: false });
        }

        if (!db.objectStoreNames.contains("operations")) {
          const opsStore = db.createObjectStore("operations", {
            keyPath: "id",
            autoIncrement: true,
          });
          opsStore.createIndex("type", "type", { unique: false });
          opsStore.createIndex("timestamp", "timestamp", { unique: false });
        }
      };
    });
  }

  async saveData(data: any): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["data"], "readwrite");
      const store = transaction.objectStore("data");
      const request = store.put({
        ...data,
        timestamp: Date.now(),
        synced: false,
      });

      request.onsuccess = () => resolve();
      request.onerror = () => reject(request.error);
    });
  }

  async getData(id: string): Promise<any> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["data"], "readonly");
      const store = transaction.objectStore("data");
      const request = store.get(id);

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async getAllData(): Promise<any[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["data"], "readonly");
      const store = transaction.objectStore("data");
      const request = store.getAll();

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async getUnsyncedData(): Promise<any[]> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["data"], "readonly");
      const store = transaction.objectStore("data");
      const index = store.index("synced");
      const request = index.getAll(false);

      request.onsuccess = () => resolve(request.result);
      request.onerror = () => reject(request.error);
    });
  }

  async markAsSynced(id: string): Promise<void> {
    if (!this.db) await this.init();

    return new Promise((resolve, reject) => {
      const transaction = this.db!.transaction(["data"], "readwrite");
      const store = transaction.objectStore("data");
      const getRequest = store.get(id);

      getRequest.onsuccess = () => {
        const data = getRequest.result;
        if (data) {
          data.synced = true;
          const putRequest = store.put(data);
          putRequest.onsuccess = () => resolve();
          putRequest.onerror = () => reject(putRequest.error);
        } else {
          resolve();
        }
      };

      getRequest.onerror = () => reject(getRequest.error);
    });
  }

  async sync(): Promise<void> {
    const unsyncedData = await this.getUnsyncedData();

    this.notifySyncStatus({
      pending: unsyncedData.length,
      syncing: true,
      lastSync: null,
    });

    for (const item of unsyncedData) {
      try {
        const response = await fetch("/api/sync", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify(item),
        });

        if (response.ok) {
          await this.markAsSynced(item.id);
        }
      } catch (error) {
        console.error("Sync failed:", error);
      }
    }

    this.notifySyncStatus({
      pending: 0,
      syncing: false,
      lastSync: Date.now(),
    });
  }

  onSyncStatusChange(callback: (status: SyncStatus) => void): void {
    this.syncCallbacks.push(callback);
  }

  private notifySyncStatus(status: SyncStatus): void {
    this.syncCallbacks.forEach((callback) => callback(status));
  }
}

export const offlineManager = new OfflineManager();
```

```typescript
"use client";

interface ConflictResolution {
  strategy: "server-wins" | "client-wins" | "last-write-wins" | "manual";
  resolver?: (server: any, client: any) => any;
}

class ConflictResolver {
  private strategy: ConflictResolution;

  constructor(strategy: ConflictResolution) {
    this.strategy = strategy;
  }

  resolve(serverData: any, clientData: any): any {
    switch (this.strategy.strategy) {
      case "server-wins":
        return serverData;

      case "client-wins":
        return clientData;

      case "last-write-wins":
        return serverData.timestamp > clientData.timestamp
          ? serverData
          : clientData;

      case "manual":
        if (this.strategy.resolver) {
          return this.strategy.resolver(serverData, clientData);
        }
        return serverData;

      default:
        return serverData;
    }
  }
}

// 使用示例
const resolver = new ConflictResolver({
  strategy: "manual",
  resolver: (server, client) => {
    // 自定义冲突解决逻辑

      ...server,
      ...client,
      mergedAt: Date.now(),
    };
  },
});
```

```typescript
export async function checkStorageQuota() {
  if ("storage" in navigator && "estimate" in navigator.storage) {
```

#### 批量操作

```typescript
"use client";

class BatchProcessor {
  private queue: any[] = [];
  private processing = false;
  private batchSize = 50;
  private batchDelay = 100;

  add(item: any): void {
    this.queue.push(item);
    this.process();
  }

  private async process(): Promise<void> {
    if (this.processing || this.queue.length === 0) {
      return;
    }

    this.processing = true;

    while (this.queue.length > 0) {
      const batch = this.queue.splice(0, this.batchSize);
      await this.processBatch(batch);
      await new Promise((resolve) => setTimeout(resolve, this.batchDelay));
    }

    this.processing = false;
  }

  private async processBatch(items: any[]): Promise<void> {
    // 批量处理逻辑
    const db = await openDB();
    const transaction = db.transaction(["data"], "readwrite");
    const store = transaction.objectStore("data");

    for (const item of items) {
      store.put(item);
    }

    return new Promise((resolve, reject) => {
      transaction.oncomplete = () => resolve();
      transaction.onerror = () => reject(transaction.error);
    });
  }
}

async function openDB(): Promise<IDBDatabase> {
  return new Promise((resolve, reject) => {
    const request = indexedDB.open("batch_db", 1);
    request.onsuccess = () => resolve(request.result);
    request.onerror = () => reject(request.error);
  });
}
```

## 7. 常见问题

### 7.1 基础问题

#### 问题一: 如何选择离线存储方案?

**问题**: localStorage、IndexedDB、Cache API 应该选哪个?

**简短回答**: 小数据用 localStorage,大数据用 IndexedDB,静态资源用 Cache API。

**详细解释**:

localStorage 适合存储少量配置数据(5MB 限制),IndexedDB 适合存储大量结构化数据,Cache API 适合缓存网络请求和静态资源。

**代码示例**:

```typescript
// localStorage - 配置数据
localStorage.setItem("theme", "dark");

// IndexedDB - 大量数据
await db.put("posts", { id: 1, title: "Post 1" });

// Cache API - 静态资源
await caches.open("v1").then((cache) => cache.add("/styles.css"));
```

#### 问题二: 如何处理离线时的表单提交?

**问题**: 用户离线时提交表单,如何保证数据不丢失?

**简短回答**: 将表单数据存入队列,网络恢复后自动提交。

**详细解释**:

使用 IndexedDB 存储待提交的表单数据,监听 online 事件,网络恢复后遍历队列提交数据。

**代码示例**:

```typescript
// 离线时保存到队列
await offlineQueue.add({
  url: "/api/submit",
  method: "POST",
  data: formData,
});

// 网络恢复后自动提交
window.addEventListener("online", async () => {
  await offlineQueue.processQueue();
});
```

#### 问题三: 如何检测网络状态?

**问题**: 如何准确检测用户是否在线?

**简短回答**: 使用 navigator.onLine 和 online/offline 事件。

**详细解释**:

navigator.onLine 提供当前网络状态,online/offline 事件监听状态变化。但这只能检测网络连接,不能检测实际网络可用性。

**代码示例**:

```typescript
const [isOnline, setIsOnline] = useState(navigator.onLine);

useEffect(() => {
  window.addEventListener("online", () => setIsOnline(true));
  window.addEventListener("offline", () => setIsOnline(false));
}, []);
```

### 7.2 进阶问题

#### 问题四: 如何处理数据冲突?

**问题**: 离线修改的数据与服务器数据冲突怎么办?

**简短回答**: 使用时间戳或版本号,实现冲突解决策略。

**详细解释**:

为每条数据添加时间戳或版本号,同步时比较版本,根据策略(服务器优先、客户端优先、最后写入优先)解决冲突。

**代码示例**:

```typescript

  }
  return client; // 客户端数据更新
};
```

**问题**: 大量离线数据如何避免性能问题?
