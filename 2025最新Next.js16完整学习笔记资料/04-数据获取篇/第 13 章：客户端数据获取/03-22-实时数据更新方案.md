# 实时数据更新方案

## 1. 概述与背景

### 1.1 什么是实时数据更新

实时数据更新是指在 Web 应用中实现服务器到客户端的即时数据推送,让用户能够实时看到最新的数据变化。在 Next.js 16 中,有多种方式实现实时数据更新。

实时数据更新的核心特点:

- **即时性**: 数据变化立即推送到客户端
- **双向通信**: 服务器和客户端可以互相发送消息
- **低延迟**: 减少数据更新的延迟
- **高效性**: 避免频繁的轮询请求
- **可靠性**: 保证消息的送达

### 1.2 为什么需要实时数据更新

在现代 Web 应用中,实时数据更新越来越重要:

- **聊天应用**: 即时消息推送
- **协作工具**: 多人实时编辑
- **数据监控**: 实时数据展示
- **通知系统**: 即时通知推送
- **在线游戏**: 实时状态同步

### 1.3 实时方案对比

**对比表格**:

| 方案         | 双向通信 | 实时性 | 复杂度 | 浏览器支持 | 适用场景   |
| :----------- | :------- | :----- | :----- | :--------- | :--------- |
| Polling      | 否       | 低     | 简单   | 所有浏览器 | 低频更新   |
| Long Polling | 否       | 中等   | 中等   | 所有浏览器 | 中频更新   |
| SSE          | 否       | 高     | 简单   | 现代浏览器 | 服务器推送 |
| WebSocket    | 是       | 最高   | 复杂   | 现代浏览器 | 双向通信   |

```typescript
"use client";

import { useState, useEffect, useRef } from "react";

export default function AdaptivePolling() {
  const [data, setData] = useState(null);
  const [interval, setInterval] = useState(5000);
  const timeoutRef = useRef<NodeJS.Timeout>();

  useEffect(() => {
    const fetchData = async () => {
      const start = Date.now();

      try {
        const response = await fetch("/api/data");
        const newData = await response.json();

        // 如果数据有变化,缩短轮询间隔
        if (JSON.stringify(newData) !== JSON.stringify(data)) {
          setInterval((prev) => Math.max(1000, prev / 2));
        } else {
          // 如果数据没变化,延长轮询间隔
          setInterval((prev) => Math.min(30000, prev * 1.5));
        }

        setData(newData);
      } catch (error) {
        console.error("Fetch error:", error);
      }

      // 下次轮询
      timeoutRef.current = setTimeout(fetchData, interval);
    };

    fetchData();

    return () => {
      if (timeoutRef.current) {
        clearTimeout(timeoutRef.current);
      }
    };
  }, [interval]);

  return <div>{JSON.stringify(data)}</div>;
}
```

```typescript
"use client";

import { useState, useEffect } from "react";

export default function LongPolling() {
  const [messages, setMessages] = useState<any[]>([]);

  useEffect(() => {
    let isActive = true;

    const longPoll = async () => {
      while (isActive) {
        try {
          const response = await fetch("/api/long-poll");
          const data = await response.json();

          if (isActive && data) {
            setMessages((prev) => [...prev, data]);
          }
        } catch (error) {
          console.error("Long poll error:", error);
          // 错误时等待一段时间再重试
          await new Promise((resolve) => setTimeout(resolve, 5000));
        }
      }
    };

    longPoll();

    return () => {
      isActive = false;
    };
  }, []);

  return (
    <div>
      <h2>Messages</h2>
      <ul>
        {messages.map((msg, index) => (
          <li key={index}>{msg.text}</li>
        ))}
      </ul>
    </div>
  );
}
```

```typescript
// app/api/sse/route.ts
import { NextRequest } from "next/server";

export async function GET(request: NextRequest) {
  const encoder = new TextEncoder();

  const stream = new ReadableStream({
    start(controller) {
      const sendEvent = (data: any) => {
        const message = `data: ${JSON.stringify(data)}\n\n`;
        controller.enqueue(encoder.encode(message));
      };

      // 每 2 秒发送一次更新
      const interval = setInterval(() => {
        sendEvent({
          id: Date.now().toString(),
          message: `Update at ${new Date().toISOString()}`,
          timestamp: Date.now(),
        });
      }, 2000);

      // 清理
      request.signal.addEventListener("abort", () => {
        clearInterval(interval);
        controller.close();
      });
    },
  });

  return new Response(stream, {
    headers: {
      "Content-Type": "text/event-stream",
      "Cache-Control": "no-cache",
      Connection: "keep-alive",
    },
  });
}
```

```typescript
import { useState, useEffect, useRef, useCallback } from "react";

export default function WebSocketWithReconnect() {
  const [messages, setMessages] = useState<any[]>([]);
  const [status, setStatus] = useState<
    "connecting" | "connected" | "disconnected"
  >("connecting");
  const wsRef = useRef<WebSocket | null>(null);
  const reconnectTimeoutRef = useRef<NodeJS.Timeout>();
  const reconnectAttemptsRef = useRef(0);

  const connect = useCallback(() => {
    const ws = new WebSocket("ws://localhost:3001");
    wsRef.current = ws;

    ws.onopen = () => {
      setStatus("connected");
      reconnectAttemptsRef.current = 0;
    };

    ws.onmessage = (event) => {
      const message = JSON.parse(event.data);
      setMessages((prev) => [...prev, message]);

    ws.onerror = (error) => {
      console.error("WebSocket error:", error);
    };

    ws.onclose = () => {
      setStatus("disconnected");

      // 自动重连
      const delay = Math.min(
        1000 * Math.pow(2, reconnectAttemptsRef.current),
        30000
      );

        reconnectAttemptsRef.current++;
        connect();
      }, delay);
    };
  }, []);

  useEffect(() => {
    connect();

    return () => {
      if (reconnectTimeoutRef.current) {
        clearTimeout(reconnectTimeoutRef.current);
      }
      if (wsRef.current) {
        wsRef.current.close();
      }
    };
  }, [connect]);

  return (
    <div>
      <p>Status: {status}</p>
      <ul>
        {messages.map((msg, index) => (
          <li key={index}>{msg.text}</li>
        ))}
      </ul>
```

```typescript
interface WebSocketConfig {
  url: string;
  protocols?: string | string[];
  reconnect?: boolean;
  reconnectInterval?: number;
  maxReconnectAttempts?: number;
  heartbeatInterval?: number;
}

class WebSocketClient {
  private ws: WebSocket | null = null;
  private config: WebSocketConfig;
  private reconnectAttempts = 0;
  private heartbeatTimer?: NodeJS.Timeout;

  constructor(config: WebSocketConfig) {
    this.config = {
      reconnect: true,
      reconnectInterval: 1000,
      maxReconnectAttempts: 5,
      heartbeatInterval: 30000,
      ...config,
    };
  }

  connect(): void {
    this.ws = new WebSocket(this.config.url, this.config.protocols);

    this.ws.onopen = () => {
      this.reconnectAttempts = 0;
      this.startHeartbeat();
    };

    this.ws.onclose = () => {
      this.stopHeartbeat();
      if (
        this.config.reconnect &&
        this.reconnectAttempts < this.config.maxReconnectAttempts!
      ) {
        setTimeout(() => {
          this.reconnectAttempts++;
          this.connect();
        }, this.config.reconnectInterval! * Math.pow(2, this.reconnectAttempts));
      }
    };
  }

  private startHeartbeat(): void {
    this.heartbeatTimer = setInterval(() => {
      if (this.ws?.readyState === WebSocket.OPEN) {
        this.ws.send(JSON.stringify({ type: "ping" }));
      }
    }, this.config.heartbeatInterval);
  }

  private stopHeartbeat(): void {
    if (this.heartbeatTimer) {
      clearInterval(this.heartbeatTimer);
    }
  }

  send(data: any): void {
    if (this.ws?.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }

  close(): void {
    this.config.reconnect = false;
    this.ws?.close();
  }
}
```

```typescript
interface SSEConfig {
  url: string;
  withCredentials?: boolean;
  retry?: number;
}

class SSEClient {
  private eventSource: EventSource | null = null;
  private config: SSEConfig;
  private listeners: Map<string, ((data: any) => void)[]> = new Map();

  constructor(config: SSEConfig) {
    this.config = config;
  }

  connect(): void {
    this.eventSource = new EventSource(this.config.url, {
      withCredentials: this.config.withCredentials,
    });

    this.eventSource.onmessage = (event) => {
      const data = JSON.parse(event.data);
      this.emit("message", data);
    };

    this.eventSource.onerror = (error) => {
      this.emit("error", error);
    };
  }

  on(event: string, callback: (data: any) => void): void {
    if (!this.listeners.has(event)) {
      this.listeners.set(event, []);
    }
    this.listeners.get(event)!.push(callback);
  }

  private emit(event: string, data: any): void {
    const callbacks = this.listeners.get(event);
    if (callbacks) {
      callbacks.forEach((callback) => callback(data));
    }
  }

  close(): void {
    this.eventSource?.close();
  }
}
```

```typescript
"use client";

import { useState, useEffect, useRef, useCallback } from "react";

interface UseWebSocketOptions {
  onOpen?: () => void;
  onClose?: () => void;
  onMessage?: (data: any) => void;
  onError?: (error: Event) => void;
  reconnect?: boolean;
  reconnectInterval?: number;
}

export function useWebSocket(url: string, options: UseWebSocketOptions = {}) {
  const [status, setStatus] = useState<
    "connecting" | "connected" | "disconnected"
  >("connecting");
  const [messages, setMessages] = useState<any[]>([]);
  const wsRef = useRef<WebSocket | null>(null);
  const reconnectTimeoutRef = useRef<NodeJS.Timeout>();
  const reconnectAttemptsRef = useRef(0);

  const connect = useCallback(() => {
    const ws = new WebSocket(url);
    wsRef.current = ws;

    ws.onopen = () => {
      setStatus("connected");
      reconnectAttemptsRef.current = 0;
      options.onOpen?.();
    };

    ws.onmessage = (event) => {
      const data = JSON.parse(event.data);
      setMessages((prev) => [...prev, data]);
      options.onMessage?.(data);
    };

    ws.onerror = (error) => {
      options.onError?.(error);
    };

    ws.onclose = () => {
      setStatus("disconnected");
      options.onClose?.();

      if (options.reconnect) {
        const delay =
          (options.reconnectInterval || 1000) *
          Math.pow(2, reconnectAttemptsRef.current);
        reconnectTimeoutRef.current = setTimeout(() => {
          reconnectAttemptsRef.current++;
          connect();
        }, Math.min(delay, 30000));
      }
    };
  }, [url, options]);

  useEffect(() => {
    connect();

    return () => {
      if (reconnectTimeoutRef.current) {
        clearTimeout(reconnectTimeoutRef.current);
      }
      wsRef.current?.close();
    };
  }, [connect]);

  const send = useCallback((data: any) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(data));
    }
  }, []);

  return { status, messages, send };
}
```

```typescript
import { useEffect, useRef } from "react";

export function useWebSocketWithHeartbeat(url: string) {
  const wsRef = useRef<WebSocket | null>(null);
  const heartbeatTimerRef = useRef<NodeJS.Timeout>();
  const pongTimeoutRef = useRef<NodeJS.Timeout>();

  useEffect(() => {
    const ws = new WebSocket(url);
    wsRef.current = ws;

    const startHeartbeat = () => {
      heartbeatTimerRef.current = setInterval(() => {
        if (ws.readyState === WebSocket.OPEN) {
          ws.send(JSON.stringify({ type: "ping" }));

          // 等待 pong 响应
          pongTimeoutRef.current = setTimeout(() => {
            // 没有收到 pong,关闭连接
            ws.close();
          }, 5000);
        }
      }, 30000);
    };

    ws.onopen = () => {
      startHeartbeat();
    };

    ws.onmessage = (event) => {
      const data = JSON.parse(event.data);

      if (data.type === "pong") {
        // 收到 pong,清除超时
        if (pongTimeoutRef.current) {
          clearTimeout(pongTimeoutRef.current);
        }
      }
    };

    return () => {
      if (heartbeatTimerRef.current) {
        clearInterval(heartbeatTimerRef.current);
      }
      if (pongTimeoutRef.current) {
        clearTimeout(pongTimeoutRef.current);
      }
      ws.close();
    };
  }, [url]);

  return wsRef;
}
```

```typescript
"use client";

class RetryStrategy {
  private attempts = 0;
  private maxAttempts: number;
  private baseDelay: number;
  private maxDelay: number;

  constructor(maxAttempts = 5, baseDelay = 1000, maxDelay = 30000) {
    this.maxAttempts = maxAttempts;
    this.baseDelay = baseDelay;
    this.maxDelay = maxDelay;
  }

  shouldRetry(): boolean {
    return this.attempts < this.maxAttempts;
  }

  getDelay(): number {
    const delay = this.baseDelay * Math.pow(2, this.attempts);
    return Math.min(delay, this.maxDelay);
  }

  increment(): void {
    this.attempts++;
  }

  reset(): void {
    this.attempts = 0;
  }
}
```

## 7. 常见问题

### 7.1 基础问题

#### 问题一: 应该选择哪种实时方案?

**问题**: Polling、SSE、WebSocket 应该选哪个?

**简短回答**: 单向推送用 SSE,双向通信用 WebSocket,简单场景用 Polling。

**详细解释**:

SSE 适合服务器到客户端的单向推送,如通知、实时数据。WebSocket 适合双向通信,如聊天、协作。Polling 最简单但效率低。

**代码示例**:

```typescript
// SSE - 服务器推送通知
const eventSource = new EventSource("/api/notifications");

// WebSocket - 聊天应用
const ws = new WebSocket("ws://localhost:3001/chat");

// Polling - 简单数据更新
setInterval(() => fetch("/api/data"), 5000);
```

#### 问题二: 如何处理连接断开?

**问题**: WebSocket 连接断开后如何自动重连?

**简短回答**: 监听 onclose 事件,使用指数退避策略重连。

**详细解释**:

在 onclose 事件中实现重连逻辑,使用指数退避避免频繁重连。

**代码示例**:

```typescript
ws.onclose = () => {
```

#### 问题三: 如何实现心跳检测?

**问题**: 如何检测 WebSocket 连接是否正常?

**简短回答**: 定期发送 ping 消息,等待 pong 响应。

**详细解释**:

每隔一段时间发送 ping 消息,如果在超时时间内没有收到 pong 响应,则认为连接断开。
