# 数据加密与脱敏

## 1. 数据加密基础

### 1.1 加密类型

**对称加密**:

使用相同的密钥进行加密和解密。

| 算法 | 密钥长度       | 特点            | 适用场景     |
| ---- | -------------- | --------------- | ------------ |
| AES  | 128/192/256 位 | 速度快,安全性高 | 大量数据加密 |
| DES  | 56 位          | 已不安全        | 不推荐使用   |
| 3DES | 168 位         | 较慢,逐渐淘汰   | 遗留系统     |

**非对称加密**:

使用公钥加密,私钥解密。

| 算法    | 密钥长度     | 特点                  | 适用场景          |
| ------- | ------------ | --------------------- | ----------------- |
| RSA     | 2048/4096 位 | 安全但慢              | 密钥交换,数字签名 |
| ECC     | 256/384 位   | 更短的密钥,相同安全性 | 移动设备,IoT      |
| Ed25519 | 256 位       | 快速,安全             | 数字签名          |

**哈希函数**:

单向加密,不可逆。

| 算法    | 输出长度 | 特点                | 适用场景            |
| ------- | -------- | ------------------- | ------------------- |
| SHA-256 | 256 位   | 安全,广泛使用       | 密码哈希,数据完整性 |
| SHA-512 | 512 位   | 更安全              | 高安全要求          |
| bcrypt  | 可变     | 自动加盐,可调整成本 | 密码存储            |
| Argon2  | 可变     | 抗 GPU 攻击         | 密码存储(推荐)      |

```typescript
// lib/encryption.ts
import crypto from "crypto";

const ALGORITHM = "aes-256-gcm";
const KEY = Buffer.from(process.env.ENCRYPTION_KEY!, "hex"); // 32 bytes
const IV_LENGTH = 16;
const AUTH_TAG_LENGTH = 16;

export function encrypt(text: string): string {
  const iv = crypto.randomBytes(IV_LENGTH);
  const cipher = crypto.createCipheriv(ALGORITHM, KEY, iv);

  let encrypted = cipher.update(text, "utf8", "hex");
  encrypted += cipher.final("hex");

  const authTag = cipher.getAuthTag();

  // 返回格式: iv:authTag:encrypted
  return `${iv.toString("hex")}:${authTag.toString("hex")}:${encrypted}`;
}

export function decrypt(encryptedData: string): string {
  const parts = encryptedData.split(":");

  if (parts.length !== 3) {
    throw new Error("Invalid encrypted data format");
  }

  const iv = Buffer.from(parts[0], "hex");
  const authTag = Buffer.from(parts[1], "hex");
  const encrypted = parts[2];

  const decipher = crypto.createDecipheriv(ALGORITHM, KEY, iv);
  decipher.setAuthTag(authTag);

  let decrypted = decipher.update(encrypted, "hex", "utf8");
  decrypted += decipher.final("utf8");

  return decrypted;
}

// 生成加密密钥
export function generateEncryptionKey(): string {
  return crypto.randomBytes(32).toString("hex");
}
```

```typescript
import crypto from "crypto";

const MASTER_KEY = process.env.MASTER_KEY!;

export function encryptEnvValue(value: string): string {
  const algorithm = "aes-256-cbc";
  const key = crypto.scryptSync(MASTER_KEY, "salt", 32);
  const iv = crypto.randomBytes(16);

  const cipher = crypto.createCipheriv(algorithm, key, iv);
  let encrypted = cipher.update(value, "utf8", "hex");
  encrypted += cipher.final("hex");

  return `${iv.toString("hex")}:${encrypted}`;
}

export function decryptEnvValue(encryptedValue: string): string {
  const algorithm = "aes-256-cbc";
  const key = crypto.scryptSync(MASTER_KEY, "salt", 32);

  const [ivHex, encrypted] = encryptedValue.split(":");
  const iv = Buffer.from(ivHex, "hex");

  const decipher = crypto.createDecipheriv(algorithm, key, iv);
  let decrypted = decipher.update(encrypted, "hex", "utf8");
  decrypted += decipher.final("utf8");

  return decrypted;
}
```

## 2. 数据脱敏

### 2.1 脱敏策略

**完全脱敏**:

完全隐藏敏感信息。

```typescript
// lib/masking.ts
export function maskEmail(email: string): string {
  const [username, domain] = email.split("@");

  if (username.length <= 2) {
    return `**@${domain}`;
  }

  const visibleChars = 2;
  const masked =
    username.slice(0, visibleChars) +
    "*".repeat(username.length - visibleChars);

  return `${masked}@${domain}`;
}

export function maskPhone(phone: string): string {
  // 保留前3位和后4位
  if (phone.length <= 7) {
    return "*".repeat(phone.length);
  }

  return phone.slice(0, 3) + "*".repeat(phone.length - 7) + phone.slice(-4);
}

export function maskCreditCard(cardNumber: string): string {
  // 只显示后4位
  return "*".repeat(cardNumber.length - 4) + cardNumber.slice(-4);
}

export function maskSSN(ssn: string): string {
  // 格式: XXX-XX-1234
  return "***-**-" + ssn.slice(-4);
}

export function maskName(name: string): string {
  const parts = name.split(" ");

  return parts
    .map((part, index) => {
      if (index === 0) {
        // 保留姓氏
        return part;
      }
      // 名字只显示首字母
      return part.charAt(0) + "*".repeat(part.length - 1);
    })
    .join(" ");
}

// 使用示例
console.log(maskEmail("john.doe@example.com")); // jo******@example.com
console.log(maskPhone("13812345678")); // 138****5678
console.log(maskCreditCard("1234567890123456")); // ************3456
console.log(maskSSN("123-45-6789")); // ***-**-6789
console.log(maskName("Zhang San")); // Zhang S**
```

```typescript
// lib/partial-masking.ts
export interface MaskingOptions {
  keepStart?: number;
  keepEnd?: number;
  maskChar?: string;
}

export function partialMask(
  value: string,
  options: MaskingOptions = {}
): string {
  const { keepStart = 0, keepEnd = 0, maskChar = "*" } = options;

  if (value.length <= keepStart + keepEnd) {
    return maskChar.repeat(value.length);
  }

  const start = value.slice(0, keepStart);
  const end = value.slice(-keepEnd);
  const middle = maskChar.repeat(value.length - keepStart - keepEnd);

  return start + middle + end;
}

// 使用示例
export function maskBankAccount(account: string): string {
  return partialMask(account, { keepStart: 4, keepEnd: 4 });
}

export function maskIDCard(idCard: string): string {
  return partialMask(idCard, { keepStart: 6, keepEnd: 4 });
}

export function maskAddress(address: string): string {
  // 保留省市,隐藏详细地址
  const parts = address.split(" ");

  if (parts.length <= 2) {
    return address;
  }

  return (
    parts.slice(0, 2).join(" ") +
    " " +
    "*".repeat(parts.slice(2).join(" ").length)
  );
}

console.log(maskBankAccount("6222021234567890")); // 6222************7890
console.log(maskIDCard("110101199001011234")); // 110101********1234
console.log(maskAddress("北京市 朝阳区 建国路1号")); // 北京市 朝阳区 *****
```

```typescript
export enum UserRole {
  ADMIN = "admin",
  MANAGER = "manager",
  USER = "user",
  GUEST = "guest",
}

export interface SensitiveData {
  email: string;
  phone: string;
  ssn: string;
  creditCard: string;
  address: string;
}

export function maskDataByRole(
  data: SensitiveData,
  role: UserRole
): Partial<SensitiveData> {
  switch (role) {
    case UserRole.ADMIN:
      // 管理员可以看到所有信息
      return data;

    case UserRole.MANAGER:
      // 经理可以看到部分信息
      return {
        email: data.email,
        phone: maskPhone(data.phone),
        ssn: maskSSN(data.ssn),
        creditCard: maskCreditCard(data.creditCard),
        address: maskAddress(data.address),
      };

    case UserRole.USER:
      // 普通用户只能看到脱敏后的信息
      return {
        email: maskEmail(data.email),
        phone: maskPhone(data.phone),
        ssn: maskSSN(data.ssn),
        creditCard: maskCreditCard(data.creditCard),
        address: maskAddress(data.address),
      };

    case UserRole.GUEST:
      // 访客看不到任何敏感信息
      return {
        email: "***",
        phone: "***",
        ssn: "***",
        creditCard: "***",
        address: "***",
      };

    default:
      return {};
  }
}
```

```typescript
// lib/tde.ts
import crypto from "crypto";

export class TransparentDataEncryption {
  private key: Buffer;
  private algorithm = "aes-256-gcm";

  constructor(masterKey: string) {
    this.key = Buffer.from(masterKey, "hex");
  }

  encryptRow(data: Record<string, any>): Record<string, any> {
    const encrypted: Record<string, any> = {};

    for (const [key, value] of Object.entries(data)) {
      if (this.shouldEncrypt(key)) {
        encrypted[key] = this.encryptValue(String(value));
      } else {
        encrypted[key] = value;
      }
    }

    return encrypted;
  }

  decryptRow(data: Record<string, any>): Record<string, any> {
    const decrypted: Record<string, any> = {};

    for (const [key, value] of Object.entries(data)) {
      if (this.shouldEncrypt(key) && typeof value === "string") {
        try {
          decrypted[key] = this.decryptValue(value);
        } catch {
          decrypted[key] = value;
        }
      } else {
        decrypted[key] = value;
      }
    }

    return decrypted;
  }

  private shouldEncrypt(fieldName: string): boolean {
    const sensitiveFields = ["ssn", "creditCard", "password", "apiKey"];
    return sensitiveFields.includes(fieldName);
  }

  private encryptValue(value: string): string {
    const iv = crypto.randomBytes(16);
    const cipher = crypto.createCipheriv(this.algorithm, this.key, iv);

    let encrypted = cipher.update(value, "utf8", "hex");
    encrypted += cipher.final("hex");

    const authTag = cipher.getAuthTag();

    return `${iv.toString("hex")}:${authTag.toString("hex")}:${encrypted}`;
  }

  private decryptValue(encryptedValue: string): string {
    const [ivHex, authTagHex, encrypted] = encryptedValue.split(":");

    const iv = Buffer.from(ivHex, "hex");
    const authTag = Buffer.from(authTagHex, "hex");

    const decipher = crypto.createDecipheriv(this.algorithm, this.key, iv);
    decipher.setAuthTag(authTag);

    let decrypted = decipher.update(encrypted, "hex", "utf8");
    decrypted += decipher.final("utf8");

    return decrypted;
  }
}
```

```typescript
// lib/key-rotation.ts
import crypto from "crypto";

export class KeyRotation {
  private currentKey: Buffer;
  private previousKey: Buffer | null;

  constructor(currentKey: string, previousKey?: string) {
    this.currentKey = Buffer.from(currentKey, "hex");
    this.previousKey = previousKey ? Buffer.from(previousKey, "hex") : null;
  }

  encrypt(text: string): string {
    const iv = crypto.randomBytes(16);
    const cipher = crypto.createCipheriv("aes-256-gcm", this.currentKey, iv);

    let encrypted = cipher.update(text, "utf8", "hex");
    encrypted += cipher.final("hex");

    const authTag = cipher.getAuthTag();

    // 添加版本标识
    return `v2:${iv.toString("hex")}:${authTag.toString("hex")}:${encrypted}`;
  }

  decrypt(encryptedData: string): string {
    const parts = encryptedData.split(":");
    const version = parts[0];

    if (version === "v2") {
      return this.decryptV2(parts.slice(1).join(":"));
    } else if (version === "v1" && this.previousKey) {
      return this.decryptV1(parts.slice(1).join(":"));
    } else {
      throw new Error("Unsupported encryption version");
    }
  }

  private decryptV2(data: string): string {
    const [ivHex, authTagHex, encrypted] = data.split(":");

    const iv = Buffer.from(ivHex, "hex");
    const authTag = Buffer.from(authTagHex, "hex");

    const decipher = crypto.createDecipheriv(
      "aes-256-gcm",
      this.currentKey,
      iv
    );
    decipher.setAuthTag(authTag);

    let decrypted = decipher.update(encrypted, "hex", "utf8");
    decrypted += decipher.final("utf8");

    return decrypted;
  }

  private decryptV1(data: string): string {
    if (!this.previousKey) {
      throw new Error("Previous key not available");
    }

    const [ivHex, authTagHex, encrypted] = data.split(":");

    const iv = Buffer.from(ivHex, "hex");
    const authTag = Buffer.from(authTagHex, "hex");

    const decipher = crypto.createDecipheriv(
      "aes-256-gcm",
      this.previousKey,
      iv
    );
    decipher.setAuthTag(authTag);

    let decrypted = decipher.update(encrypted, "hex", "utf8");
    decrypted += decipher.final("utf8");

    return decrypted;
  }

  // 重新加密旧数据
  async reencryptData() {
    if (!this.previousKey) {
      throw new Error("Previous key not available for re-encryption");
    }

    // 查找所有使用旧密钥加密的数据
    const users = await db.users.findMany({
      where: {
        ssn: {
          startsWith: "v1:",
        },
      },
    });

    // 重新加密
    for (const user of users) {
      const decrypted = this.decrypt(user.ssn);
      const reencrypted = this.encrypt(decrypted);

      await db.users.update({
        where: { id: user.id },
        data: { ssn: reencrypted },
      });
    }
  }
}

// 使用示例

  process.env.PREVIOUS_ENCRYPTION_KEY
);
```

```typescript
// lib/client-encryption.ts
import { webcrypto } from "crypto";

export class ClientEncryption {
  private async generateKey(): Promise<CryptoKey> {
    return await webcrypto.subtle.generateKey(
      {
        name: "AES-GCM",
        length: 256,
      },
      true,
      ["encrypt", "decrypt"]
    );
  }

  async encrypt(data: string, key: CryptoKey): Promise<string> {
    const iv = webcrypto.getRandomValues(new Uint8Array(12));
    const encodedData = new TextEncoder().encode(data);

    const encrypted = await webcrypto.subtle.encrypt(
      {
        name: "AES-GCM",
        iv,
      },
      key,
      encodedData
    );

    const encryptedArray = new Uint8Array(encrypted);
    const combined = new Uint8Array(iv.length + encryptedArray.length);
    combined.set(iv);
    combined.set(encryptedArray, iv.length);

    return Buffer.from(combined).toString("base64");
  }

  async decrypt(encryptedData: string, key: CryptoKey): Promise<string> {
    const combined = Buffer.from(encryptedData, "base64");
    const iv = combined.slice(0, 12);
    const data = combined.slice(12);

    const decrypted = await webcrypto.subtle.decrypt(
      {
        name: "AES-GCM",
        iv,
      },
      key,
      data
    );

    return new TextDecoder().decode(decrypted);
  }
}
```

```typescript
import { useState, useEffect } from "react";

interface SecureInputProps {
  value: string;
  onChange: (value: string) => void;
  type?: "text" | "password" | "email";
  mask?: boolean;
}

export function SecureInput({
  value,
  onChange,
  type = "text",
  mask = false,
}: SecureInputProps) {
  const [displayValue, setDisplayValue] = useState("");
  const [isFocused, setIsFocused] = useState(false);

  useEffect(() => {
    if (mask && !isFocused && value) {
      // 脱敏显示
      setDisplayValue("*".repeat(value.length));
    } else {
      setDisplayValue(value);
    }
  }, [value, mask, isFocused]);

  const handleChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    onChange(e.target.value);
  };

  return (
    <input
      type={type}
      value={displayValue}
      onChange={handleChange}
      onFocus={() => setIsFocused(true)}
      onBlur={() => setIsFocused(false)}
      autoComplete="off"
      data-lpignore="true" // 防止密码管理器
      className="secure-input"
    />
  );
}
```

```typescript
// lib/log-masking.ts
export interface LogMaskingConfig {
  patterns: Array<{
    name: string;
    regex: RegExp;
    replacement: string;
  }>;
}

export const defaultMaskingConfig: LogMaskingConfig = {
  patterns: [
    {
      name: "email",
      regex: /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b/g,
      replacement: "***@***.***",
    },
    {
      name: "phone",
      regex: /\b\d{3}[-.]?\d{3}[-.]?\d{4}\b/g,
      replacement: "***-***-****",
    },
    {
      name: "ssn",
      regex: /\b\d{3}-\d{2}-\d{4}\b/g,
      replacement: "***-**-****",
    },
    {
      name: "creditCard",
      regex: /\b\d{4}[-\s]?\d{4}[-\s]?\d{4}[-\s]?\d{4}\b/g,
      replacement: "****-****-****-****",
    },
    {
      name: "password",
      regex: /"password"\s*:\s*"[^"]+"/g,
      replacement: '"password":"***"',
    },
    {
      name: "token",
      regex: /"token"\s*:\s*"[^"]+"/g,
      replacement: '"token":"***"',
    },
  ],
};

export function maskLog(
  message: string,
  config: LogMaskingConfig = defaultMaskingConfig
): string {
  let masked = message;

  for (const pattern of config.patterns) {
    masked = masked.replace(pattern.regex, pattern.replacement);
  }

  return masked;
}

// 自定义 logger
export class SecureLogger {
  private config: LogMaskingConfig;

  constructor(config: LogMaskingConfig = defaultMaskingConfig) {
    this.config = config;
  }

  log(message: string, ...args: any[]) {
    const maskedMessage = maskLog(message, this.config);
    const maskedArgs = args.map((arg) =>
      typeof arg === "string" ? maskLog(arg, this.config) : arg
    );
    console.log(maskedMessage, ...maskedArgs);
  }

  error(message: string, ...args: any[]) {
    const maskedMessage = maskLog(message, this.config);
    const maskedArgs = args.map((arg) =>
      typeof arg === "string" ? maskLog(arg, this.config) : arg
    );
    console.error(maskedMessage, ...maskedArgs);
  }

  warn(message: string, ...args: any[]) {
    const maskedMessage = maskLog(message, this.config);
    const maskedArgs = args.map((arg) =>
      typeof arg === "string" ? maskLog(arg, this.config) : arg
    );
    console.warn(maskedMessage, ...maskedArgs);
  }
}
```

### 5.2 API 响应脱敏

```typescript
// lib/response-masking.ts
import { NextResponse } from "next/server";

export function maskResponse(data: any, userRole: string): any {
  if (Array.isArray(data)) {
    return data.map((item) => maskResponse(item, userRole));
  }

  if (typeof data === "object" && data !== null) {
    const masked: any = {};

    for (const [key, value] of Object.entries(data)) {
      if (shouldMaskField(key, userRole)) {
        masked[key] = maskValue(key, value);
      } else if (typeof value === "object") {
        masked[key] = maskResponse(value, userRole);
      } else {
        masked[key] = value;
      }
    }

    return masked;
  }

  return data;
}

function shouldMaskField(fieldName: string, userRole: string): boolean {
  const sensitiveFields = ["ssn", "creditCard", "password", "apiKey"];

  if (userRole === "admin") {
    return false; // 管理员可以看到所有字段
  }

  return sensitiveFields.includes(fieldName);
}

function maskValue(fieldName: string, value: any): string {
  if (typeof value !== "string") {
    return "***";
  }

  switch (fieldName) {
    case "ssn":
      return maskSSN(value);
    case "creditCard":
      return maskCreditCard(value);
    case "password":
    case "apiKey":
      return "***";
    default:
      return value;
  }
}

// API 路由中使用
// app/api/users/route.ts
export async function GET(request: Request) {
  const session = await getSession(request);
  const users = await db.users.findMany();

  const maskedUsers = maskResponse(users, session.role);

  return NextResponse.json(maskedUsers);
}
```

## 常见问题

### 1. 加密和哈希的区别?

**加密**:

- 可逆,可以解密
- 需要密钥
- 用于保护数据机密性
- 示例: AES, RSA

**哈希**:

- 不可逆,无法解密
- 不需要密钥
- 用于验证数据完整性
- 示例: SHA-256, bcrypt

```typescript
// 加密 - 可逆
const encrypted = encrypt("sensitive data");
const decrypted = decrypt(encrypted); // 'sensitive data'

// 哈希 - 不可逆
const hashed = hash("password");
// 无法从 hashed 恢复 'password'
// 只能验证
const isValid = verify("password", hashed); // true
```

### 2. 如何选择加密算法?

| 场景         | 推荐算法      | 原因                |
| ------------ | ------------- | ------------------- |
| 大量数据加密 | AES-256-GCM   | 快速,安全,支持认证  |
| 密钥交换     | RSA-2048/4096 | 非对称加密,安全     |
| 密码存储     | bcrypt/Argon2 | 自动加盐,可调整成本 |
| 数字签名     | Ed25519       | 快速,安全           |
| 数据完整性   | SHA-256       | 广泛支持,安全       |

### 3. 密钥应该如何管理?

```typescript
// 不要这样做
const SECRET_KEY = "1234567890abcdef"; // 硬编码

// 应该这样做

}
```

### 5. 前端加密是否安全?

前端加密可以提供额外的保护层,但不能替代后端加密:

```typescript
// 前端加密的局限性
// 1. JavaScript 代码可见
// 2. 密钥管理困难
// 3. 容易被绕过

// 正确的做法
// 1. 使用 HTTPS 传输
// 2. 前端加密作为额外保护
// 3. 后端必须再次加密

// 前端
const encrypted = await clientEncrypt(sensitiveData);
await fetch("/api/save", {
  method: "POST",
  body: JSON.stringify({ data: encrypted }),
});

// 后端
export async function POST(request: Request) {
  const { data } = await request.json();

  // 解密前端加密的数据
  const decrypted = await clientDecrypt(data);

  // 使用服务器密钥重新加密
  const serverEncrypted = encrypt(decrypted);

  await db.save({ data: serverEncrypted });
}
```

## 注意事项

### 1. 加密密钥管理

1. **永远不要硬编码密钥**

   - 使用环境变量
   - 使用密钥管理服务(AWS KMS, Azure Key Vault)
   - 定期轮换密钥

2. **密钥存储**

   - 加密存储密钥
   - 限制访问权限
   - 备份密钥(安全存储)

3. **密钥轮换**
   - 定期更换密钥
   - 保留旧密钥用于解密历史数据
   - 逐步迁移到新密钥

### 2. 性能考虑

1. **加密开销**

   - 加密会增加 CPU 使用
   - 考虑使用硬件加速
   - 只加密必要的字段

2. **缓存策略**

   - 解密后的数据可以缓存
   - 设置合理的缓存过期时间
   - 注意内存安全

3. **批量操作**
   - 批量加密/解密可以提高效率
   - 使用异步处理
   - 考虑使用队列

### 3. 合规要求

1. **了解法规**

   - GDPR(欧盟)
   - CCPA(加州)
   - HIPAA(医疗)
   - PCI DSS(支付卡)

2. **数据分类**

   - 识别敏感数据
   - 确定保护级别
   - 实施相应措施

3. **审计日志**
   - 记录所有加密/解密操作
   - 记录密钥访问
   - 定期审查日志

### 4. 脱敏策略

1. **选择合适的脱敏方法**

   - 完全脱敏: 高安全性,低可用性
   - 部分脱敏: 平衡安全性和可用性
   - 动态脱敏: 根据权限调整

2. **一致性**

   - 相同数据使用相同脱敏规则
   - 文档化脱敏策略
   - 定期审查和更新

3. **测试**
   - 验证脱敏效果
   - 确保不影响业务逻辑
   - 性能测试
