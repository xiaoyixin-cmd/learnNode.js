# 身份认证安全

## 1. 身份认证基础

### 1.1 认证与授权的区别

**认证(Authentication)** 和 **授权(Authorization)** 是两个不同的概念:

| 概念 | 定义         | 问题        | 示例               |
| ---- | ------------ | ----------- | ------------------ |
| 认证 | 验证用户身份 | 你是谁?     | 用户名密码登录     |
| 授权 | 验证用户权限 | 你能做什么? | 管理员可以删除用户 |

**认证流程**:

```
用户 -> 提供凭证 -> 系统验证 -> 颁发令牌 -> 访问资源
```

**授权流程**:

```
用户 -> 携带令牌 -> 系统验证令牌 -> 检查权限 -> 允许/拒绝访问
```

## 2. 密码安全

### 2.1 密码哈希

永远不要以明文形式存储密码。使用强哈希算法。

### 2.2 密码强度验证

```typescript
// lib/password-validator.ts
export interface PasswordStrength {
  score: number;
  feedback: string[];
  isStrong: boolean;
}

export function validatePasswordStrength(password: string): PasswordStrength {
  const feedback: string[] = [];
  let score = 0;

  // 长度检查
  if (password.length >= 8) score += 1;
  if (password.length >= 12) score += 1;
  if (password.length >= 16) score += 1;

  if (password.length < 8) {
    feedback.push("密码至少需要8个字符");
  }

  // 复杂度检查
  if (/[a-z]/.test(password)) score += 1;
  else feedback.push("需要包含小写字母");

  if (/[A-Z]/.test(password)) score += 1;
  else feedback.push("需要包含大写字母");

  if (/[0-9]/.test(password)) score += 1;
  else feedback.push("需要包含数字");

  if (/[^a-zA-Z0-9]/.test(password)) score += 1;
  else feedback.push("需要包含特殊字符");

  // 常见密码检查
  const commonPasswords = ["password", "123456", "qwerty", "admin"];
  if (commonPasswords.some((p) => password.toLowerCase().includes(p))) {
    score -= 2;
    feedback.push("不要使用常见密码");
  }

  return {
    score,
    feedback,
    isStrong: score >= 5,
  };
}

// 使用示例
export function isStrongPassword(password: string): boolean {
  const result = validatePasswordStrength(password);
  return result.isStrong;
}
```

```typescript
    .sign(secret);

  try {
    const { payload } = await jwtVerify(token, secret, {

      audience,
```

## 4. 会话管理

### 4.1 安全的会话存储

```typescript
// lib/session.ts
import { Redis } from "@upstash/redis";
import crypto from "crypto";

const redis = new Redis({
  url: process.env.UPSTASH_REDIS_URL!,
  token: process.env.UPSTASH_REDIS_TOKEN!,
});

export interface SessionData {
  userId: string;
  username: string;
  role: string;
  createdAt: number;
  lastActivity: number;
  ipAddress: string;
  userAgent: string;
}

export async function createSession(
  userId: string,
  userData: Omit<SessionData, "userId" | "createdAt" | "lastActivity">
): Promise<string> {
  const sessionId = crypto.randomBytes(32).toString("hex");

  const sessionData: SessionData = {
    userId,
    ...userData,
    createdAt: Date.now(),
    lastActivity: Date.now(),
  };

  // 会话有效期7天
  await redis.setex(
    `session:${sessionId}`,
    7 * 24 * 60 * 60,
    JSON.stringify(sessionData)
  );

  // 记录用户的所有会话
  await redis.sadd(`user:${userId}:sessions`, sessionId);

  return sessionId;
}

export async function getSession(
  sessionId: string
): Promise<SessionData | null> {
  const data = await redis.get(`session:${sessionId}`);

  if (!data) return null;

  const session = JSON.parse(data as string) as SessionData;

  // 更新最后活动时间
  session.lastActivity = Date.now();
  await redis.setex(
    `session:${sessionId}`,
    7 * 24 * 60 * 60,
    JSON.stringify(session)
  );

  return session;
}

export async function deleteSession(sessionId: string) {
  const session = await getSession(sessionId);

  if (session) {
    await redis.srem(`user:${session.userId}:sessions`, sessionId);
  }

  await redis.del(`session:${sessionId}`);
}

export async function deleteAllUserSessions(userId: string) {
  const sessions = await redis.smembers(`user:${userId}:sessions`);

  if (sessions && sessions.length > 0) {
    const pipeline = redis.pipeline();

    sessions.forEach((sessionId) => {
      pipeline.del(`session:${sessionId}`);
    });

    pipeline.del(`user:${userId}:sessions`);

    await pipeline.exec();
  }
}
```

```typescript
import type { NextRequest } from "next/server";
import { getSession } from "./lib/session";

export async function middleware(request: NextRequest) {
  const sessionId = request.cookies.get("session")?.value;

  if (!sessionId) {
    return NextResponse.redirect(new URL("/auth/signin", request.url));
  }

  const session = await getSession(sessionId);

  if (!session) {
    const response = NextResponse.redirect(
      new URL("/auth/signin", request.url)
    );
    response.cookies.delete("session");
    return response;
  }

  // 检查会话是否过期(30分钟无活动)
  const inactiveTime = Date.now() - session.lastActivity;
  const maxInactiveTime = 30 * 60 * 1000; // 30 minutes

  if (inactiveTime > maxInactiveTime) {
    await deleteSession(sessionId);
    const response = NextResponse.redirect(
      new URL("/auth/signin", request.url)
    );
    response.cookies.delete("session");
    return response;
  }

  // 将用户信息添加到请求头
  const response = NextResponse.next();
  response.headers.set("X-User-Id", session.userId);
  response.headers.set("X-User-Role", session.role);

  return response;
}
```

### 5.1 TOTP 实现

```typescript
// lib/totp.ts
import * as OTPAuth from "otpauth";
import QRCode from "qrcode";

export async function generateTOTPSecret(username: string) {
  const totp = new OTPAuth.TOTP({
    issuer: "YourApp",
    label: username,
    algorithm: "SHA1",
    digits: 6,
    period: 30,
  });

  const secret = totp.secret.base32;
  const uri = totp.toString();

  // 生成二维码
  const qrCode = await QRCode.toDataURL(uri);

  return { secret, qrCode, uri };
}

export function verifyTOTP(token: string, secret: string): boolean {
  const totp = new OTPAuth.TOTP({
    secret: OTPAuth.Secret.fromBase32(secret),
    algorithm: "SHA1",
    digits: 6,
    period: 30,
  });

  // 允许前后30秒的时间窗口
  const delta = totp.validate({ token, window: 1 });

  return delta !== null;
}
```

### 5.2 备用码生成

```typescript
// lib/backup-codes.ts
import crypto from "crypto";
import bcrypt from "bcryptjs";

export function generateBackupCodes(count: number = 10): string[] {
  const codes: string[] = [];

  for (let i = 0; i < count; i++) {
    const code = crypto.randomBytes(4).toString("hex").toUpperCase();
    codes.push(code);
  }

  return codes;
}

export async function hashBackupCodes(codes: string[]): Promise<string[]> {
  return Promise.all(codes.map((code) => bcrypt.hash(code, 10)));
}

export async function verifyBackupCode(
  code: string,
  hashedCodes: string[]
): Promise<number> {
  for (let i = 0; i < hashedCodes.length; i++) {
    const isValid = await bcrypt.compare(code, hashedCodes[i]);
    if (isValid) return i;
  }
  return -1;
}

// app/api/auth/mfa/backup/route.ts
import { NextResponse } from "next/server";

export async function POST(request: Request) {
  const { userId, backupCode } = await request.json();

  const user = await findUserById(userId);

  if (!user || !user.mfaEnabled) {
    return NextResponse.json({ error: "MFA未启用" }, { status: 400 });
  }

  const backupCodes = await getBackupCodes(userId);
  const codeIndex = await verifyBackupCode(backupCode, backupCodes);

  if (codeIndex === -1) {
    return NextResponse.json({ error: "备用码无效" }, { status: 400 });
  }

  // 删除已使用的备用码
  await removeBackupCode(userId, codeIndex);

  // 创建会话
  const sessionId = await createSession(userId, {
    username: user.username,
    role: user.role,
    ipAddress: request.headers.get("x-forwarded-for") || "",
    userAgent: request.headers.get("user-agent") || "",
  });

}
```

### 5.3 MFA 登录流程

```typescript
// app/api/auth/login-mfa/route.ts
import { NextResponse } from "next/server";
import { verifyPassword } from "@/lib/password";
import { verifyTOTP } from "@/lib/totp";

export async function POST(request: Request) {
  const { username, password, mfaToken } = await request.json();

  // 第一步: 验证用户名和密码
  const user = await findUserByUsername(username);

  if (!user) {
    return NextResponse.json({ error: "用户名或密码错误" }, { status: 401 });
  }

  const isPasswordValid = await verifyPassword(password, user.password);

  if (!isPasswordValid) {
    return NextResponse.json({ error: "用户名或密码错误" }, { status: 401 });
  }

  // 第二步: 检查是否启用MFA
  if (!user.mfaEnabled) {
    // 直接创建会话
    const sessionId = await createSession(user.id, {
      username: user.username,
      role: user.role,
      ipAddress: request.headers.get("x-forwarded-for") || "",
      userAgent: request.headers.get("user-agent") || "",
    });

    return NextResponse.json({ success: true, sessionId });
  }

  // 第三步: 验证MFA令牌
  if (!mfaToken) {
    return NextResponse.json(
      { error: "需要MFA验证", requireMFA: true },
      { status: 401 }
    );
  }

  const isMFAValid = verifyTOTP(mfaToken, user.mfaSecret);

  if (!isMFAValid) {
    return NextResponse.json({ error: "MFA验证码无效" }, { status: 401 });
  }

  // 创建会话
  const sessionId = await createSession(user.id, {
    username: user.username,
    role: user.role,
    ipAddress: request.headers.get("x-forwarded-for") || "",
    userAgent: request.headers.get("user-agent") || "",
  });

  return NextResponse.json({ success: true, sessionId });
}
```

```typescript
// lib/auth-middleware.ts

import type { NextRequest } from "next/server";
import { getSession } from "./session";

export function requireAuth() {
  return async (request: NextRequest) => {
    const sessionId = request.cookies.get("session")?.value;

    if (!sessionId) {
      return NextResponse.json({ error: "未授权" }, { status: 401 });
    }

    const session = await getSession(sessionId);

    if (!session) {
      return NextResponse.json({ error: "会话无效" }, { status: 401 });
    }

    return session;
  };
}

export function requirePermission(permission: Permission) {
  return async (request: NextRequest) => {
    const session = await requireAuth()(request);

    if (session instanceof NextResponse) {
      return session;
    }

    const userRole = session.role as Role;

    if (!hasPermission(userRole, permission)) {
      return NextResponse.json({ error: "权限不足" }, { status: 403 });
    }

    return session;
  };
}

export function requireRole(role: Role) {
  return async (request: NextRequest) => {
    const session = await requireAuth()(request);

    if (session instanceof NextResponse) {
      return session;
    }

    if (session.role !== role) {
      return NextResponse.json({ error: "权限不足" }, { status: 403 });
    }

    return session;
  };
}
```

### 7.1 防止暴力破解

```typescript
// lib/rate-limiter.ts
import { Redis } from "@upstash/redis";

const redis = new Redis({
  url: process.env.UPSTASH_REDIS_URL!,
  token: process.env.UPSTASH_REDIS_TOKEN!,
});

export async function checkLoginAttempts(
  identifier: string
): Promise<{ allowed: boolean; remainingAttempts: number }> {
  const key = `login:attempts:${identifier}`;
  const maxAttempts = 5;
  const windowMs = 15 * 60 * 1000; // 15 minutes

  const attempts = await redis.get(key);
  const currentAttempts = attempts ? parseInt(attempts as string) : 0;

  if (currentAttempts >= maxAttempts) {
    return { allowed: false, remainingAttempts: 0 };
  }

  return {
    allowed: true,
    remainingAttempts: maxAttempts - currentAttempts,
  };
}

export async function recordLoginAttempt(identifier: string) {
  const key = `login:attempts:${identifier}`;
  const windowMs = 15 * 60 * 1000; // 15 minutes

  const current = await redis.incr(key);

  if (current === 1) {
    await redis.expire(key, Math.floor(windowMs / 1000));
  }
}

export async function resetLoginAttempts(identifier: string) {
  const key = `login:attempts:${identifier}`;
  await redis.del(key);
}

// app/api/auth/login/route.ts
import {
  checkLoginAttempts,
  recordLoginAttempt,
  resetLoginAttempts,
} from "@/lib/rate-limiter";

export async function POST(request: Request) {
  const { username, password } = await request.json();
  const ip = request.headers.get("x-forwarded-for") || "unknown";

  // 检查登录尝试次数
  const { allowed, remainingAttempts } = await checkLoginAttempts(ip);

  if (!allowed) {
    return NextResponse.json(
      { error: "登录尝试次数过多,请15分钟后再试" },
      { status: 429 }
    );
  }

  const user = await verifyCredentials(username, password);

  if (!user) {
    await recordLoginAttempt(ip);
    return NextResponse.json(
      {
        error: "用户名或密码错误",
        remainingAttempts: remainingAttempts - 1,
      },
      { status: 401 }
    );
  }

  // 登录成功,重置尝试次数
  await resetLoginAttempts(ip);

  const sessionId = await createSession(user.id, {
    username: user.username,
    role: user.role,
    ipAddress: ip,
    userAgent: request.headers.get("user-agent") || "",
  });

  return NextResponse.json({ success: true, sessionId });
}
```

## 常见问题

### 1. Session vs JWT 如何选择?

**Session 的优势**:

- 服务器端可以随时撤销会话
- 不需要在客户端存储敏感信息
- 更容易实现单点登录

**JWT 的优势**:

- 无状态,易于水平扩展
- 减少数据库查询
- 跨域友好

**选择建议**:

- 单体应用或需要严格会话控制: 使用 Session
- 微服务架构或需要跨域: 使用 JWT
- 混合方案: 短期 JWT + 长期 Refresh Token(存储在服务器)

## 注意事项

### 1. 密码安全

1. **永远不要以明文存储密码**

   - 使用 bcrypt、scrypt 或 Argon2
   - 不要使用 MD5 或 SHA1

2. **实施强密码策略**

   - 最小长度 8-12 字符
   - 要求大小写字母、数字、特殊字符
   - 定期提醒用户更换密码

3. **防止密码泄露**
   - 不要在日志中记录密码
   - 不要在错误消息中返回密码
   - 使用 HTTPS 传输密码

### 2. Session 管理

1. **设置合理的过期时间**

   - 访问令牌: 15-30 分钟
   - 刷新令牌: 7-30 天
   - 会话: 根据业务需求

2. **实施会话固定攻击防护**

   - 登录后重新生成 Session ID
   - 使用 httpOnly 和 secure Cookie
   - 设置 SameSite 属性

3. **支持会话撤销**
   - 提供登出功能
   - 支持远程登出
   - 密码更改后使所有会话失效

### 3. JWT 使用

1. **不要在 JWT 中存储敏感信息**

   - JWT 是 Base64 编码,不是加密
   - 任何人都可以解码查看内容
   - 只存储必要的标识信息

2. **设置合理的过期时间**

   - 访问令牌应该短期(15 分钟)
   - 使用刷新令牌机制
   - 不要设置过长的过期时间

3. **验证 JWT 的所有声明**
   - 验证签名
   - 验证过期时间
   - 验证 issuer 和 audience

### 4. MFA 实施

1. **提供备用方案**

   - 生成备用码
   - 支持多种 MFA 方式
   - 提供恢复流程

2. **用户体验**

   - 不要强制所有用户启用 MFA
   - 提供清晰的设置指南
   - 支持"信任设备"功能

3. **安全存储**
   - MFA 密钥必须加密存储
   - 备用码必须哈希存储
   - 定期审计 MFA 使用情况

### 5. 权限控制

1. **最小权限原则**

   - 默认拒绝所有访问
   - 只授予必要的权限
   - 定期审查权限

2. **权限分离**

   - 区分认证和授权
   - 使用 RBAC 或 ABAC
   - 支持细粒度权限控制

3. **审计日志**
   - 记录所有权限变更
   - 记录敏感操作
   - 定期审查日志
