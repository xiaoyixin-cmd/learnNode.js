# 08-07-字体加载优化方案

字体加载是影响页面性能的重要因素,不当的字体加载会导致 FOIT(Flash of Invisible Text)或 FOUT(Flash of Unstyled Text)问题。

## 核心概念

### 字体加载策略

| 策略                   | 说明             | 优点     | 缺点       |
| ---------------------- | ---------------- | -------- | ---------- |
| font-display: swap     | 立即显示后备字体 | 无 FOIT  | 有 FOUT    |
| font-display: block    | 短暂阻塞后显示   | 无 FOUT  | 有 FOIT    |
| font-display: fallback | 极短阻塞后显示   | 平衡     | 可能不加载 |
| font-display: optional | 仅在缓存时使用   | 性能最优 | 可能不显示 |
| 预加载                 | 提前加载字体     | 快速显示 | 增加请求   |

### Next.js 字体优化特性

1. 自动字体优化
2. 零布局偏移
3. 自动自托管
4. CSS size-adjust
5. 字体子集化
6. 预加载关键字体

```typescript
// lib/fontPerformance.ts
export function monitorFontPerformance() {
  if (typeof window === "undefined") return;

  const observer = new PerformanceObserver((list) => {
    for (const entry of list.getEntries()) {
      if (entry.entryType === "resource" && entry.name.includes("font")) {
        console.log("Font loaded:", {
          name: entry.name,
          duration: entry.duration,
          size: entry.transferSize,
        });
      }
    }
  });

  observer.observe({ entryTypes: ["resource"] });
}
```

```typescript
export async function loadFont(fontName: string, fontUrl: string) {
  const font = new FontFace(fontName, `url(${fontUrl})`);

  try {
    const loadedFont = await font.load();
    document.fonts.add(loadedFont);
    return true;
```

```typescript
// components/SafeFont.tsx
"use client";

import { useEffect, useState } from "react";

export function SafeFont({ children }: { children: React.ReactNode }) {
  const [fontError, setFontError] = useState(false);

  useEffect(() => {
    const timeout = setTimeout(() => {
      if (!document.fonts.check("12px Inter")) {
        setFontError(true);
      }
    }, 3000);

    return () => clearTimeout(timeout);
  }, []);

  if (fontError) {
    return <div style={{ fontFamily: "system-ui" }}>{children}</div>;
  }

  return <>{children}</>;
}
```

## 注意事项

### 1. 字体格式选择

优先使用 woff2 格式,文件更小,加载更快。

### 2. 字体子集化

只加载需要的字符集,减少文件大小。

### 3. 字体预加载

对关键字体使用 preload,提升首屏性能。

### 4. 字体 fallback

设置合适的 fallback 字体,避免 FOIT。

### 5. 字体缓存

设置长期缓存,减少重复加载。

## 常见问题

### 1. 如何优化字体加载速度?

使用 next/font,启用 preload,使用 woff2 格式,字体子集化。

FOIT 是文字不可见闪烁,FOUT 是无样式文字闪烁。

使用 font-display: swap 或 fallback。

使用 next/font/local 导入本地字体文件。

### 5. 如何优化中文字体?

使用字体子集化,只加载常用字符。

```typescript
// lib/monitorFonts.ts
export function monitorFonts() {
  if (typeof window === "undefined") return;

  const observer = new PerformanceObserver((list) => {
    for (const entry of list.getEntries()) {
      if (entry.entryType === "resource" && entry.name.includes("font")) {
        console.log("Font performance:", {
          name: entry.name,
          duration: entry.duration,
          size: entry.transferSize,
          cached: entry.transferSize === 0,
        });
      }
    }
  });

  observer.observe({ entryTypes: ["resource"] });

  document.fonts.ready.then(() => {
    console.log("All fonts loaded");
  });
}
```
