# 08-06-图片优化最佳实践

图片通常是网页中最大的资源,优化图片可以显著提升页面加载速度和用户体验。Next.js 提供了强大的 Image 组件和优化功能。

## 核心概念

### 图片优化技术

| 技术     | 说明           | 性能提升 |
| -------- | -------------- | -------- |
| 格式优化 | WebP/AVIF      | 高       |
| 尺寸优化 | 响应式图片     | 高       |
| 懒加载   | loading="lazy" | 高       |
| 优先加载 | priority       | 中       |
| 占位符   | placeholder    | 中       |
| CDN      | 边缘缓存       | 高       |

### Next.js Image 组件特性

1. **自动格式优化**: 自动转换为 WebP/AVIF
2. **自动尺寸优化**: 根据设备提供合适尺寸
3. **懒加载**: 默认启用懒加载
4. **占位符**: 支持 blur 和 empty 占位符
5. **优先加载**: 支持 priority 属性
6. **响应式**: 支持 sizes 属性

// lib/addWatermark.ts
export async function addWatermark(imageSrc: string): Promise<string> {
  const canvas = document.createElement("canvas");
  const ctx = canvas.getContext("2d");
  const img = new Image();

  return new Promise((resolve) => {
    img.onload = () => {
      canvas.width = img.width;
      canvas.height = img.height;

      ctx?.drawImage(img, 0, 0);
      ctx!.font = "30px Arial";
      ctx!.fillStyle = "rgba(255, 255, 255, 0.5)";
      ctx!.fillText("Watermark", 10, 50);

      resolve(canvas.toDataURL());
    };

    img.src = imageSrc;
  });
}
```

### 实战场景十六: 图片压缩

// lib/compressImage.ts
export async function compressImage(file: File, quality = 0.8): Promise<Blob> {
  const canvas = document.createElement("canvas");
  const ctx = canvas.getContext("2d");
  const img = new Image();

  return new Promise((resolve) => {
    img.onload = () => {
      canvas.width = img.width;
      canvas.height = img.height;
      ctx?.drawImage(img, 0, 0);

      canvas.toBlob((blob) => resolve(blob!), "image/jpeg", quality);
    };

    img.src = URL.createObjectURL(file);
  });
}
```

## 注意事项

### 1. 图片格式选择

根据浏览器支持选择合适的图片格式,优先使用 AVIF 和 WebP。

### 2. 图片尺寸优化

根据实际显示尺寸提供合适的图片,避免加载过大的图片。

### 3. 懒加载策略

对首屏以外的图片使用懒加载,首屏关键图片使用 priority。

### 4. 占位符使用

使用 blur 占位符提升用户体验,避免布局偏移。

### 5. CDN 配置

使用 CDN 加速图片加载,减少服务器压力。

### 自动化优化

使用 Sharp 自动优化图片:

```javascript
// scripts/optimizeImages.js
const sharp = require("sharp");
const fs = require("fs");
const path = require("path");

async function optimizeImage(inputPath, outputPath) {
  await sharp(inputPath).webp({ quality: 85 }).toFile(outputPath);
}

// 批量优化
const imagesDir = path.join(__dirname, "../public/images");
const files = fs.readdirSync(imagesDir);

files.forEach(async (file) => {
  if (file.match(/\.(jpg|jpeg|png)$/)) {
    const inputPath = path.join(imagesDir, file);
    const outputPath = path.join(
      imagesDir,
      file.replace(/\.(jpg|jpeg|png)$/, ".webp")
    );
    await optimizeImage(inputPath, outputPath);
  }
});
```

### CI/CD 集成

在 CI/CD 流程中自动优化图片:

```yaml
# .github/workflows/optimize-images.yml
name: Optimize Images

on:
  push:
    paths:
      - "public/images/**"

jobs:
  optimize:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Optimize images
        run: |
          npm install sharp
          node scripts/optimizeImages.js
      - name: Commit changes
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "GitHub Action"
          git add public/images
          git commit -m "Optimize images" || exit 0
          git push
```
