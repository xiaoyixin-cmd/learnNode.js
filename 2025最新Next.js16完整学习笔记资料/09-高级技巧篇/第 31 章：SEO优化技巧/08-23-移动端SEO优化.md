# 移动端 SEO 优化

移动端 SEO 是现代网站优化的重点,Google 已采用移动优先索引。本文介绍如何优化 Next.js 应用的移动端 SEO 表现。

## 核心概念

### 移动优先索引

Google 主要使用移动版本的内容进行索引和排名。

### 响应式设计

网站能够适应不同屏幕尺寸的设计方式。

### 移动友好性

网站在移动设备上的可用性和用户体验。

### 触摸优化

针对触摸操作优化的交互设计。

```typescript
"use client";

import Image from "next/image";
import { useState } from "react";

interface Props {
  src: string;
  alt: string;
  width: number;
  height: number;
  priority?: boolean;
}

export function OptimizedImage({
  src,
  alt,
  width,
  height,
  priority = false,
}: Props) {
  const [isLoading, setIsLoading] = useState(true);

  return (
    <div className="relative">
      <Image
        src={src}
        alt={alt}
        width={width}
        height={height}
        priority={priority}
        quality={75} // 移动端使用较低质量
        sizes="(max-width: 640px) 100vw, (max-width: 1024px) 50vw, 33vw"
        onLoadingComplete={() => setIsLoading(false)}
        className={`
          transition-opacity duration-300
          ${isLoading ? "opacity-0" : "opacity-100"}
        `}
      />
      {isLoading && (
        <div className="absolute inset-0 bg-gray-200 animate-pulse" />
      )}
    </div>
  );
}
```

```typescript
test("移动端 SEO 检查", async ({ page }) => {
  await page.goto("/");

  // 检查 viewport 设置
  const viewport = await page.getAttribute('meta[name="viewport"]', "content");

  // 检查触摸目标大小

  for (const button of buttons) {

    if (box) {
```

```typescript
import { headers } from "next/headers";

export default function Page() {
  const headersList = headers();
  const userAgent = headersList.get("user-agent") || "";
  const isMobile = /mobile/i.test(userAgent);

  return <div>{isMobile ? <MobileContent /> : <DesktopContent />}</div>;
}

// 方案 3: 使用客户端检测
("use client");

import { useEffect, useState } from "react";

export default function Page() {
  const [isMobile, setIsMobile] = useState(false);

  useEffect(() => {
    const checkMobile = () => {
      setIsMobile(window.innerWidth < 768);
    };

    checkMobile();
    window.addEventListener("resize", checkMobile);

    return () => window.removeEventListener("resize", checkMobile);
  }, []);

  return <div>{isMobile ? <MobileContent /> : <DesktopContent />}</div>;
}
```

### 4. 移动端和桌面端应该使用不同的 URL 吗?

**对比**:

| 方案                          | 优点               | 缺点                 | 推荐度     |
| ----------------------------- | ------------------ | -------------------- | ---------- |
| 响应式设计 (同一 URL)         | SEO 友好, 维护简单 | 可能加载不必要的资源 | ⭐⭐⭐⭐⭐ |
| 独立移动站 (m.example.com)    | 可完全定制         | 维护成本高, SEO 复杂 | ⭐⭐       |
| 动态服务 (同一 URL, 不同内容) | 灵活               | 实现复杂             | ⭐⭐⭐     |

**推荐方案**: 响应式设计

## 注意事项

### 1. Viewport 配置

```typescript
// app/layout.tsx
export const metadata = {
  viewport: {
    width: "device-width",
    initialScale: 1,
    maximumScale: 5, // 不要禁用缩放
    userScalable: true, // 允许用户缩放
  },
};
```

### 2. 触摸目标大小

- 最小 44x44 像素
- 目标之间至少 8 像素间距
- 使用 padding 增加点击区域

### 3. 字体大小

- 正文至少 16px
- 避免使用小于 12px 的字体
- 使用相对单位(rem, em)

### 4. 避免的做法

- ❌ 禁用缩放
- ❌ 使用 Flash
- ❌ 弹窗广告
- ❌ 横向滚动
- ❌ 固定宽度布局

## 总结

移动端 SEO 优化是现代网站的必备要求。本文介绍了:

1. **响应式设计**: 使用 Tailwind CSS 实现移动优先布局
2. **性能优化**: Core Web Vitals、图片、字体、JavaScript 优化
3. **用户体验**: 触摸优化、移动端导航、表单优化
4. **SEO 检测**: Google Mobile-Friendly Test、Lighthouse CI、自动化测试
5. **最佳实践**: Viewport 配置、触摸目标、字体大小

关键要点:

- 采用移动优先设计
- 优化 Core Web Vitals
- 确保触摸目标足够大
- 使用响应式图片
- 优化字体加载
- 避免禁用缩放

### 2. 移动端手势优化

```tsx
// components/SwipeableCard.tsx
"use client";

import { useState } from "react";

export function SwipeableCard({ children }: { children: React.ReactNode }) {
  const [touchStart, setTouchStart] = useState(0);
  const [touchEnd, setTouchEnd] = useState(0);

  const handleTouchStart = (e: React.TouchEvent) => {
    setTouchStart(e.targetTouches[0].clientX);
  };

  const handleTouchMove = (e: React.TouchEvent) => {
    setTouchEnd(e.targetTouches[0].clientX);
  };

  const handleTouchEnd = () => {
    if (touchStart - touchEnd > 150) {
      // 向左滑动
      console.log("Swiped left");
    }

    if (touchStart - touchEnd < -150) {
      // 向右滑动
      console.log("Swiped right");
    }
  };

  return (
    <div
      onTouchStart={handleTouchStart}
      onTouchMove={handleTouchMove}
      onTouchEnd={handleTouchEnd}
    >
      {children}
    </div>
  );
}
```

### 3. 移动端性能监控

```typescript
// lib/mobile-performance.ts
export function monitorMobilePerformance() {
  if (typeof window === "undefined") return;

  // 检测设备类型
  const isMobile = /iPhone|iPad|iPod|Android/i.test(navigator.userAgent);

  if (isMobile) {
    // 监控网络状态
    if ("connection" in navigator) {
      const connection = (navigator as any).connection;
      console.log("Network type:", connection.effectiveType);
      console.log("Downlink:", connection.downlink);
      console.log("RTT:", connection.rtt);
    }

    // 监控电池状态
    if ("getBattery" in navigator) {
      (navigator as any).getBattery().then((battery: any) => {
        console.log("Battery level:", battery.level * 100 + "%");
        console.log("Charging:", battery.charging);
      });
    }

    // 监控内存使用
    if ("memory" in performance) {
      const memory = (performance as any).memory;
      console.log("Used JS Heap:", memory.usedJSHeapSize / 1048576 + " MB");
      console.log("Total JS Heap:", memory.totalJSHeapSize / 1048576 + " MB");
    }
  }
}
```

```typescript
// lib/mobile-debug.ts
export function enableMobileDebug() {
  if (typeof window === "undefined") return;

  // 显示触摸点
  document.addEventListener("touchstart", (e) => {
    Array.from(e.touches).forEach((touch) => {
      const dot = document.createElement("div");
      dot.style.cssText = `
        position: fixed;
        width: 20px;
        height: 20px;
        background: red;
        border-radius: 50%;
        pointer-events: none;
        z-index: 9999;
        left: ${touch.clientX - 10}px;
        top: ${touch.clientY - 10}px;
      `;
      document.body.appendChild(dot);
      setTimeout(() => dot.remove(), 1000);
    });
  });

  // 显示设备信息
  console.log({
    userAgent: navigator.userAgent,
    screenWidth: window.screen.width,
    screenHeight: window.screen.height,
    devicePixelRatio: window.devicePixelRatio,
    orientation: window.screen.orientation?.type,
  });
}
```
