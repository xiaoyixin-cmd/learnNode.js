# 页面速度 SEO 影响

页面速度是 Google 排名算法的重要因素之一。快速的页面加载不仅提升用户体验,还能显著改善 SEO 表现。本文介绍如何优化页面速度以提升 SEO 效果。

## 核心概念

### Core Web Vitals

Google 提出的核心性能指标,包括 LCP、FID、CLS。

### 页面加载时间

从用户请求到页面完全加载的时间。

### 首屏时间

用户看到首屏内容所需的时间。

### 交互时间

页面变得可交互所需的时间。

### 2.2 FID (First Input Delay) 优化

FID 衡量用户首次交互到浏览器响应的时间,目标是 100 毫秒以内。

```typescript
// app/page.tsx
"use client";

import { useState, useTransition } from "react";

export default function InteractivePage() {
  const [isPending, startTransition] = useTransition();
  const [data, setData] = useState([]);

  const handleClick = () => {
    // 使用 transition 降低优先级
    startTransition(() => {
      // 执行非紧急更新
      setData(generateLargeData());
    });
  };

  return (
    <div>
      <button onClick={handleClick} disabled={isPending}>
        {isPending ? "加载中..." : "加载数据"}
      </button>

      {data.map((item, index) => (
        <div key={index}>{item}</div>
      ))}
    </div>
  );
}

function generateLargeData() {
  return Array.from({ length: 1000 }, (_, i) => `Item ${i}`);
}
```

**FID 优化清单**:

| 优化项        | 方法           | 效果           |
| ------------- | -------------- | -------------- |
| 代码分割      | dynamic import | 减少主线程阻塞 |
| 延迟加载      | 非关键 JS 延后 | 提升交互响应   |
| Web Workers   | 后台处理       | 释放主线程     |
| useTransition | React 18 特性  | 优化更新优先级 |

### 2.3 CLS (Cumulative Layout Shift) 优化

CLS 衡量页面视觉稳定性,目标是 0.1 以下。

```typescript
// app/page.tsx
import Image from "next/image";

export default function StablePage() {

    <div>
      {/* 1. 为图片设置明确的尺寸 */}

        src="/banner.jpg"
        alt="Banner"
        width={1200}

          width: "100%",
          height: "auto",
```

| 预留空间 | min-height         | 避免内容跳动 |

```typescript
// lib/performance.ts
export function reportWebVitals(metric: any) {
  const { id, name, label, value } = metric;

  // 发送到分析服务
  if (typeof window !== "undefined" && window.gtag) {
    window.gtag("event", name, {
      event_category:
        label === "web-vital" ? "Web Vitals" : "Next.js custom metric",
      value: Math.round(name === "CLS" ? value * 1000 : value),
      event_label: id,
      non_interaction: true,
    });
  }

  // 发送到自定义API
  fetch("/api/analytics", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      metric: name,
      value,
      id,
      timestamp: Date.now(),
    }),
  });
}

import { useReportWebVitals } from "next/web-vitals";
```

## 注意事项

### 1. 优先级排序

- 首屏内容优先加载
- 关键资源使用 priority
