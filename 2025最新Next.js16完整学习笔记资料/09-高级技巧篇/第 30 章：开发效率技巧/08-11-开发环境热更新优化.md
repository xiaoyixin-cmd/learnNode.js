# 08-11-开发环境热更新优化

开发环境的热更新速度直接影响开发效率。本文介绍 Next.js 中优化热更新的实战技巧。

## 核心概念

### 热更新机制

| 机制         | 原理           | 速度 | 适用场景   |
| ------------ | -------------- | ---- | ---------- |
| HMR          | 模块热替换     | 快   | 开发环境   |
| Fast Refresh | React 组件刷新 | 快   | React 组件 |
| Full Reload  | 完整重载       | 慢   | 配置变更   |
| Incremental  | 增量编译       | 中   | 大型项目   |

## 实战场景

### 实战场景一: 优化 Webpack 配置

```javascript
// next.config.js
module.exports = {
  webpack: (config, { dev, isServer }) => {
    if (dev && !isServer) {
      // 优化HMR性能
      config.watchOptions = {
        poll: 1000,
        aggregateTimeout: 300,
        ignored: /node_modules/,
      };

      // 启用缓存
      config.cache = {
        type: "filesystem",
        buildDependencies: {
          config: [__filename],
        },
      };

      // 优化解析
      config.resolve.symlinks = false;
      config.resolve.cacheWithContext = false;
    }

    return config;
  },
};
```

## 常见问题

### 1. 热更新不生效怎么办?

检查文件监听配置和防火墙设置。

### 2. 热更新速度慢如何优化?

启用缓存、优化文件监听、使用 Turbopack。

### 3. 如何减少内存占用?

调整缓存策略、减少并行任务数。

### 4. TypeScript 编译慢怎么办?

启用增量编译、跳过类型检查。

### 5. CSS 更新慢如何解决?

启用 CSS 优化、使用 CSS Modules。

### 6. 如何优化大型项目?

使用持久化缓存、并行编译、代码分割。

### 7. Source Map 生成慢怎么办?

使用更快的 Source Map 类型。

### 8. 如何监控热更新性能?

使用 Webpack Bundle Analyzer 等工具。

### 9. 开发服务器启动慢如何优化?

减少初始编译范围、使用缓存。

### 10. 如何处理第三方库更新?

清除缓存、重启开发服务器。

### 11. 文件监听占用 CPU 高怎么办?

增加轮询间隔、减少监听文件数。

### 12. 如何优化 Docker 环境热更新?

使用卷挂载、配置文件监听。

### 13. WSL 环境热更新慢如何解决?

使用原生 Windows 路径、优化文件监听。

### 14. 如何处理符号链接?

禁用符号链接解析以提升性能。

### 15. 多页面应用如何优化?

使用按需编译、减少缓冲页面数。

### 16. 如何优化 Monorepo 项目?

配置工作区、优化依赖解析。

### 17. 环境变量更新不生效?

重启开发服务器或配置热更新。

### 18. 如何优化样式组件?

启用编译器优化、使用 SWC。

### 19. 图片更新不生效怎么办?

清除.next 缓存、重启服务器。

### 20. 如何调试热更新问题?

启用详细日志、检查 Webpack 配置。
