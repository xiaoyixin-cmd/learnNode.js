# 拦截路由实现图片预览

拦截路由是 Next.js 16 中的一个强大功能,允许在客户端导航时拦截路由并显示不同的内容,同时保持 URL 的变化。这个功能特别适合实现图片预览、模态框等场景。本文将详细介绍如何使用拦截路由实现各种图片预览功能。

## 拦截路由基础

### 拦截路由语法

拦截路由使用特殊的文件夹命名约定:

| 约定       | 说明           | 示例                               |
| ---------- | -------------- | ---------------------------------- |
| `(.)`      | 匹配同级路由   | `(.)photo` 拦截 `/photo`           |
| `(..)`     | 匹配上一级路由 | `(..)photo` 拦截 `../photo`        |
| `(..)(..)` | 匹配上两级路由 | `(..)(..)photo` 拦截 `../../photo` |
| `(...)`    | 匹配根路由     | `(...)photo` 拦截 `/photo`         |

### 工作原理

```
客户端导航: /gallery → /gallery/photo/1
→ 拦截路由生效,显示模态框

直接访问: /gallery/photo/1
→ 拦截路由不生效,显示完整页面

刷新页面: /gallery/photo/1
→ 拦截路由不生效,显示完整页面
```

### 增强的模态框组件

```typescript
// components/photo-modal.tsx
"use client";

import { useRouter } from "next/navigation";
import { useEffect, useRef } from "react";

interface PhotoModalProps {
  children: React.ReactNode;
  onPrev?: () => void;
  onNext?: () => void;
  hasPrev?: boolean;
  hasNext?: boolean;
}

export function PhotoModal({
  children,
  onPrev,
  onNext,
  hasPrev,
  hasNext,
}: PhotoModalProps) {
  const router = useRouter();
  const dialogRef = useRef<HTMLDialogElement>(null);

  useEffect(() => {
    dialogRef.current?.showModal();

    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === "Escape") {
        closeModal();
      } else if (e.key === "ArrowLeft" && hasPrev && onPrev) {
        onPrev();
      } else if (e.key === "ArrowRight" && hasNext && onNext) {
        onNext();
      }
    };

    window.addEventListener("keydown", handleKeyDown);
    return () => window.removeEventListener("keydown", handleKeyDown);
  }, [hasPrev, hasNext, onPrev, onNext]);

  const closeModal = () => {
    dialogRef.current?.close();
    router.back();
  };

  return (
    <dialog
      ref={dialogRef}
      className="fixed inset-0 bg-transparent p-0 max-w-none max-h-none"
      onClose={closeModal}
    >
      <div
        className="fixed inset-0 bg-black/90 flex items-center justify-center"
        onClick={closeModal}
      >
        {/* 上一张按钮 */}
        {hasPrev && (
          <button
            onClick={(e) => {
              e.stopPropagation();
              onPrev?.();
            }}
            className="absolute left-4 top-1/2 -translate-y-1/2 bg-white/10 hover:bg-white/20 text-white p-3 rounded-full transition"
          >
            <svg
              className="w-6 h-6"
              fill="none"
              stroke="currentColor"
              viewBox="0 0 24 24"
            >
              <path
                strokeLinecap="round"
                strokeLinejoin="round"
                strokeWidth={2}
                d="M15 19l-7-7 7-7"
              />
            </svg>
          </button>
        )}

        {/* 内容区域 */}
        <div
          className="relative max-w-7xl max-h-[90vh] mx-auto"
          onClick={(e) => e.stopPropagation()}
        >
          <button
            onClick={closeModal}
            className="absolute -top-12 right-0 text-white hover:text-gray-300"
          >
            <svg
              className="w-8 h-8"
              fill="none"
              stroke="currentColor"
              viewBox="0 0 24 24"
            >
              <path
                strokeLinecap="round"
                strokeLinejoin="round"
                strokeWidth={2}
                d="M6 18L18 6M6 6l12 12"
              />
            </svg>
          </button>
          {children}
        </div>

        {/* 下一张按钮 */}
        {hasNext && (
          <button
            onClick={(e) => {
              e.stopPropagation();
              onNext?.();
            }}
            className="absolute right-4 top-1/2 -translate-y-1/2 bg-white/10 hover:bg-white/20 text-white p-3 rounded-full transition"
          >
            <svg
              className="w-6 h-6"
              fill="none"
              stroke="currentColor"
              viewBox="0 0 24 24"
            >
              <path
                strokeLinecap="round"
                strokeLinejoin="round"
                strokeWidth={2}
                d="M9 5l7 7-7 7"
              />
            </svg>
          </button>
        )}
      </div>
    </dialog>
  );
}
```

### 使用导航模态框

```typescript
// app/@modal/(..)gallery/photo/[id]/page.tsx
"use client";

import { notFound, useRouter } from "next/navigation";
import { getPhoto, getPhotos } from "@/lib/photos";
import { PhotoModal } from "@/components/photo-modal";
import { use } from "react";

export default function PhotoModalWithNav({
  params,
}: {
  params: { id: string };
}) {
  const router = useRouter();
  const photo = use(getPhoto(params.id));
  const photos = use(getPhotos());

  if (!photo) {
    notFound();
  }

  const currentIndex = photos.findIndex((p) => p.id === params.id);
  const hasPrev = currentIndex > 0;
  const hasNext = currentIndex < photos.length - 1;

  return (
    <PhotoModal
      hasPrev={hasPrev}
      hasNext={hasNext}
      onPrev={() => {
        if (hasPrev) {
          router.push(`/gallery/photo/${photos[currentIndex - 1].id}`);
        }
      }}
      onNext={() => {
        if (hasNext) {
          router.push(`/gallery/photo/${photos[currentIndex + 1].id}`);
        }
      }}
    >
      <div className="relative">
        <img
          src={photo.url}
          alt={photo.title}
          className="max-w-full max-h-[80vh] object-contain"
        />
        <div className="mt-4 text-white">
          <h2 className="text-2xl font-bold">{photo.title}</h2>
          <p className="text-gray-300 mt-2">{photo.description}</p>
          <div className="mt-2 text-sm text-gray-400">
            {currentIndex + 1} / {photos.length}
          </div>
        </div>
      </div>
    </PhotoModal>
  );
}
```

## 实战场景三: 图片缩放预览

### 可缩放的图片组件

```typescript
// components/zoomable-image.tsx
"use client";

import { useState, useRef, useEffect } from "react";

interface ZoomableImageProps {
  src: string;
  alt: string;
}

export function ZoomableImage({ src, alt }: ZoomableImageProps) {
  const [scale, setScale] = useState(1);
  const [position, setPosition] = useState({ x: 0, y: 0 });
  const [isDragging, setIsDragging] = useState(false);
  const [dragStart, setDragStart] = useState({ x: 0, y: 0 });
  const imageRef = useRef<HTMLDivElement>(null);

  const handleWheel = (e: React.WheelEvent) => {
    e.preventDefault();
    const delta = e.deltaY > 0 ? 0.9 : 1.1;
    const newScale = Math.min(Math.max(scale * delta, 0.5), 5);
    setScale(newScale);
  };

  const handleMouseDown = (e: React.MouseEvent) => {
    if (scale > 1) {
      setIsDragging(true);
      setDragStart({
        x: e.clientX - position.x,
        y: e.clientY - position.y,
      });
    }
  };

  const handleMouseMove = (e: React.MouseEvent) => {
    if (isDragging) {
      setPosition({
        x: e.clientX - dragStart.x,
        y: e.clientY - dragStart.y,
      });
    }
  };

  const handleMouseUp = () => {
    setIsDragging(false);
  };

  const resetZoom = () => {
    setScale(1);
    setPosition({ x: 0, y: 0 });
  };

  return (
    <div className="relative">
      <div
        ref={imageRef}
        className="overflow-hidden cursor-move"
        onWheel={handleWheel}
        onMouseDown={handleMouseDown}
        onMouseMove={handleMouseMove}
        onMouseUp={handleMouseUp}
        onMouseLeave={handleMouseUp}
      >
        <img
          src={src}
          alt={alt}
          className="max-w-full max-h-[80vh] object-contain transition-transform"
          style={{
            transform: `scale(${scale}) translate(${position.x / scale}px, ${
              position.y / scale
            }px)`,
            cursor: isDragging ? "grabbing" : scale > 1 ? "grab" : "default",
          }}
        />
      </div>

      {/* 缩放控制 */}
      <div className="absolute bottom-4 right-4 flex gap-2">
        <button
          onClick={() => setScale((s) => Math.max(s - 0.2, 0.5))}
          className="bg-white/10 hover:bg-white/20 text-white p-2 rounded"
        >
          -
        </button>
        <button
          onClick={resetZoom}
          className="bg-white/10 hover:bg-white/20 text-white px-3 py-2 rounded text-sm"
        >
          {Math.round(scale * 100)}%
        </button>
        <button
          onClick={() => setScale((s) => Math.min(s + 0.2, 5))}
          className="bg-white/10 hover:bg-white/20 text-white p-2 rounded"
        >
          +
        </button>
      </div>
    </div>
  );
}
```

## 适用场景

### 何时使用拦截路由

| 场景         | 说明                 | 拦截路由约定 |
| ------------ | -------------------- | ------------ |
| 同级路由拦截 | 拦截同一目录下的路由 | `(.)`        |
| 上级路由拦截 | 拦截父目录的路由     | `(..)`       |
| 根路由拦截   | 从任何位置拦截根路由 | `(...)`      |
| 深层路由拦截 | 拦截多层父目录的路由 | `(..)(..)`   |

### 拦截路由 vs 客户端状态

| 特性       | 拦截路由     | 客户端状态 |
| ---------- | ------------ | ---------- |
| URL 变化   | 有           | 无         |
| 浏览器历史 | 支持         | 不支持     |
| 直接访问   | 显示完整页面 | 无法访问   |
| SEO        | 友好         | 不友好     |
| 分享链接   | 可以         | 不可以     |
| 刷新保持   | 显示完整页面 | 丢失状态   |

## 注意事项

### 1. 拦截路由只在客户端导航时生效

```typescript

"use client";

import { useRouter } from "next/navigation";
import { useEffect, useRef, useState } from "react";

export function Modal({ children }: { children: React.ReactNode }) {
  const router = useRouter();
  const dialogRef = useRef<HTMLDialogElement>(null);
  const [isClosing, setIsClosing] = useState(false);

  useEffect(() => {
    dialogRef.current?.showModal();
  }, []);

  const closeModal = () => {
    setIsClosing(true);
    setTimeout(() => {
      dialogRef.current?.close();
      router.back();
    }, 300); // 等待动画完成
  };

  return (
    <dialog
      ref={dialogRef}
      className={`transition-all duration-300 ${
        isClosing ? "opacity-0 scale-95" : "opacity-100 scale-100"
      }`}
      onClose={closeModal}
    >
      {children}
    </dialog>
  );
}
```

### 7. 如何实现图片懒加载?

```typescript
// components/lazy-image.tsx
"use client";

import { useState, useEffect, useRef } from "react";

interface LazyImageProps {
  src: string;
  alt: string;
  className?: string;
}

export function LazyImage({ src, alt, className }: LazyImageProps) {
  const [isLoaded, setIsLoaded] = useState(false);
  const [isInView, setIsInView] = useState(false);
  const imgRef = useRef<HTMLImageElement>(null);

  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setIsInView(true);
          observer.disconnect();
        }
      },
      { threshold: 0.1 }
    );

    if (imgRef.current) {
      observer.observe(imgRef.current);
    }

    return () => observer.disconnect();
  }, []);

  return (
    <div className="relative">
      {!isLoaded && (
        <div className="absolute inset-0 bg-gray-200 animate-pulse" />
      )}
      <img
        ref={imgRef}
        src={isInView ? src : ""}
        alt={alt}
        className={className}
        onLoad={() => setIsLoaded(true)}
      />
    </div>
  );
}
```

```typescript
// components/share-button.tsx
"use client";

export function ShareButton({ url, title }: { url: string; title: string }) {
  const handleShare = async () => {
    if (navigator.share) {
      try {
        await navigator.share({
          title,
          url,
        });
      } catch (error) {
        console.error("Share failed:", error);
      }

    }
  };

  return (
    <button
      onClick={handleShare}
      className="bg-blue-600 text-white px-4 py-2 rounded"
    >
      Share
    </button>
  );
}
```

## 总结

拦截路由是 Next.js 16 中实现图片预览等模态框功能的强大工具。通过本文的学习,我们掌握了:

### 核心概念

1. **拦截路由语法**: `(.)` `(..)` `(..)(..)` `(...)`
2. **工作原理**: 只在客户端导航时生效
3. **必须提供**: 拦截路由和完整页面
4. **默认插槽**: 必须提供 `default.tsx`

### 实战应用

我们通过四个实战场景学习了拦截路由的应用:

1. **基础图片预览**: 简单的模态框实现
2. **带导航的图片预览**: 支持键盘和按钮导航
3. **图片缩放预览**: 支持缩放和拖拽
4. **图片网格预览**: 相册和图片的多层路由

### 最佳实践

1. **始终提供完整页面**: 确保直接访问和刷新时有良好体验
2. **使用 Link 组件**: 确保拦截路由生效
3. **提供 default.tsx**: 避免刷新页面时出错
4. **优化图片加载**: 使用懒加载和预加载
5. **添加键盘导航**: 提升用户体验

### 注意事项

1. 拦截路由只在客户端导航时生效
2. 必须同时提供拦截路由和完整页面
3. 拦截路由路径是相对于定义位置的
4. 必须提供默认插槽
5. 使用 router.back() 关闭模态框

通过合理使用拦截路由,可以创建流畅的图片预览体验,同时保持良好的 SEO 和可访问性。
