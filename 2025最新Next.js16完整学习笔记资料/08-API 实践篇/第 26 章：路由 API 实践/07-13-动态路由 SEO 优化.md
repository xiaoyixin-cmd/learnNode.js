# 动态路由 SEO 优化

动态路由的 SEO 优化是 Next.js 应用中的重要课题。本文将详细介绍如何优化动态路由的 SEO,包括元数据生成、结构化数据、sitemap 生成、robots.txt 配置等内容。

## 元数据优化

### 动态元数据生成

```typescript
// app/blog/[slug]/page.tsx
import { Metadata } from "next";
import { getPost } from "@/lib/posts";

export async function generateMetadata({
  params,
}: {
  params: { slug: string };
}): Promise<Metadata> {
  const post = await getPost(params.slug);

  if (!post) {
    return {
      title: "Post Not Found",
    };
  }

  return {
    title: post.title,
    description: post.excerpt,
    keywords: post.tags.join(", "),
    authors: [{ name: post.author }],
    openGraph: {
      title: post.title,
      description: post.excerpt,
      type: "article",
      publishedTime: post.publishedAt,
      authors: [post.author],
      images: [
        {
          url: post.coverImage,
          width: 1200,
          height: 630,
          alt: post.title,
        },
      ],
    },
    twitter: {
      card: "summary_large_image",
      title: post.title,
      description: post.excerpt,
      images: [post.coverImage],
    },
  };
}

export default async function BlogPost({
  params,
}: {
  params: { slug: string };
}) {
  const post = await getPost(params.slug);

  return (
    <article>
      <h1>{post.title}</h1>
      <div dangerouslySetInnerHTML={{ __html: post.content }} />
    </article>
  );
}
```

## Sitemap 生成

### 动态 Sitemap

```typescript
// app/sitemap.ts
import { MetadataRoute } from "next";
import { getPosts } from "@/lib/posts";
import { getProducts } from "@/lib/products";

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const baseUrl = "https://example.com";

  // 静态页面
  const staticPages = [
    {
      url: baseUrl,
      lastModified: new Date(),
      changeFrequency: "daily" as const,
      priority: 1,
    },
    {
      url: `${baseUrl}/about`,
      lastModified: new Date(),
      changeFrequency: "monthly" as const,
      priority: 0.8,
    },
  ];

  // 博客文章
  const posts = await getPosts();
  const postPages = posts.map((post) => ({
    url: `${baseUrl}/blog/${post.slug}`,
    lastModified: new Date(post.updatedAt),
    changeFrequency: "weekly" as const,
    priority: 0.7,
  }));

  // 产品页面
  const products = await getProducts();
  const productPages = products.map((product) => ({
    url: `${baseUrl}/products/${product.id}`,
    lastModified: new Date(product.updatedAt),
    changeFrequency: "daily" as const,
    priority: 0.9,
  }));

  return [...staticPages, ...postPages, ...productPages];
}
```

### 多语言 Sitemap

```typescript
// app/sitemap.ts
import { MetadataRoute } from "next";
import { getPosts } from "@/lib/posts";

const languages = ["en", "zh", "ja"];

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const baseUrl = "https://example.com";
  const posts = await getPosts();

  const sitemapEntries: MetadataRoute.Sitemap = [];

  for (const lang of languages) {
    // 首页
    sitemapEntries.push({
      url: `${baseUrl}/${lang}`,
      lastModified: new Date(),
      alternates: {
        languages: Object.fromEntries(
          languages.map((l) => [l, `${baseUrl}/${l}`])
        ),
      },
    });

    // 博客文章
    for (const post of posts) {
      sitemapEntries.push({
        url: `${baseUrl}/${lang}/blog/${post.slug}`,
        lastModified: new Date(post.updatedAt),
        alternates: {
          languages: Object.fromEntries(
            languages.map((l) => [l, `${baseUrl}/${l}/blog/${post.slug}`])
          ),
        },
      });
    }
  }

  return sitemapEntries;
}
```

```typescript
// app/robots.ts
import { MetadataRoute } from "next";

export default function robots(): MetadataRoute.Robots {
  const baseUrl = process.env.NEXT_PUBLIC_URL || "https://example.com";
  const isProduction = process.env.NODE_ENV === "production";

  if (!isProduction) {
    // 开发/测试环境:禁止所有爬虫
    return {
      rules: {
        userAgent: "*",
        disallow: "/",
      },
    };
  }

  // 生产环境:允许爬虫
  return {
    rules: [
      {
        userAgent: "*",
        allow: "/",
        disallow: ["/admin/", "/api/", "/private/"],
      },
      {
        userAgent: "Googlebot",
        allow: "/",
        disallow: ["/admin/"],
      },
    ],
    sitemap: `${baseUrl}/sitemap.xml`,
  };
}
```

```typescript
// middleware.ts
import { NextResponse } from "next/server";
import type { NextRequest } from "next/server";

export function middleware(request: NextRequest) {
  const url = request.nextUrl.clone();

  // 移除尾部斜杠(除了根路径)
  if (url.pathname !== "/" && url.pathname.endsWith("/")) {
    url.pathname = url.pathname.slice(0, -1);
    return NextResponse.redirect(url, 301);
  }

  return NextResponse.next();
}
```

## 适用场景

### SEO 优化策略选择

| 场景     | 推荐策略                | 说明                    |
| -------- | ----------------------- | ----------------------- |
| 博客文章 | 静态生成 + ISR          | 内容相对稳定,适合预生成 |
| 产品页面 | 静态生成 + 按需重新验证 | 库存变化时重新验证      |
| 用户资料 | 服务端渲染              | 内容个性化,不适合缓存   |
| 搜索结果 | 客户端渲染              | 动态内容,不需要 SEO     |

### 元数据优化优先级

| 元数据类型  | 优先级 | 说明                 |
| ----------- | ------ | -------------------- |
| title       | 高     | 最重要的 SEO 因素    |
| description | 高     | 影响点击率           |
| Open Graph  | 中     | 社交媒体分享         |
| 结构化数据  | 中     | 富媒体搜索结果       |
| keywords    | 低     | 现代搜索引擎不太重视 |

```typescript
      item: item.href,
    })),
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
      />
      <nav aria-label="Breadcrumb">
        <ol>
          {items.map((item, index) => (
            <li key={index}>
              {index < items.length - 1 ? (
                <a href={item.href}>{item.name}</a>
              ) : (
                <span>{item.name}</span>
              )}
            </li>
          ))}
        </ol>

    </>
  );
```

## 总结

动态路由的 SEO 优化是一个系统工程,需要从多个方面入手。通过本文的学习,我们掌握了:

### 核心技术

1. **元数据生成**: 使用 `generateMetadata` 动态生成元数据
2. **结构化数据**: 添加 JSON-LD 结构化数据
3. **Sitemap 生成**: 动态生成 sitemap.xml
4. **Robots.txt**: 配置爬虫访问规则
5. **URL 优化**: 语义化 URL 和规范化

### 最佳实践

1. **使用静态生成**: 尽可能使用 SSG + ISR
2. **优化图片**: 使用 Next.js Image 组件
3. **语义化 HTML**: 使用正确的 HTML 标签
4. **性能优化**: 优化 Core Web Vitals
5. **移动端优化**: 响应式设计和移动端友好

### 注意事项

1. 避免重复内容
2. 正确处理 404 页面
3. 使用 301 重定向
4. 优化移动端体验
5. 监控性能指标

通过合理应用这些 SEO 优化技术,可以显著提升动态路由页面的搜索引擎排名和用户体验。
