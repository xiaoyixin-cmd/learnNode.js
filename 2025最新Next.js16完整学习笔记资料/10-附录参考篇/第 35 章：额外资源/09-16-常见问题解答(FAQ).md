## 10. 错误处理

### 10.1 "headers() and cookies() are now async" 错误

**问题**: 升级到 Next.js 16 后报错

**解答**:

```typescript
// ❌ Next.js 15
import { headers } from "next/headers";

export function Component() {
  const headersList = headers();
  const userAgent = headersList.get("user-agent");
}

// ✅ Next.js 16
import { headers } from "next/headers";

export async function Component() {
  const headersList = await headers();
  const userAgent = headersList.get("user-agent");
}
```

### 11.2 如何定义 Server Action 类型?

**问题**: Server Action 类型错误

**解答**:

```typescript
"use server";

import { z } from "zod";

const schema = z.object({
  title: z.string().min(1),
  content: z.string().min(1),
});

export async function createPost(formData: FormData): Promise<{
  success?: boolean;
  error?: string;
}> {
  const data = {
    title: formData.get("title"),
    content: formData.get("content"),
  };

  const result = schema.safeParse(data);

  if (!result.success) {
    return { error: "Invalid data" };
  }

  await db.post.create({ data: result.data });

  return { success: true };
}
```

## 12. 认证和授权

### 12.1 如何实现用户认证?

**问题**: 如何添加登录功能

**解答**:

```bash
npm install next-auth
```

```typescript
// app/api/auth/[...nextauth]/route.ts
import NextAuth from "next-auth";
import CredentialsProvider from "next-auth/providers/credentials";

const handler = NextAuth({
  providers: [
    CredentialsProvider({
      credentials: {
        email: { label: "Email", type: "email" },
        password: { label: "Password", type: "password" },
      },
      async authorize(credentials) {
        // 验证用户
        const user = await db.user.findUnique({
          where: { email: credentials?.email },
        });

        if (
          user &&
          (await verifyPassword(credentials?.password, user.password))
        ) {
          return user;
        }

        return null;
      },
    }),
  ],
});

export { handler as GET, handler as POST };
```

```typescript
import { getToken } from "next-auth/jwt";
import { NextResponse } from "next/server";
import type { NextRequest } from "next/server";

export async function middleware(request: NextRequest) {
  const token = await getToken({ req: request });
```

## 5. 渲染和缓存

### 5.1 Server Component 和 Client Component 有什么区别?

**问题**: 不知道何时使用哪种组件

**解答**:

**Server Component** (默认):

- 在服务端渲染
- 可以直接访问数据库
- 不能使用 hooks
- 不能使用浏览器 API

**Client Component** ('use client'):

- 在客户端渲染
- 可以使用 hooks
- 可以使用浏览器 API
- 可以添加交互

**使用场景**:

| 场景           | 组件类型 |
| :------------- | :------- |
| 数据获取       | Server   |
| 静态内容       | Server   |
| 交互功能       | Client   |
| 使用 hooks     | Client   |
| 使用浏览器 API | Client   |

---

### 5.2 如何控制缓存?

**问题**: 数据不更新

**解答**:

**fetch 缓存**:

```typescript
// 不缓存

// 缓存并重新验证
fetch(url, { next: { revalidate: 3600 } });

// 永久缓存
fetch(url, { cache: "force-cache" });
```

**路由缓存**:

```typescript

export const revalidate = 3600; // 1小时重新验证
export const dynamic = "force-dynamic"; // 强制动态渲染
```
