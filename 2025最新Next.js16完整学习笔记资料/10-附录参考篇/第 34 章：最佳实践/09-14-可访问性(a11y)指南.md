# 09-14-可访问性(a11y)指南

## 1. 概述

### 1.1 什么是可访问性

可访问性(Accessibility,简称 a11y)是指让所有人,包括残障人士,都能使用 Web 应用。这包括视觉、听觉、运动和认知障碍的用户。

### 1.2 为什么需要可访问性

**重要性**:

- 法律要求(ADA、Section 508、WCAG)
- 扩大用户群体
- 改善 SEO
- 提升用户体验
- 社会责任

### 1.3 WCAG 标准

| 级别 | 说明 | 要求     |
| :--- | :--- | :------- |
| A    | 基本 | 最低要求 |
| AA   | 中等 | 推荐标准 |
| AAA  | 高级 | 最高标准 |

### 1.4 四大原则

**POUR 原则**:

1. **Perceivable**(可感知): 用户能感知内容
2. **Operable**(可操作): 用户能操作界面
3. **Understandable**(可理解): 用户能理解内容
4. **Robust**(健壮): 内容能被各种工具访问

### 4.3 焦点陷阱

**模态框焦点陷阱**:

```typescript
"use client";

import { useEffect, useRef } from "react";

export function Modal({ isOpen, onClose, children }: ModalProps) {
  const modalRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (!isOpen) return;

    const modal = modalRef.current;
    if (!modal) return;

    const focusableElements = modal.querySelectorAll(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );

    const firstElement = focusableElements[0] as HTMLElement;
    const lastElement = focusableElements[
      focusableElements.length - 1
    ] as HTMLElement;

    firstElement?.focus();

    const handleTab = (e: KeyboardEvent) => {
      if (e.key !== "Tab") return;

      if (e.shiftKey) {
        if (document.activeElement === firstElement) {
          e.preventDefault();
          lastElement?.focus();
        }
      } else {
        if (document.activeElement === lastElement) {
          e.preventDefault();
          firstElement?.focus();
        }
      }
    };

    modal.addEventListener("keydown", handleTab);
    return () => modal.removeEventListener("keydown", handleTab);
  }, [isOpen]);

  if (!isOpen) return null;

  return (
    <div ref={modalRef} role="dialog" aria-modal="true" className="modal">
      {children}
      <button onClick={onClose}>关闭</button>
    </div>
  );
}
```

**解答**:

// ✅ 使用 aria-label

  return (
    <button aria-label="关闭">

  return (
    <button>
      <CloseIcon aria-hidden="true" />

  );
}
