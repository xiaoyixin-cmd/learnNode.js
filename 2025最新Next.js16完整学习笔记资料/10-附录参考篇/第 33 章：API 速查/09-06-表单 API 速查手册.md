# 09-06-表单 API 速查手册

## 1. 概述 (Overview)

### 1.1 什么是表单 API

表单 API 是 Next.js 提供的用于处理表单提交、数据验证和服务器操作的接口。Next.js 16 通过 Server Actions 提供了强大的表单处理能力。

### 1.2 Next.js 16 表单处理特性

**核心特性**:

- Server Actions: 服务器端表单处理
- useFormStatus: 表单状态钩子
- useFormState: 表单状态管理
- useOptimistic: 乐观更新
- Progressive Enhancement: 渐进增强支持

### 1.3 表单处理方式对比

| 方式           | 执行环境 | 适用场景       | 优势           |
| :------------- | :------- | :------------- | :------------- |
| Server Actions | 服务器   | 数据提交、验证 | 安全、SEO 友好 |
| Client Actions | 客户端   | 交互验证       | 即时反馈       |
| API Routes     | 服务器   | RESTful API    | 灵活性高       |
| Form Actions   | 混合     | 渐进增强       | 无 JS 也能工作 |

## 2. Server Actions 基础

### 2.1 定义 Server Action

**功能**: 在服务器端处理表单提交

**基本语法**:

```typescript
"use server";

export async function submitForm(formData: FormData) {
  const name = formData.get("name");
  const email = formData.get("email");

  // 处理数据
  await saveToDatabase({ name, email });

  return { success: true };
}
```

**使用示例**:

```typescript
// app/actions/form.ts
"use server";

import { revalidatePath } from "next/cache";
import { redirect } from "next/navigation";

export async function createPost(formData: FormData) {
  const title = formData.get("title") as string;
  const content = formData.get("content") as string;

  // 验证数据
  if (!title || !content) {
    return { error: "Title and content are required" };
  }

  // 保存到数据库
  const post = await db.post.create({
    data: { title, content },
  });

  // 重新验证缓存
  revalidatePath("/posts");

  // 重定向到新文章
  redirect(`/posts/${post.id}`);
}

// app/posts/new/page.tsx
import { createPost } from "@/app/actions/form";

export default function NewPost() {
  return (
    <form action={createPost}>
      <input type="text" name="title" required />
      <textarea name="content" required />
      <button type="submit">Create Post</button>
    </form>
  );
}
```

**注意事项**:

- 必须在文件顶部添加 'use server'
- 只能在 Server Components 中直接使用
- 自动处理 CSRF 保护
- 支持渐进增强

**适用场景**:

- 表单提交
- 数据创建/更新/删除
- 文件上传
- 数据验证

### 2.2 内联 Server Action

**功能**: 在组件内定义 Server Action

**使用示例**:

```typescript
// app/posts/[id]/page.tsx
export default function Post({ params }: { params: Promise<{ id: string }> }) {
  async function updatePost(formData: FormData) {
    "use server";

    const { id } = await params;
    const title = formData.get("title") as string;

    await db.post.update({
      where: { id },
      data: { title },

    revalidatePath(`/posts/${id}`);

  return (
    <form action={updatePost}>
      <input type="text" name="title" />
      <button type="submit">Update</button>
    </form>
  );
}
```

**注意事项**:

- 'use server' 放在函数内部
- 只能在该组件中使用
- 可以访问组件作用域

**适用场景**:

- 组件特定的操作
- 需要访问组件状态
- 简单的表单处理

**功能**: 管理表单状态和错误

**类型签名**:

```typescript
function useFormState<State>(
```

**使用示例**:

```typescript
// 复杂状态管理
("use server");

interface FormState {
  errors?: {
    title?: string[];
    content?: string[];
  };
  message?: string;
}

export async function createPost(
  prevState: FormState,
  formData: FormData
): Promise<FormState> {
  const title = formData.get("title") as string;
  const content = formData.get("content") as string;

  // 验证
  const errors: FormState["errors"] = {};

  if (!title || title.length < 3) {
    errors.title = ["Title must be at least 3 characters"];
  }

  if (!content || content.length < 10) {
    errors.content = ["Content must be at least 10 characters"];
  }

  if (Object.keys(errors).length > 0) {
    return { errors };
  }

  // 保存
  await db.post.create({ data: { title, content } });

  return { message: "Post created successfully" };
}

// 使用
("use client");

import { useFormState } from "react-dom";
import { createPost } from "@/app/actions/post";

export default function NewPost() {

        <textarea name="content" />
```

## 4. 表单验证

### 4.1 服务器端验证

**功能**: 在服务器端验证表单数据

**使用示例**:

```typescript
// app/actions/user.ts
"use server";

import { z } from "zod";

const userSchema = z.object({
  name: z.string().min(2, "Name must be at least 2 characters"),
  email: z.string().email("Invalid email address"),
  age: z.number().min(18, "Must be at least 18 years old"),
});

export async function createUser(formData: FormData) {
  const rawData = {
    name: formData.get("name"),
    email: formData.get("email"),
    age: Number(formData.get("age")),
  };

  // 验证
  const result = userSchema.safeParse(rawData);

  if (!result.success) {
    return {
      errors: result.error.flatten().fieldErrors,
    };
  }

  // 保存用户
  const user = await db.user.create({
    data: result.data,
  });

  return { success: true, user };
}

// 自定义验证
export async function registerUser(formData: FormData) {
  const email = formData.get("email") as string;
  const password = formData.get("password") as string;
  const confirmPassword = formData.get("confirmPassword") as string;

  const errors: Record<string, string> = {};

  // 邮箱验证
  if (!email || !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
    errors.email = "Invalid email address";
  }

  // 密码验证
  if (!password || password.length < 8) {
    errors.password = "Password must be at least 8 characters";
  }

  // 密码确认
  if (password !== confirmPassword) {
    errors.confirmPassword = "Passwords do not match";
  }

  // 检查邮箱是否已存在
  const existingUser = await db.user.findUnique({
    where: { email },
  });

  if (existingUser) {
    errors.email = "Email already exists";
  }

  if (Object.keys(errors).length > 0) {
    return { errors };
  }

  // 创建用户
  const user = await db.user.create({
    data: { email, password: await hashPassword(password) },
  });

  return { success: true, user };
}
```

**验证库推荐**:

| 库           | 特点            | 适用场景     |
| :----------- | :-------------- | :----------- |
| Zod          | TypeScript 优先 | 类型安全验证 |
| Yup          | 简单易用        | 快速验证     |
| Joi          | 功能丰富        | 复杂验证     |
| Validator.js | 轻量级          | 简单验证     |

**注意事项**:

- 永远在服务器端验证
- 客户端验证只是辅助
- 返回清晰的错误信息
- 考虑安全性

**适用场景**:

- 用户注册
- 数据提交
- 表单验证
- 数据完整性检查

### 4.2 客户端验证

**功能**: 在客户端提供即时反馈

**使用示例**:

```typescript
// app/components/RegisterForm.tsx
"use client";

import { useState } from "react";
import { registerUser } from "@/app/actions/user";

export function RegisterForm() {
  const [errors, setErrors] = useState<Record<string, string>>({});

  function validateEmail(email: string) {
    if (!email) return "Email is required";
    if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
      return "Invalid email address";
    }
    return "";
  }

  function validatePassword(password: string) {
    if (!password) return "Password is required";
    if (password.length < 8) {
      return "Password must be at least 8 characters";
    }
    return "";
  }

  async function handleSubmit(formData: FormData) {
    const email = formData.get("email") as string;
    const password = formData.get("password") as string;

    // 客户端验证
    const newErrors: Record<string, string> = {};

    const emailError = validateEmail(email);
    if (emailError) newErrors.email = emailError;

    const passwordError = validatePassword(password);
    if (passwordError) newErrors.password = passwordError;

    if (Object.keys(newErrors).length > 0) {
      setErrors(newErrors);
      return;
    }

    // 提交到服务器
    const result = await registerUser(formData);

    if (result.errors) {
      setErrors(result.errors);
    }
  }

  return (
    <form action={handleSubmit}>
      <div>
        <input
          type="email"
          name="email"
          onChange={(e) => {
            const error = validateEmail(e.target.value);
            setErrors((prev) => ({ ...prev, email: error }));
          }}
        />
        {errors.email && <p className="error">{errors.email}</p>}
      </div>

      <div>
        <input
          type="password"
          name="password"
          onChange={(e) => {
            const error = validatePassword(e.target.value);
            setErrors((prev) => ({ ...prev, password: error }));
          }}
        />
        {errors.password && <p className="error">{errors.password}</p>}
      </div>

      <button type="submit">Register</button>
    </form>
  );
}

// 使用 HTML5 验证
export function SimpleForm() {

      />

      <input
        type="password"
        name="password"
        required
        minLength={8}
        title="Password must be at least 8 characters"
      />

    </form>
```

## 5. 文件上传

### 5.1 基本文件上传

**功能**: 处理文件上传

```typescript
// app/actions/upload.ts
"use server";

import { writeFile } from "fs/promises";
import { join } from "path";

export async function uploadFile(formData: FormData) {
  const file = formData.get("file") as File;

  if (!file) {
    return { error: "No file uploaded" };
  }

  // 验证文件类型
  const allowedTypes = ["image/jpeg", "image/png", "image/gif"];
  if (!allowedTypes.includes(file.type)) {
    return { error: "Invalid file type" };
  }

  // 验证文件大小 (5MB)
  if (file.size > 5 * 1024 * 1024) {
    return { error: "File too large" };
  }

  // 保存文件
  const bytes = await file.arrayBuffer();
  const buffer = Buffer.from(bytes);

  const path = join(process.cwd(), "public", "uploads", file.name);
  await writeFile(path, buffer);

  return { success: true, path: `/uploads/${file.name}` };
}

// app/upload/page.tsx
import { uploadFile } from "@/app/actions/upload";

export default function UploadPage() {
  return (
    <form action={uploadFile}>
      <input type="file" name="file" accept="image/*" />

    </form>
  );
}

// 多文件上传
("use server");

export async function uploadMultipleFiles(formData: FormData) {
  const files = formData.getAll("files") as File[];

  if (files.length === 0) {
    return { error: "No files uploaded" };
  }

  const uploadedFiles = [];

  for (const file of files) {
    const bytes = await file.arrayBuffer();
    const buffer = Buffer.from(bytes);

    const path = join(process.cwd(), "public", "uploads", file.name);
    await writeFile(path, buffer);

    uploadedFiles.push(`/uploads/${file.name}`);
  }

  return { success: true, files: uploadedFiles };
}

// 使用
export default function MultiUploadPage() {
  return (
    <form action={uploadMultipleFiles}>
      <input type="file" name="files" multiple accept="image/*" />
      <button type="submit">Upload</button>
    </form>
  );
}
```

### 5.2 云存储上传

**功能**: 上传文件到云存储服务

**使用示例**:

```typescript
// app/actions/upload-s3.ts
"use server";

import { S3Client, PutObjectCommand } from "@aws-sdk/client-s3";
import { getSignedUrl } from "@aws-sdk/s3-request-presigner";

const s3Client = new S3Client({
  region: process.env.AWS_REGION,
  credentials: {
    accessKeyId: process.env.AWS_ACCESS_KEY_ID!,
    secretAccessKey: process.env.AWS_SECRET_ACCESS_KEY!,
  },
});

export async function uploadToS3(formData: FormData) {
  const file = formData.get("file") as File;

  if (!file) {
    return { error: "No file provided" };
  }

  const bytes = await file.arrayBuffer();
  const buffer = Buffer.from(bytes);

  const key = `uploads/${Date.now()}-${file.name}`;

  const command = new PutObjectCommand({
    Bucket: process.env.AWS_BUCKET_NAME,
    Key: key,
    Body: buffer,
    ContentType: file.type,
  });

  await s3Client.send(command);

  const url = `https://${process.env.AWS_BUCKET_NAME}.s3.${process.env.AWS_REGION}.amazonaws.com/${key}`;

  return { success: true, url };
}

// 使用预签名 URL
export async function getUploadUrl(filename: string, contentType: string) {
  const key = `uploads/${Date.now()}-${filename}`;

  const command = new PutObjectCommand({
    Bucket: process.env.AWS_BUCKET_NAME,
    Key: key,
    ContentType: contentType,
  });

  const signedUrl = await getSignedUrl(s3Client, command, { expiresIn: 3600 });

  return { signedUrl, key };
}

// 客户端直接上传
("use client");

export function DirectUpload() {
  async function handleUpload(formData: FormData) {
    const file = formData.get("file") as File;

    // 获取预签名 URL
    const { signedUrl } = await getUploadUrl(file.name, file.type);

    // 直接上传到 S3
    await fetch(signedUrl, {
      method: "PUT",
      body: file,
      headers: {
        "Content-Type": file.type,
      },
    });

    alert("Upload successful!");
  }

  return (
```

### 7.1 用户注册登录

**场景描述**: 处理用户认证表单

**实现方案**:

```typescript
// app/actions/auth.ts
"use server";

import { z } from "zod";
import { cookies } from "next/headers";
import { redirect } from "next/navigation";

const registerSchema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
  name: z.string().min(2),
});

export async function register(prevState: any, formData: FormData) {
  const result = registerSchema.safeParse({
    email: formData.get("email"),
    password: formData.get("password"),
    name: formData.get("name"),
  });

  if (!result.success) {
    return {
      errors: result.error.flatten().fieldErrors,
    };
  }

  // 检查用户是否存在
  const existing = await db.user.findUnique({
    where: { email: result.data.email },
  });

  if (existing) {
    return {
      errors: { email: ["Email already exists"] },
    };
  }

  // 创建用户
  const user = await db.user.create({
    data: {
      ...result.data,
      password: await hashPassword(result.data.password),
    },
  });

  // 设置 session
  const cookieStore = await cookies();
  cookieStore.set("session", await createSession(user.id), {
    httpOnly: true,
    secure: process.env.NODE_ENV === "production",
    sameSite: "lax",
    maxAge: 60 * 60 * 24 * 7, // 7 days
  });

  redirect("/dashboard");
}
```

### 9.5 如何处理文件上传进度?

**问题**: 需要显示上传进度

**解决方案**:

```typescript
"use client";

import { useState } from "react";

export function UploadWithProgress() {
  const [progress, setProgress] = useState(0);

  async function handleUpload(formData: FormData) {
    const file = formData.get("file") as File;

    // 获取预签名 URL
    const { signedUrl } = await getUploadUrl(file.name, file.type);

    // 使用 XMLHttpRequest 跟踪进度
    return new Promise((resolve, reject) => {
      const xhr = new XMLHttpRequest();

      xhr.upload.addEventListener("progress", (e) => {
        if (e.lengthComputable) {
          const percent = (e.loaded / e.total) * 100;
          setProgress(percent);
        }
      });

      xhr.addEventListener("load", () => resolve(xhr.response));
      xhr.addEventListener("error", () => reject(xhr.statusText));

      xhr.open("PUT", signedUrl);
      xhr.setRequestHeader("Content-Type", file.type);
      xhr.send(file);
    });
  }

  return (
    <form action={handleUpload}>
      <input type="file" name="file" />
      <button type="submit">Upload</button>
      {progress > 0 && (
        <div>
          <progress value={progress} max="100" />
          <span>{Math.round(progress)}%</span>
        </div>
      )}
    </form>
  );
}
```
