# 媒体性能监控

## 1. 概述 (Overview)

媒体性能监控是确保网站提供良好用户体验的关键环节。通过监控图像、视频等媒体资源的加载性能,我们可以及时发现和解决性能问题。在 Next.js 16 中,我们可以使用多种工具和 API 来监控媒体性能。

### 1.1 为什么需要性能监控

**发现问题**: 及时发现性能瓶颈和问题。

**优化决策**: 基于数据做出优化决策。

**用户体验**: 确保良好的用户体验。

**业务指标**: 性能直接影响转化率和用户留存。

### 1.2 监控指标

**加载时间**: 媒体资源的加载时间。

**文件大小**: 媒体文件的体积。

**Core Web Vitals**: LCP、FID、CLS 等指标。

**带宽消耗**: 总带宽使用量。

**错误率**: 加载失败的比例。

### 1.3 Next.js 16 的监控特性

🆕 **Next.js 16 新增**: 改进的性能监控 API。

⚡ **Next.js 16 增强**: 更好的 Web Vitals 集成。

🆕 **Next.js 16 新增**: 自动性能报告。

```tsx
export default function ResourceMonitor() {

      for (const entry of list.getEntries()) {
        if (entry.entryType === "resource") {
          const resourceEntry = entry as PerformanceResourceTiming;

          // 只监控图像和视频

            resourceEntry.name.match(

            });
          }
        }
      }
    });

    observer.observe({ entryTypes: ["resource"] });

    return () => observer.disconnect();

}
```

## 6. 自定义性能监控 (Custom Performance Monitoring)

### 6.1 性能监控 Hook

```tsx
// app/hooks/usePerformanceMonitor.ts
"use client";

import { useEffect, useState } from "react";

interface PerformanceMetrics {
  fcp: number;
  lcp: number;
  fid: number;
  cls: number;
  ttfb: number;
}

export function usePerformanceMonitor() {
  const [metrics, setMetrics] = useState<Partial<PerformanceMetrics>>({});

  useEffect(() => {
    const observer = new PerformanceObserver((list) => {
      for (const entry of list.getEntries()) {
        if (
          entry.entryType === "paint" &&
          entry.name === "first-contentful-paint"
        ) {
          setMetrics((prev) => ({ ...prev, fcp: entry.startTime }));
        }

        if (entry.entryType === "largest-contentful-paint") {
          setMetrics((prev) => ({ ...prev, lcp: entry.startTime }));
        }

        if (entry.entryType === "first-input") {
          const fidEntry = entry as PerformanceEventTiming;
          setMetrics((prev) => ({
            ...prev,
            fid: fidEntry.processingStart - fidEntry.startTime,
          }));
        }

        if (
          entry.entryType === "layout-shift" &&
          !(entry as any).hadRecentInput
        ) {
          setMetrics((prev) => ({
            ...prev,
            cls: (prev.cls || 0) + (entry as any).value,
          }));
        }
      }
    });

    observer.observe({
      entryTypes: [
        "paint",
        "largest-contentful-paint",
        "first-input",
        "layout-shift",
      ],
    });

    // TTFB
    const navEntry = performance.getEntriesByType(
      "navigation"
    )[0] as PerformanceNavigationTiming;
    if (navEntry) {
      setMetrics((prev) => ({
        ...prev,
        ttfb: navEntry.responseStart - navEntry.requestStart,
      }));
    }

    return () => observer.disconnect();
  }, []);

  return metrics;
}
```

### 6.3 带宽监控

```tsx
// app/hooks/useBandwidthMonitor.ts
"use client";

import { useEffect, useState } from "react";

export function useBandwidthMonitor() {
  const [bandwidth, setBandwidth] = useState({
    totalBytes: 0,
    imageBytes: 0,
    videoBytes: 0,
  });

  useEffect(() => {
    const observer = new PerformanceObserver((list) => {
      for (const entry of list.getEntries()) {
        if (entry.entryType === "resource") {
          const resourceEntry = entry as PerformanceResourceTiming;
          const size = resourceEntry.transferSize;

          setBandwidth((prev) => ({
            totalBytes: prev.totalBytes + size,
            imageBytes:
              prev.imageBytes +
              (resourceEntry.name.match(/\.(jpg|jpeg|png|gif|webp|avif)$/i)
                ? size
                : 0),
            videoBytes:
              prev.videoBytes +
              (resourceEntry.name.match(/\.(mp4|webm)$/i) ? size : 0),
          }));
        }
      }
    });

    observer.observe({ entryTypes: ["resource"] });

    return () => observer.disconnect();
  }, []);

  return {
    totalMB: (bandwidth.totalBytes / 1024 / 1024).toFixed(2),
    imageMB: (bandwidth.imageBytes / 1024 / 1024).toFixed(2),
    videoMB: (bandwidth.videoBytes / 1024 / 1024).toFixed(2),
  };
}
```

```tsx
"use client";

import { useRef, useState, useEffect } from "react";

export default function CourseVideoMonitor() {
  const videoRef = useRef<HTMLVideoElement>(null);
  const [metrics, setMetrics] = useState({
    loadTime: 0,
    bufferCount: 0,
    playTime: 0,
  });

  useEffect(() => {
    const video = videoRef.current;
    if (!video) return;

    const startTime = Date.now();

    const handleLoadedData = () => {
      setMetrics((prev) => ({ ...prev, loadTime: Date.now() - startTime }));
    };

    const handleWaiting = () => {
      setMetrics((prev) => ({ ...prev, bufferCount: prev.bufferCount + 1 }));
    };

    const handleTimeUpdate = () => {
      setMetrics((prev) => ({ ...prev, playTime: video.currentTime }));
    };

    video.addEventListener("loadeddata", handleLoadedData);
    video.addEventListener("waiting", handleWaiting);
    video.addEventListener("timeupdate", handleTimeUpdate);

    return () => {
      video.removeEventListener("loadeddata", handleLoadedData);
      video.removeEventListener("waiting", handleWaiting);
      video.removeEventListener("timeupdate", handleTimeUpdate);
    };
  }, []);

  return (
    <div>
      <video ref={videoRef} controls>
        <source src="/videos/course.mp4" type="video/mp4" />
      </video>
      <div className="mt-4 p-4 bg-gray-100 rounded">
        <p>加载时间: {metrics.loadTime}ms</p>
        <p>缓冲次数: {metrics.bufferCount}</p>
        <p>播放时长: {metrics.playTime.toFixed(2)}s</p>
      </div>
    </div>
  );
}
```

```tsx
"use client";

import { useRef, useState, useEffect } from "react";

export default function LiveStreamMonitor() {
  const videoRef = useRef<HTMLVideoElement>(null);
  const [streamMetrics, setStreamMetrics] = useState({
    bitrate: 0,
    fps: 0,
    droppedFrames: 0,
    bufferHealth: 0,
  });

  useEffect(() => {
    const video = videoRef.current;
    if (!video) return;

    const interval = setInterval(() => {
      if (video.getVideoPlaybackQuality) {
        const quality = video.getVideoPlaybackQuality();
        setStreamMetrics((prev) => ({
          ...prev,
          droppedFrames: quality.droppedVideoFrames,
        }));
      }

      // 计算缓冲区健康度
      if (video.buffered.length > 0) {
        const buffered = video.buffered.end(0) - video.currentTime;
        setStreamMetrics((prev) => ({
          ...prev,
          bufferHealth: buffered,
        }));
      }
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return (
    <div>
      <video ref={videoRef} controls autoPlay>
        <source src="/live/stream.m3u8" type="application/x-mpegURL" />
      </video>
      <div className="mt-4 p-4 bg-gray-100 rounded">
        <h3 className="font-bold mb-2">直播质量</h3>
        <p>丢帧数: {streamMetrics.droppedFrames}</p>
        <p>缓冲区: {streamMetrics.bufferHealth.toFixed(2)}s</p>
      </div>
    </div>
  );
}
```

```tsx
import { useEffect, useState } from "react";

export default function GameAssetMonitor() {

    totalSize: 0,

              loadTime: Date.now() - startTime,

        }
      }

    observer.observe({ entryTypes: ["resource"] });

    return () => observer.disconnect();

  return (

      <p>加载时间: {assetMetrics.loadTime}ms</p>
      <div className="mt-2 w-full bg-gray-200 rounded-full h-2">
        <div
          className="bg-blue-500 h-2 rounded-full"
          style={{
            width: `${
              assetMetrics.totalAssets > 0
                ? (assetMetrics.loadedAssets / assetMetrics.totalAssets) * 100
                : 0
            }%`,
          }}
        />
      </div>
    </div>
  );
}
```

## 9. 注意事项 (Precautions)

### 9.1 性能开销

**监控本身的开销**: 性能监控会消耗资源,不要过度监控。

**采样策略**: 对于高流量网站,使用采样减少开销。

```tsx
// 只监控10%的用户
if (Math.random() < 0.1) {
  // 启用监控
}
```

### 9.2 隐私考虑

**用户隐私**: 不要收集敏感信息。

**GDPR 合规**: 遵守数据保护法规。

**用户同意**: 获取用户同意后再收集数据。

### 9.3 数据存储

**数据量**: 性能数据可能很大,需要合理存储。

**数据保留**: 设置合理的数据保留期限。

**数据聚合**: 聚合数据减少存储需求。

### 9.4 错误处理

**监控失败**: 监控代码本身可能失败,需要容错处理。

```tsx
try {
  // 监控代码
} catch (error) {
  console.error("监控失败:", error);
}
```

### 10.3 如何优化监控性能?

**问题**: 监控影响页面性能怎么办?

**解决方案**:

1. 使用 PerformanceObserver 而非轮询
2. 批量发送数据
3. 使用 requestIdleCallback
4. 异步发送数据

```tsx
"use client";

import { useEffect } from "react";

export default function OptimizedMonitor() {
  useEffect(() => {
    const metrics: any[] = [];

    const observer = new PerformanceObserver((list) => {
      metrics.push(...list.getEntries());
    });

    observer.observe({ entryTypes: ["resource"] });

    // 批量发送
    const interval = setInterval(() => {
      if (metrics.length > 0) {
        fetch("/api/metrics", {
          method: "POST",
          body: JSON.stringify(metrics),
        });
        metrics.length = 0;
      }
    }, 10000); // 每10秒发送一次

    return () => {
      clearInterval(interval);
      observer.disconnect();
    };
  }, []);

  return null;
}
```

### 10.4 如何监控第三方资源?

**问题**: 如何监控 CDN 等第三方资源?

**解决方案**: 使用 Resource Timing API。

```tsx
"use client";

import { useEffect } from "react";

export default function ThirdPartyMonitor() {
  useEffect(() => {
    const observer = new PerformanceObserver((list) => {
      for (const entry of list.getEntries()) {
        const resourceEntry = entry as PerformanceResourceTiming;

        // 检查是否是第三方资源
        if (!resourceEntry.name.includes(window.location.hostname)) {
          console.log("第三方资源:", {
            url: resourceEntry.name,
            duration: resourceEntry.duration,
            size: resourceEntry.transferSize,
          });
        }
      }
    });

    observer.observe({ entryTypes: ["resource"] });

    return () => observer.disconnect();
  }, []);

  return null;
}
```

### 10.6 如何生成性能报告?

**问题**: 如何生成可视化的性能报告?

**解决方案**: 使用 Lighthouse 或自定义报告。

```bash
# 使用Lighthouse
npx lighthouse https://your-site.com --output html --output-path ./report.html
```

```tsx
// 自定义报告组件
"use client";

import { usePerformanceMonitor } from "@/hooks/usePerformanceMonitor";

export default function PerformanceReport() {
  const metrics = usePerformanceMonitor();

  const getScore = (metric: string, value: number) => {
    const thresholds: any = {
      lcp: { good: 2500, poor: 4000 },
      fid: { good: 100, poor: 300 },
      cls: { good: 0.1, poor: 0.25 },
    };

    const threshold = thresholds[metric];
    if (!threshold) return "unknown";

    if (value <= threshold.good) return "good";
    if (value <= threshold.poor) return "needs-improvement";
    return "poor";
  };

  return (
    <div className="p-4">
      <h2 className="text-2xl font-bold mb-4">性能报告</h2>
      <div className="space-y-4">
        {Object.entries(metrics).map(([key, value]) => (
          <div key={key} className="flex items-center justify-between">
            <span className="font-semibold">{key.toUpperCase()}</span>
            <span
              className={`px-2 py-1 rounded ${
                getScore(key, value as number) === "good"
                  ? "bg-green-200"
                  : getScore(key, value as number) === "needs-improvement"
                  ? "bg-yellow-200"
                  : "bg-red-200"
              }`}
            >
              {typeof value === "number" ? value.toFixed(2) : value}
            </span>
          </div>
        ))}
      </div>
    </div>
  );
}
```
