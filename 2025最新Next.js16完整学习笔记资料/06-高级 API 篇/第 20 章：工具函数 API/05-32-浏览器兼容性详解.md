# 05-32-浏览器兼容性详解

## 概述

Next.js 16 提供了完善的浏览器兼容性支持,通过自动 polyfill、代码转译和特性检测,确保应用在各种浏览器中正常运行。Next.js 默认支持现代浏览器,同时提供了配置选项来支持旧版浏览器。通过 browserslist 配置,你可以精确控制目标浏览器范围。

### 支持的浏览器

- **Chrome**: 64+
- **Edge**: 79+
- **Firefox**: 67+
- **Safari**: 12.1+
- **iOS Safari**: 12.1+
- **Opera**: 51+

### 兼容性策略

| 策略       | 描述          | 适用场景         |
| ---------- | ------------- | ---------------- |
| 现代浏览器 | 默认配置      | 大多数应用       |
| 扩展支持   | 添加 polyfill | 需要支持旧浏览器 |
| 渐进增强   | 特性检测      | 优雅降级         |

## 6. 注意事项

### 6.1 Polyfill 体积

避免加载过多 polyfill:

```tsx
// 按需加载
if (!("fetch" in window)) {
  import("whatwg-fetch");
}
```

### 6.2 性能影响

Polyfill 会增加包体积:

```js
// 只为需要的浏览器添加
{
  "browserslist": [
    ">0.3%",
    "not dead"
  ]
}
```

### 6.3 特性检测

使用特性检测而非浏览器检测:

```tsx
// ✓ 推荐
if ("IntersectionObserver" in window) {
  // 使用IntersectionObserver
}

// ✗ 不推荐
if (navigator.userAgent.includes("Chrome")) {
  // 基于浏览器判断
}
```

## 9. 深入理解 Polyfill

### 9.1 Polyfill 原理

Polyfill 是用于填补浏览器功能缺失的代码:

```tsx
// Polyfill示例: Array.prototype.includes
if (!Array.prototype.includes) {
  Array.prototype.includes = function (element) {
    return this.indexOf(element) !== -1;
  };
}
```

### 9.2 常用 Polyfill

```tsx
// app/polyfills.ts
import "core-js/stable";
import "regenerator-runtime/runtime";
import "intersection-observer";
import "resize-observer-polyfill";
import "whatwg-fetch";
```

### 9.3 按需加载策略

```tsx
"use client";

import { useEffect } from "react";

export default function DynamicPolyfills() {
  useEffect(() => {
    const loadPolyfills = async () => {
      const polyfills = [];

      if (!("IntersectionObserver" in window)) {
        polyfills.push(import("intersection-observer"));
      }

      if (!("ResizeObserver" in window)) {
        polyfills.push(import("resize-observer-polyfill"));
      }

      if (!("fetch" in window)) {
        polyfills.push(import("whatwg-fetch"));
      }

      await Promise.all(polyfills);
      console.log("Polyfills loaded");
    };

    loadPolyfills();
  }, []);

  return null;
}
```

```tsx
import { useEffect, useState } from "react";

export default function GracefulDegradation() {
  const [useModernAPI, setUseModernAPI] = useState(true);

  useEffect(() => {
    setUseModernAPI("IntersectionObserver" in window);
  }, []);

  if (useModernAPI) {
    return <ModernComponent />;

function ModernComponent() {
  useEffect(() => {
    const observer = new IntersectionObserver((entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          console.log("Element is visible");
        }
      });
    });

    const element = document.querySelector(".target");
    if (element) {
      observer.observe(element);
    }

    return () => observer.disconnect();
  }, []);

  return <div className="target">Modern Component</div>;
}

function LegacyComponent() {
  useEffect(() => {
    const handleScroll = () => {
      const element = document.querySelector(".target");
      if (element) {
        const rect = element.getBoundingClientRect();
        if (rect.top < window.innerHeight) {
          console.log("Element is visible");
        }
      }
    };

    window.addEventListener("scroll", handleScroll);
    return () => window.removeEventListener("scroll", handleScroll);
  }, []);

  return <div className="target">Legacy Component</div>;
}
```

```tsx
export default function FallbackExample() {

  useEffect(() => {
    const fetchData = async () => {
      if ("fetch" in window) {
        const response = await fetch("/api/data");
        const result = await response.json();
        setData(result);
      } else {
        // 降级到XMLHttpRequest
        const xhr = new XMLHttpRequest();
        xhr.open("GET", "/api/data");

          setData(JSON.parse(xhr.responseText));
        };
        xhr.send();
      }
    };

    fetchData();
  }, []);

  return <div>{data ? JSON.stringify(data) : "Loading..."}</div>;
}
```
