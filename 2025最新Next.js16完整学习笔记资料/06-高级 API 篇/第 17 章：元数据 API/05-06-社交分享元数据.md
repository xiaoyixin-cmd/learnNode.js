# 社交分享元数据

## 1. 概述

社交分享元数据决定了你的网站在社交媒体平台上的展示效果。正确配置这些元数据可以显著提升分享链接的点击率和用户参与度。

### 1.1 核心协议

- **Open Graph**: Facebook、LinkedIn 等平台使用
- **Twitter Cards**: Twitter 专用
- **其他平台**: 微信、微博等国内平台

### 1.2 重要性

| 平台     | 协议          | 影响         |
| -------- | ------------- | ------------ |
| Facebook | Open Graph    | 分享预览效果 |
| Twitter  | Twitter Cards | 推文展示效果 |
| LinkedIn | Open Graph    | 职业网络分享 |
| 微信     | Open Graph    | 朋友圈分享   |
| WhatsApp | Open Graph    | 消息预览     |

### 1.3 最佳实践

- 提供高质量的预览图片
- 编写吸引人的标题和描述
- 确保图片尺寸符合平台要求
- 测试不同平台的展示效果
- 使用动态元数据适配不同内容

### 5.2 微信 JS-SDK 配置

```tsx
// components/WeChatShare.tsx
"use client";

import { useEffect } from "react";
import Script from "next/script";

export function WeChatShare({
  title,
  desc,
  link,
  imgUrl,
}: {
  title: string;
  desc: string;
  link: string;
  imgUrl: string;
}) {
  useEffect(() => {
    if (typeof window !== "undefined" && (window as any).wx) {
      const wx = (window as any).wx;

      // 获取微信配置
      fetch("/api/wechat/config", {
        method: "POST",
        body: JSON.stringify({ url: window.location.href }),
      })
        .then((res) => res.json())
        .then((config) => {
          wx.config({
            debug: false,
            appId: config.appId,
            timestamp: config.timestamp,
            nonceStr: config.nonceStr,
            signature: config.signature,
            jsApiList: ["updateAppMessageShareData", "updateTimelineShareData"],
          });

          wx.ready(() => {
            // 分享给朋友
            wx.updateAppMessageShareData({
              title,
              desc,
              link,
              imgUrl,
            });

            // 分享到朋友圈
            wx.updateTimelineShareData({
              title,
              link,
              imgUrl,
            });
          });
        });
    }
  }, [title, desc, link, imgUrl]);

  return (
    <Script
      src="https://res.wx.qq.com/open/js/jweixin-1.6.0.js"
      strategy="afterInteractive"
    />
  );
}
```

## 9. 注意事项

### 9.1 图片要求

**尺寸**: 不同平台有不同的图片尺寸要求,建议使用 1200x630 像素的通用尺寸

**格式**: 使用 JPG 或 PNG 格式,避免使用 GIF

**大小**: 控制在 1MB 以内,确保快速加载

**内容**: 避免在图片边缘放置重要内容,可能被裁剪

### 9.2 文本长度

**标题**:

- Facebook: 60-90 个字符
- Twitter: 70 个字符以内
- LinkedIn: 60 个字符以内

**描述**:

- Facebook: 200 个字符以内
- Twitter: 200 个字符以内
- LinkedIn: 150 个字符以内

### 9.3 缓存问题

社交平台会缓存元数据,更新后需要手动刷新:

```tsx
// 添加版本号强制更新
export const metadata: Metadata = {
  openGraph: {
    images: [`https://example.com/og-image.jpg?v=${Date.now()}`],
  },
};
```

### 9.4 URL 规范

使用绝对 URL 而不是相对 URL:

```tsx
// ❌ 错误
images: ["/og-image.jpg"];

// ✅ 正确
images: ["https://example.com/og-image.jpg"];
```

### 9.5 多语言支持

为不同语言版本配置不同的元数据:

```tsx
export async function generateMetadata({
  params,
}: {
  params: { lang: string };
}): Promise<Metadata> {
  const translations = {
    zh: {
      title: "中文标题",
      description: "中文描述",
    },
    en: {
      title: "English Title",
      description: "English Description",
    },
  };

  const t = translations[params.lang as keyof typeof translations];

  return {
    title: t.title,
    description: t.description,
    openGraph: {
      locale: params.lang === "zh" ? "zh_CN" : "en_US",
    },
  };
}
```

### 9.6 性能优化

动态生成图片时注意性能:

```tsx
// 使用Edge Runtime提升性能
export const runtime = "edge";

// 缓存生成的图片
export const revalidate = 3600; // 1小时
```

## 10. 常见问题

### 10.1 为什么社交平台显示的图片不是我设置的?

**问题**: 设置了 Open Graph 图片,但社交平台显示的是旧图片或默认图片。

**原因**:

1. 社交平台缓存了旧的元数据
2. 图片 URL 不是绝对路径
3. 图片尺寸不符合要求
4. 图片加载失败

**解决方案**:

```tsx
// 1. 使用绝对URL
export const metadata: Metadata = {
  openGraph: {
    images: ["https://example.com/og-image.jpg"], // 绝对URL
  },
};

// 2. 添加版本号
export const metadata: Metadata = {
  openGraph: {
    images: [`https://example.com/og-image.jpg?v=${Date.now()}`],
  },
};

// 3. 使用Facebook调试工具刷新缓存
// https://developers.facebook.com/tools/debug/
```

### 10.2 如何为不同页面设置不同的分享图片?

### 10.3 Twitter Card 不显示?

**问题**: 设置了 Twitter 元数据,但分享时不显示卡片。

**原因**:

### 10.7 如何支持多语言分享?

**问题**: 不同语言版本需要不同的分享内容。

**解决方案**:

```tsx
export async function generateMetadata({
  params,
}: {
  params: { lang: string };
}): Promise<Metadata> {
  const translations = {
    zh: {
      title: "中文标题",
      description: "中文描述",
      image: "https://example.com/og-zh.jpg",
    },
    en: {
      title: "English Title",
      description: "English Description",
      image: "https://example.com/og-en.jpg",
    },
  };

  const t = translations[params.lang as keyof typeof translations];

  return {
    title: t.title,
    description: t.description,
    openGraph: {
      title: t.title,
      description: t.description,
      images: [t.image],
      locale: params.lang === "zh" ? "zh_CN" : "en_US",
    },
  };
}
```
