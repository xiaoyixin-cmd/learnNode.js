# SEO 优化最佳实践

## 1. 概述

SEO(搜索引擎优化)是提升网站在搜索引擎中排名的关键。Next.js 16 提供了强大的 SEO 优化工具和 API,帮助开发者构建对搜索引擎友好的网站。

### 1.1 核心要素

- **元数据优化**: 标题、描述、关键词
- **结构化数据**: Schema.org 标记
- **性能优化**: Core Web Vitals
- **内容质量**: 原创、有价值的内容
- **技术 SEO**: 站点地图、robots.txt、规范 URL

### 1.2 Next.js 16 SEO 优势

| 特性       | 说明           | SEO 影响           |
| ---------- | -------------- | ------------------ |
| 服务端渲染 | HTML 直接返回  | 搜索引擎可直接抓取 |
| 静态生成   | 预渲染页面     | 加载速度快         |
| 图像优化   | 自动优化图片   | 提升页面性能       |
| 元数据 API | 动态生成元数据 | 提升搜索排名       |
| 站点地图   | 自动生成       | 帮助搜索引擎索引   |

## 3. 结构化数据

### 3.1 文章结构化数据

```tsx
// app/blog/[slug]/page.tsx
export default async function BlogPost({
  params,
}: {
  params: { slug: string };
}) {
  const post = await getPost(params.slug);

  const jsonLd = {
    "@context": "https://schema.org",
    "@type": "Article",
    headline: post.title,
    description: post.excerpt,
    image: post.coverImage,
    datePublished: post.publishedAt,
    dateModified: post.updatedAt,
    author: {
      "@type": "Person",
      name: post.author.name,
      url: post.author.url,
    },
    publisher: {
      "@type": "Organization",
      name: "网站名称",
      logo: {
        "@type": "ImageObject",
        url: "https://example.com/logo.png",
      },
    },
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
      />
      <article>
        <h1>{post.title}</h1>
        <div dangerouslySetInnerHTML={{ __html: post.content }} />
      </article>
    </>
  );
}
```

```tsx
// app/sitemap.ts
import { MetadataRoute } from "next";

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const posts = await getAllPosts();
  const products = await getAllProducts();

  const postUrls = posts.map((post) => ({
    url: `https://example.com/blog/${post.slug}`,
    lastModified: post.updatedAt,
    changeFrequency: "weekly" as const,
    priority: 0.8,
  }));

  const productUrls = products.map((product) => ({
    url: `https://example.com/product/${product.id}`,
    lastModified: product.updatedAt,
    changeFrequency: "daily" as const,
    priority: 0.9,
  }));

  return [
    {
      url: "https://example.com",
      lastModified: new Date(),
      changeFrequency: "daily",
      priority: 1,
    },
    ...postUrls,
    ...productUrls,
  ];
}
```

```tsx
export async function generateMetadata({
```

## 9. 注意事项

### 9.1 避免过度优化

不要堆砌关键词,保持内容自然:

```tsx
// ❌ 错误 - 关键词堆砌
export const metadata: Metadata = {
  title: "Next.js Next.js Next.js 教程 Next.js 学习",
  description: "Next.js Next.js Next.js 最好的 Next.js 教程",
};

// ✅ 正确 - 自然的标题和描述
export const metadata: Metadata = {
  title: "Next.js 16 完整教程 - 从入门到精通",
  description: "学习Next.js 16的核心特性,包括App Router、Server Components等",
};
```

### 9.2 保持内容原创

搜索引擎会惩罚重复内容:

```tsx
// 为重复内容设置规范URL
export const metadata: Metadata = {
  alternates: {
    canonical: "https://example.com/original-page",
  },
};
```

### 9.3 移动端优先

确保网站在移动设备上表现良好:

```tsx
export const viewport: Viewport = {
  width: "device-width",
  initialScale: 1,
  maximumScale: 5,
};
```

### 9.4 页面加载速度

优化页面加载速度是 SEO 的重要因素:

```tsx
// 使用动态导入减少初始加载
const HeavyComponent = dynamic(() => import("@/components/Heavy"), {
  loading: () => <Skeleton />,
```

### 9.5 HTTPS

确保网站使用 HTTPS:

```tsx
// next.config.js
```

### 9.6 定期更新内容

搜索引擎喜欢新鲜的内容:

```tsx
// 完整的SEO配置示例
// app/blog/[slug]/page.tsx
import { Metadata } from "next";

export async function generateMetadata({
  params,
}: {
  params: { slug: string };
}): Promise<Metadata> {
  const post = await getPost(params.slug);

  return {
    title: post.title,
    description: post.excerpt,
    keywords: post.tags,
    authors: [{ name: post.author.name }],
    openGraph: {
      type: "article",
      title: post.title,
      description: post.excerpt,
      images: [post.coverImage],
      publishedTime: post.publishedAt,
      modifiedTime: post.updatedAt,
    },
    twitter: {
      card: "summary_large_image",
      title: post.title,
      description: post.excerpt,
      images: [post.coverImage],
    },
    alternates: {
      canonical: `https://example.com/blog/${params.slug}`,
    },
  };
}

export default async function BlogPost({
  params,
}: {
  params: { slug: string };
}) {
  const post = await getPost(params.slug);

  const jsonLd = {
    "@context": "https://schema.org",
    "@type": "Article",
    headline: post.title,
    description: post.excerpt,
    image: post.coverImage,
    datePublished: post.publishedAt,
    dateModified: post.updatedAt,
    author: {
      "@type": "Person",
      name: post.author.name,
    },
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
      />
      <article>
        <h1>{post.title}</h1>
        <div dangerouslySetInnerHTML={{ __html: post.content }} />
      </article>
    </>
  );
}
```
