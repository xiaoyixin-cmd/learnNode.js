## 2. 渲染模式完整指南

### 2.1 所有渲染模式对比

| 渲染模式 | TTFB | SEO  | 服务器负载 | 数据新鲜度 | 适用场景   |
| -------- | ---- | ---- | ---------- | ---------- | ---------- |
| CSR      | 快   | 差   | 无         | 实时       | 后台系统   |
| SSR      | 慢   | 优秀 | 高         | 实时       | 动态内容   |
| SSG      | 极快 | 优秀 | 无         | 静态       | 博客/文档  |
| ISR      | 极快 | 优秀 | 低         | 定期更新   | 新闻/电商  |
| PPR      | 极快 | 优秀 | 低         | 混合       | 大多数应用 |

### 2.2 渲染模式选择流程

```
决策树:

1. 内容更新频率?
   ├─ 很少变化 → SSG
   ├─ 定期变化 → ISR
   ├─ 实时变化 → SSR
   └─ 混合场景 → PPR

2. SEO 重要性?
   ├─ 非常重要 → SSG/ISR/PPR
   ├─ 一般重要 → SSR
   └─ 不重要 → CSR

3. 性能要求?
   ├─ 极致性能 → SSG
   ├─ 高性能 → ISR/PPR
   └─ 一般性能 → SSR

4. 用户个性化?
   ├─ 无个性化 → SSG
   ├─ 部分个性化 → PPR
   └─ 完全个性化 → SSR/CSR

推荐: 大多数应用使用 PPR
```

## 3. 数据获取最佳实践

### 3.1 并行 vs 串行

```tsx
// ❌ 串行: 总耗时 1500ms
export default async function Page() {
  const user = await fetchUser(); // 500ms
  const posts = await fetchPosts(); // 600ms
  const comments = await fetchComments(); // 400ms
}

// ✅ 并行: 总耗时 600ms
export default async function Page() {
  const [user, posts, comments] = await Promise.all([
    fetchUser(),
    fetchPosts(),
    fetchComments(),
  ]);
}

// ✅ 预加载: 最优性能
import { cache } from "react";

export const getUser = cache(async (id: string) => {
  return db.user.findUnique({ where: { id } });
});

export const preloadUser = (id: string) => {
  void getUser(id);
};

export default async function Page({ params }) {
  const { id } = await params;

  // 立即开始预加载
  preloadUser(id);
  preloadPosts(id);

  // 等待结果
  const user = await getUser(id);

  return <UserProfile user={user} />;
}
```

## 4. Server Components vs Client Components

### 4.1 决策流程

```
需要以下功能吗?

├─ useState/useEffect 等 Hooks? → Client Component
├─ onClick/onChange 等事件? → Client Component
├─ useRouter/useSearchParams? → Client Component
├─ 浏览器 API (window/localStorage)? → Client Component
├─ 第三方客户端库? → Client Component
└─ 以上都不需要 → Server Component (默认)
```

### 4.2 组合模式

```tsx
// ✅ 好: Server Component 包含 Client Component
export default async function Page() {
  const data = await fetchData();

  return (
    <div>
      <ServerContent data={data} />
      <ClientInteractive data={data} />
    </div>
  );
}

// ✅ 好: Client Component 通过 children 包含 Server Component
("use client");

export default function ClientWrapper({ children }) {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <button onClick={() => setIsOpen(!isOpen)}>Toggle</button>
      {isOpen && <div>{children}</div>}
    </div>
  );
}

// 使用
<ClientWrapper>
  <ServerContent />
</ClientWrapper>;

// ❌ 错误: Client Component 直接导入 Server Component
("use client");

import ServerComponent from "./ServerComponent";

export default function ClientComponent() {
  return <ServerComponent />;
}
```

### 4.3 提升客户端边界

```tsx
// ❌ 不好: 整个页面都是 Client Component
```
