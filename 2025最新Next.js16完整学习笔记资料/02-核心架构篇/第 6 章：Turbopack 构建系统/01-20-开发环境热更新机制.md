# 开发环境热更新机制

## 概述

热更新(Hot Module Replacement, HMR)是现代前端开发中不可或缺的功能,它允许开发者在不刷新整个页面的情况下替换、添加或删除模块。Turbopack 结合 Next.js 的 Fast Refresh 机制,将热更新性能提升到了新的高度。在 Next.js 16 中,Turbopack 的 HMR 响应速度可以达到 50-200ms,相比 Webpack 的 500-2000ms 提升了 10 倍以上。理解热更新机制的工作原理,对于优化开发体验和调试问题至关重要。

### 什么是热更新

热更新允许在运行时更新应用程序模块,而无需完全刷新页面。它保留应用程序状态,只更新变化的模块,极大提升了开发效率。

### Fast Refresh 特性

Fast Refresh 是 React 官方的热重载解决方案,相比传统 HMR:

- **可靠性更高**:自动处理组件错误恢复
- **状态保留**:尽可能保留组件状态
- **错误处理**:优雅的错误边界和恢复
- **更快速度**:仅重新渲染受影响的组件

### Turbopack 的优势

1. **增量更新**:只处理变化的模块
2. **并行编译**:利用多核 CPU
3. **智能缓存**:避免重复计算
4. **快速传输**:优化的 WebSocket 通信
5. **精确定位**:细粒度的依赖追踪

// HMR 系统的核心组件

class HMRSystem {
  // 文件监听器
  private fileWatcher: FileWatcher;

  // 增量编译器
  private compiler: IncrementalCompiler;

  // 依赖图
  private dependencyGraph: DependencyGraph;

  // WebSocket 服务器
  private wsServer: WebSocketServer;

  // 客户端连接
  private clients = new Set<WebSocket>();

  // HMR 更新队列
  private updateQueue: UpdateQueue;

  constructor() {
    this.fileWatcher = new FileWatcher({
      watchDir: process.cwd(),
      ignored: ["node_modules", ".next"],
    });

    this.compiler = new IncrementalCompiler();
    this.dependencyGraph = new DependencyGraph();
    this.wsServer = new WebSocketServer({ port: 3001 });
    this.updateQueue = new UpdateQueue();

    this.initialize();
  }

  private initialize() {
    // 1. 启动文件监听
    this.fileWatcher.on("change", (path) => {
      this.handleFileChange(path);
    });

    // 2. 启动 WebSocket 服务器
    this.wsServer.on("connection", (client) => {
      this.handleClientConnection(client);
    });

    // 3. 处理更新队列
    this.updateQueue.on("batch", (updates) => {
      this.processUpdateBatch(updates);
    });
  }

  // 处理文件变化
  private async handleFileChange(path: string) {
    console.log(`文件变化: ${path}`);

    // 1. 增量编译
    const compiledModule = await this.compiler.compile(path);

    // 2. 查找受影响的模块
    const affectedModules = this.dependencyGraph.findAffectedModules(path);

    // 3. 创建 HMR 更新
    const update: HMRUpdate = {
      type: "update",
      path,
      module: compiledModule,
      affectedModules,
      timestamp: Date.now(),
    };

    // 4. 加入更新队列
    this.updateQueue.add(update);
  }

  // 处理客户端连接
  private handleClientConnection(client: WebSocket) {
    console.log("客户端连接");

    this.clients.add(client);

    // 发送初始化信息
    client.send(
      JSON.stringify({
        type: "connected",
        message: "HMR 已连接",
      })
    );

    client.on("close", () => {
      this.clients.delete(client);
    });
  }

  // 处理更新批次
  private async processUpdateBatch(updates: HMRUpdate[]) {
    console.log(`处理 ${updates.length} 个更新`);

    // 1. 合并更新
    const mergedUpdate = this.mergeUpdates(updates);

    // 2. 广播给所有客户端
    this.broadcast(mergedUpdate);
  }

  // 合并多个更新
  private mergeUpdates(updates: HMRUpdate[]): HMRUpdate {
    // 合并同一文件的多个更新
    const moduleMap = new Map<string, HMRUpdate>();

    updates.forEach((update) => {
      const existing = moduleMap.get(update.path);

      if (!existing || update.timestamp > existing.timestamp) {
        moduleMap.set(update.path, update);
      }
    });

    return {
      type: "batch",
      updates: Array.from(moduleMap.values()),
      timestamp: Date.now(),
    };
  }

  // 广播更新
  private broadcast(update: HMRUpdate) {
    const message = JSON.stringify(update);

    this.clients.forEach((client) => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(message);
      }
    });

    console.log(`已广播更新给 ${this.clients.size} 个客户端`);
  }
}

class FileWatcher {
  private watcher: FSWatcher;
  private eventEmitter = new EventEmitter();
  private debounceMap = new Map<string, NodeJS.Timeout>();

  constructor(private options: WatchOptions) {
    this.watcher = chokidar.watch(options.watchDir, {
      ignored: options.ignored,
      persistent: true,
      ignoreInitial: true,
      // 性能优化选项
      awaitWriteFinish: {
        stabilityThreshold: 100,
        pollInterval: 50,
      },
      // 使用原生文件监听
      usePolling: false,
      // 聚合事件
      atomic: true,
    });

    this.setupListeners();
  }

  private setupListeners() {
    this.watcher.on("change", (path) => {
      this.handleChange(path);
    });

    this.watcher.on("add", (path) => {
      this.handleChange(path);
    });

    this.watcher.on("unlink", (path) => {
      this.handleDelete(path);
    });
  }

  // 处理文件变化(带防抖)
  private handleChange(path: string) {
    // 清除之前的定时器
    const existingTimer = this.debounceMap.get(path);
    if (existingTimer) {
      clearTimeout(existingTimer);
    }

    // 设置新的定时器
    const timer = setTimeout(() => {
      this.debounceMap.delete(path);
      this.eventEmitter.emit("change", path);
    }, 50); // 50ms 防抖

    this.debounceMap.set(path, timer);
  }

  private handleDelete(path: string) {
    this.eventEmitter.emit("delete", path);
  }

  on(event: string, listener: (path: string) => void) {
    this.eventEmitter.on(event, listener);
  }

  close() {
    this.watcher.close();
    this.debounceMap.forEach((timer) => clearTimeout(timer));
  }
}

class IncrementalCompiler {
  private cache = new Map<string, CompiledModule>();
  private turboEngine: TurboEngine;

  constructor() {
    this.turboEngine = new TurboEngine();
  }

  // 编译模块
  async compile(path: string): Promise<CompiledModule> {
    console.log(`编译: ${path}`);

    // 1. 检查缓存
    const cached = this.checkCache(path);
    if (cached) {
      console.log(`使用缓存: ${path}`);
      return cached;
    }

    // 2. 读取源代码
    const source = await fs.readFile(path, "utf-8");

    // 3. 计算哈希
    const hash = computeHash(source);

    // 4. 使用 Turbo Engine 编译
    const compiled = await this.turboEngine.compile({
      path,
      source,
      hash,
    });

    // 5. 缓存结果
    this.cache.set(path, compiled);

    console.log(`编译完成: ${path} (${compiled.code.length} bytes)`);

    return compiled;
  }

  private checkCache(path: string): CompiledModule | null {
    const cached = this.cache.get(path);

    if (!cached) {
      return null;
    }

    // 检查文件是否变化
    const currentHash = this.getFileHash(path);
    if (currentHash !== cached.hash) {
      // 缓存失效
      this.cache.delete(path);
      return null;
    }

    return cached;
  }

  private getFileHash(path: string): string {
    try {
      const source = fs.readFileSync(path, "utf-8");
      return computeHash(source);
    } catch (err) {
      return "";
    }
  }

  // 清除缓存
  clearCache(path?: string) {
    if (path) {
      this.cache.delete(path);
    } else {
      this.cache.clear();
    }
  }
}

// 细粒度依赖追踪

class DependencyGraph {
  private graph = new Map<string, Set<string>>();
  private reverseGraph = new Map<string, Set<string>>();

  // 添加依赖
  addDependency(from: string, to: string) {
    // 正向依赖
    if (!this.graph.has(from)) {
      this.graph.set(from, new Set());
    }
    this.graph.get(from)!.add(to);

    // 反向依赖
    if (!this.reverseGraph.has(to)) {
      this.reverseGraph.set(to, new Set());
    }
    this.reverseGraph.get(to)!.add(from);
  }

  // 查找受影响的模块
  findAffectedModules(path: string): string[] {
    const affected = new Set<string>();
    const visited = new Set<string>();

    const traverse = (currentPath: string) => {
      if (visited.has(currentPath)) {
        return;
      }

      visited.add(currentPath);
      affected.add(currentPath);

      // 遍历所有依赖此模块的模块
      const dependents = this.reverseGraph.get(currentPath);
      if (dependents) {
        dependents.forEach((dependent) => {
          traverse(dependent);
        });
      }
    };

    traverse(path);

    return Array.from(affected);
  }

  // 获取依赖
  getDependencies(path: string): string[] {
    return Array.from(this.graph.get(path) || []);
  }

  // 获取依赖者
  getDependents(path: string): string[] {
    return Array.from(this.reverseGraph.get(path) || []);
  }
}

// 更新队列:批量处理更新以提高效率

class UpdateQueue {
  private queue: HMRUpdate[] = [];
  private timer: NodeJS.Timeout | null = null;
  private eventEmitter = new EventEmitter();

  // 批处理延迟
  private batchDelay = 100; // 100ms

  // 添加更新
  add(update: HMRUpdate) {
    this.queue.push(update);

    // 重置定时器
    if (this.timer) {
      clearTimeout(this.timer);
    }

    // 设置批处理定时器
    this.timer = setTimeout(() => {
      this.flush();
    }, this.batchDelay);
  }

  // 刷新队列
  private flush() {
    if (this.queue.length === 0) {
      return;
    }

    const updates = [...this.queue];
    this.queue = [];

    console.log(`刷新更新队列: ${updates.length} 个更新`);

    this.eventEmitter.emit("batch", updates);
  }

  on(event: string, listener: (updates: HMRUpdate[]) => void) {
    this.eventEmitter.on(event, listener);
  }
}

// 浏览器端 HMR 客户端

class HMRClient {
  private ws: WebSocket;
  private moduleCache = new Map<string, any>();
  private updateHandlers = new Map<string, UpdateHandler>();

  constructor() {
    this.connect();
  }

  // 连接 WebSocket
  private connect() {
    const wsUrl = `ws://${window.location.hostname}:3001`;

    this.ws = new WebSocket(wsUrl);

    this.ws.onopen = () => {
      console.log("[HMR] 已连接");
    };

    this.ws.onmessage = (event) => {
      this.handleMessage(event.data);
    };

    this.ws.onclose = () => {
      console.log("[HMR] 连接关闭,尝试重连...");
      setTimeout(() => this.connect(), 1000);
    };

    this.ws.onerror = (error) => {
      console.error("[HMR] 错误:", error);
    };
  }

  // 处理服务器消息
  private handleMessage(data: string) {
    const message = JSON.parse(data);

    switch (message.type) {
      case "connected":
        console.log("[HMR]", message.message);
        break;

      case "update":
        this.applyUpdate(message);
        break;

      case "batch":
        this.applyBatchUpdate(message.updates);
        break;

      case "error":
        this.handleError(message);
        break;
    }
  }

  // 应用单个更新
  private async applyUpdate(update: HMRUpdate) {
    console.log(`[HMR] 应用更新: ${update.path}`);

    try {
      // 1. 下载新模块
      const newModule = await this.fetchModule(update.path);

      // 2. 执行更新
      await this.executeUpdate(update.path, newModule);

      // 3. 更新依赖者
      for (const affected of update.affectedModules) {
        await this.updateModule(affected);
      }

      console.log("[HMR] 更新成功");
    } catch (error) {
      console.error("[HMR] 更新失败:", error);
      this.handleUpdateError(error);
    }
  }

  // 应用批量更新
  private async applyBatchUpdate(updates: HMRUpdate[]) {
    console.log(`[HMR] 应用批量更新: ${updates.length} 个`);

    for (const update of updates) {
      await this.applyUpdate(update);
    }
  }

  // 获取新模块
  private async fetchModule(path: string): Promise<any> {
    const response = await fetch(`/_next/hmr/${path}?t=${Date.now()}`);

    if (!response.ok) {
      throw new Error(`获取模块失败: ${path}`);
    }

    const code = await response.text();

    // 执行模块代码
    return this.evaluateModule(code, path);
  }

  // 执行模块代码
  private evaluateModule(code: string, path: string): any {
    // 创建模块作用域
    const module = { exports: {} };
    const exports = module.exports;

    // 包装代码
    const wrappedCode = `
      (function(module, exports, require) {
        ${code}
      })
    `;

    // 执行
    const fn = eval(wrappedCode);
    fn(module, exports, this.require.bind(this));

    return module.exports;
  }

  // 模块加载器
  private require(path: string): any {
    return this.moduleCache.get(path);
  }

  // 执行更新
  private async executeUpdate(path: string, newModule: any) {
    // 获取旧模块
    const oldModule = this.moduleCache.get(path);

    // 更新缓存
    this.moduleCache.set(path, newModule);

    // 调用更新处理器
    const handler = this.updateHandlers.get(path);
    if (handler) {
      await handler(newModule, oldModule);
    }
  }

  // 更新模块
  private async updateModule(path: string) {
    const newModule = await this.fetchModule(path);
    await this.executeUpdate(path, newModule);
  }

  // 注册更新处理器
  registerUpdateHandler(path: string, handler: UpdateHandler) {
    this.updateHandlers.set(path, handler);
  }

  // 处理错误
  private handleError(message: any) {
    console.error("[HMR] 服务器错误:", message);
    this.showErrorOverlay(message);
  }

  private handleUpdateError(error: any) {
    console.error("[HMR] 更新错误:", error);

    // 显示错误覆层
    this.showErrorOverlay({
      message: error.message,
      stack: error.stack,
    });
  }

  // 显示错误覆层
  private showErrorOverlay(error: any) {
    const overlay = document.createElement("div");
    overlay.id = "hmr-error-overlay";
    overlay.innerHTML = `
      <div style="
        position: fixed;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
        background: rgba(0, 0, 0, 0.9);
        color: white;
        padding: 20px;
        font-family: monospace;
        z-index: 999999;
        overflow: auto;
      ">
        <h2>HMR 错误</h2>
        <pre>${error.message}</pre>
        <pre>${error.stack || ""}</pre>
        <button onclick="this.parentElement.remove()">关闭</button>
      </div>
    `;

    document.body.appendChild(overlay);
  }
}

class ReactFastRefresh {
  private runtime: any;
  private pendingUpdates = new Set<string>();

  constructor() {
    // 初始化 React Refresh 运行时
    if (typeof window !== "undefined") {
      this.runtime = require("react-refresh/runtime");
      this.runtime.injectIntoGlobalHook(window);
    }
  }

  // 注册组件更新
  registerComponent(Component: any, id: string) {
    if (!this.runtime) return;

    // 注册到 React Refresh
    this.runtime.register(Component, id);
  }

  // 执行更新
  performUpdate() {
    if (!this.runtime) return;

    console.log("[Fast Refresh] 执行更新");

    try {
      // 执行 React Refresh 更新
      this.runtime.performReactRefresh();

      console.log("[Fast Refresh] 更新成功");

      // 清除待处理更新
      this.pendingUpdates.clear();
    } catch (error) {
      console.error("[Fast Refresh] 更新失败:", error);

      // 回退到完整刷新
      window.location.reload();
    }
  }

  // 标记需要更新的组件
  markForUpdate(id: string) {
    this.pendingUpdates.add(id);

    // 延迟执行更新
    setTimeout(() => {
      if (this.pendingUpdates.size > 0) {
        this.performUpdate();
      }
    }, 0);
  }

  // 检查是否可以保留状态
  canPreserveState(Component: any): boolean {
    if (!this.runtime) return false;

    // React Refresh 运行时判断
    return this.runtime.isLikelyComponentType(Component);
  }
}

  // 判断更新类型
  determineUpdateType(
    oldComponent: any,
    newComponent: any
  ): "fast-refresh" | "full-refresh" {
    // 1. 检查是否为 React 组件
    if (!this.isReactComponent(newComponent)) {
      return "full-refresh";

    // 2. 检查组件签名是否变化
    if (this.hasSignatureChanged(oldComponent, newComponent)) {
      return "full-refresh";

    // 3. 检查 Hooks 顺序是否变化
    if (this.hasHooksOrderChanged(oldComponent, newComponent)) {
      return "full-refresh";
    }

    // 4. 可以使用 Fast Refresh
    return "fast-refresh";
  }

  private isReactComponent(Component: any): boolean {
    // 函数组件
    if (typeof Component === "function") {
      return true;
    }

    // 类组件
    if (Component.prototype && Component.prototype.isReactComponent) {
      return true;
    }

    return false;
  }

  private hasSignatureChanged(oldComp: any, newComp: any): boolean {
    // 比较函数签名

    const newSignature = this.getComponentSignature(newComp);

class StatePreservation {
  private stateMap = new WeakMap<any, any>();

  // 保存组件状态
  saveState(Component: any, state: any) {
    this.stateMap.set(Component, state);
  }

  // 恢复组件状态
  restoreState(Component: any): any {
    return this.stateMap.get(Component);
  }

  // 尝试保留状态更新组件
  updateWithStatePreservation(
    oldComponent: any,
    newComponent: any,
    currentState: any
  ): any {
    // 1. 保存当前状态
    this.saveState(oldComponent, currentState);

    // 2. 检查是否可以保留状态
    if (!this.canPreserveState(oldComponent, newComponent)) {
      console.log("[HMR] 无法保留状态,重置组件");
      return null;
    }

    // 3. 恢复状态到新组件
    const preservedState = this.restoreState(oldComponent);

    console.log("[HMR] 状态已保留");

    return preservedState;
  }

  private canPreserveState(oldComp: any, newComp: any): boolean {
    // 检查组件类型是否兼容
    if (typeof oldComp !== typeof newComp) {
      return false;
    }

    // 类组件
    if (oldComp.prototype && oldComp.prototype.isReactComponent) {
      return this.canPreserveClassComponentState(oldComp, newComp);
    }

    // 函数组件
    return this.canPreserveFunctionComponentState(oldComp, newComp);
  }

  private canPreserveClassComponentState(oldComp: any, newComp: any): boolean {
    // 类组件通常可以保留状态
    return true;
  }

  private canPreserveFunctionComponentState(
    oldComp: any,
    newComp: any
  ): boolean {
    // 函数组件需要检查 Hooks
    // 如果 Hooks 顺序变化,不能保留状态
    return !this.hasHooksChanged(oldComp, newComp);
  }

  private hasHooksChanged(oldComp: any, newComp: any): boolean {

// 并行编译多个模块

class ParallelCompiler {
  private maxConcurrency = 4; // 最大并发数
  private queue: CompileTask[] = [];
  private running = 0;

  // 编译多个模块
  async compileModules(paths: string[]): Promise<CompiledModule[]> {
    console.log(`并行编译 ${paths.length} 个模块`);

    const results: Promise<CompiledModule>[] = [];

    for (const path of paths) {
      results.push(this.scheduleCompile(path));
    }

    return Promise.all(results);
  }

  // 调度编译任务
  private scheduleCompile(path: string): Promise<CompiledModule> {
    return new Promise((resolve, reject) => {
      const task: CompileTask = {
        path,

class MultiLevelCache {
  // 内存缓存
  private memoryCache = new Map<string, CachedModule>();

  // 文件系统缓存
  private diskCachePath = ".next/cache";

  // 获取缓存
  async get(key: string): Promise<CachedModule | null> {
    // 1. 检查内存缓存
    const memCached = this.memoryCache.get(key);
    if (memCached && this.isValid(memCached)) {
      console.log(`[Cache] 内存命中: ${key}`);
      return memCached;
    }

    // 2. 检查磁盘缓存
    const diskCached = await this.getDiskCache(key);
    if (diskCached && this.isValid(diskCached)) {
      console.log(`[Cache] 磁盘命中: ${key}`);

      // 写入内存缓存
      this.memoryCache.set(key, diskCached);

      return diskCached;
    }

    console.log(`[Cache] 未命中: ${key}`);
    return null;
  }

  // 设置缓存
  async set(key: string, value: CachedModule) {
    // 1. 写入内存缓存
    this.memoryCache.set(key, value);

    // 2. 异步写入磁盘缓存
    this.setDiskCache(key, value).catch((err) => {
      console.error("[Cache] 磁盘写入失败:", err);
    });
  }

  private async getDiskCache(key: string): Promise<CachedModule | null> {
    try {
      const cachePath = path.join(this.diskCachePath, `${key}.json`);
      const data = await fs.readFile(cachePath, "utf-8");
      return JSON.parse(data);
    } catch (err) {
      return null;
    }
  }

  private async setDiskCache(key: string, value: CachedModule) {
    const cachePath = path.join(this.diskCachePath, `${key}.json`);
    await fs.mkdir(this.diskCachePath, { recursive: true });
    await fs.writeFile(cachePath, JSON.stringify(value));
  }

  private isValid(cached: CachedModule): boolean {
    // 检查是否过期
    const age = Date.now() - cached.timestamp;
    const maxAge = 60 * 60 * 1000; // 1小时

    if (age > maxAge) {
      return false;
    }

    // 检查源文件是否变化
    return cached.hash === this.getFileHash(cached.path);
  }

  private getFileHash(path: string): string {
    try {
      const content = fs.readFileSync(path, "utf-8");
      return computeHash(content);
    } catch (err) {
      return "";
    }
  }

class IncrementalTransport {
  private sentChunks = new Map<string, Set<number>>();

  // 发送更新
  async sendUpdate(client: WebSocket, update: HMRUpdate) {
    // 1. 分块
    const chunks = this.splitIntoChunks(update.module.code);

    // 2. 找出需要发送的块
    const toSend = this.findChunksToSend(update.path, chunks);

    // 3. 发送
    for (const chunk of toSend) {
      client.send(
        JSON.stringify({
          type: "chunk",
          path: update.path,
          chunkId: chunk.id,
          data: chunk.data,
        })
      );

      // 记录已发送
      this.markChunkSent(update.path, chunk.id);
    }

    console.log(`发送 ${toSend.length}/${chunks.length} 个块`);
  }

  private splitIntoChunks(code: string): Chunk[] {
    const chunkSize = 1024; // 1KB
    const chunks: Chunk[] = [];

    for (let i = 0; i < code.length; i += chunkSize) {
      chunks.push({
        id: chunks.length,
        data: code.slice(i, i + chunkSize),
        hash: computeHash(code.slice(i, i + chunkSize)),
      });
    }

    return chunks;
  }

  private findChunksToSend(path: string, chunks: Chunk[]): Chunk[] {
    const sent = this.sentChunks.get(path) || new Set();

    return chunks.filter((chunk) => {
      // 如果之前没发送过,或者内容变化了
      return !sent.has(chunk.id);
    });
  }

  private markChunkSent(path: string, chunkId: number) {
    if (!this.sentChunks.has(path)) {
      this.sentChunks.set(path, new Set());
    }

    this.sentChunks.get(path)!.add(chunkId);
  }
}

class CompressionOptimizer {
  // 压缩更新
  compress(update: HMRUpdate): CompressedUpdate {
    // 1. 移除冗余信息
    const stripped = this.stripRedundantInfo(update);

    // 2. 压缩代码
    const compressed = this.compressCode(stripped);

    return compressed;
  }

  private stripRedundantInfo(update: HMRUpdate): any {
    return {
      path: update.path,
      // 只发送代码,不发送 AST 等中间产物
      code: update.module.code,
      // 只发送必要的受影响模块
      affected: update.affectedModules.slice(0, 10),
    };
  }

  private compressCode(data: any): CompressedUpdate {
    // 使用 gzip 压缩
    const json = JSON.stringify(data);
    const compressed = zlib.gzipSync(json);

    return {
      data: compressed,
      encoding: "gzip",
    };
  }

## 常见问题

### Q1: HMR 更新失败怎么办?

**A**: 失败原因和解决方案:

1. **语法错误**: 检查代码语法,修复错误
2. **循环依赖**: 重构代码消除循环依赖
3. **状态不兼容**: 检查 Hooks 顺序,保持一致
4. **网络问题**: 检查 WebSocket 连接

### Q2: 为什么有时会完全刷新?

**A**: 以下情况会触发完全刷新:

1. 组件签名变化(参数、Hooks)
2. 非组件文件更新(配置文件等)
3. 依赖关系大幅变化
4. HMR 运行时错误

### Q3: 如何保留组件状态?

**A**: Fast Refresh 自动处理:

1. **函数组件**: 保持 Hooks 顺序不变
2. **类组件**: 通常自动保留状态
3. **全局状态**: 使用状态管理库(Redux, Zustand)
4. **URL 状态**: 保存在 URL 参数中

### Q4: HMR 性能如何优化?

**A**: 优化策略:

1. **减少监听文件**: 排除不必要的目录
2. **增量编译**: 只编译变化的模块
3. **并行处理**: 利用多核 CPU
4. **缓存**: 多层缓存机制
