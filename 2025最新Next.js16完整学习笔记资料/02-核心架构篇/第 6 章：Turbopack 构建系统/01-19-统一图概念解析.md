# 统一图概念解析

## 概述

统一图(Unified Graph)是 Turbopack 架构的核心创新之一,它彻底改变了传统构建工具处理多环境输出的方式。不同于 Webpack 为每个目标环境创建独立编译器的方式,Turbopack 使用单一的依赖图来管理所有环境的模块和依赖关系。这种设计不仅提高了构建效率,还确保了不同环境之间的一致性。理解统一图的概念和工作原理,对于充分利用 Turbopack 的性能优势至关重要。

### 什么是统一图

统一图是一个包含所有模块、依赖关系和环境信息的单一数据结构。它将客户端、服务端和边缘运行时的代码组织在同一个图中,通过标记和过滤机制为不同环境生成对应的输出。

### 传统方式的问题

在 Webpack 中,为每个环境创建独立编译器会导致:

- **重复处理**:共享模块被多次编译
- **内存浪费**:多个编译器实例占用大量内存
- **不一致性**:不同环境可能使用不同版本的模块
- **复杂度高**:需要协调多个编译器的执行

### 核心价值

1. **性能提升**:共享模块只处理一次,避免重复计算
2. **内存优化**:单一图结构减少内存占用
3. **一致性保证**:所有环境使用相同的模块版本
4. **架构简化**:单一遍历生成所有输出
5. **易于维护**:更简单的代码结构和逻辑

**图结构定义**

```typescript
// Turbopack 统一图结构

class UnifiedDependencyGraph {
  // 所有模块节点
  private nodes = new Map<string, ModuleNode>();

  // 依赖关系边
  private edges = new Map<string, Set<string>>();

  // 入口点
  private entryPoints = new Map<string, EntryPoint>();

  // 环境配置
  private environments = {
    client: new EnvironmentConfig("client"),
    server: new EnvironmentConfig("server"),
    edge: new EnvironmentConfig("edge"),
  };

  // 添加节点
  addNode(node: ModuleNode) {
    this.nodes.set(node.id, node);
  }

  // 添加边(依赖关系)
  addEdge(from: string, to: string) {
    if (!this.edges.has(from)) {
      this.edges.set(from, new Set());
    }
    this.edges.get(from)!.add(to);

    // 双向关联
    const fromNode = this.nodes.get(from);
    const toNode = this.nodes.get(to);

    if (fromNode && toNode) {
      fromNode.dependencies.add(to);
      toNode.dependents.add(from);
    }
  }

  // 获取节点
  getNode(id: string): ModuleNode | undefined {
    return this.nodes.get(id);
  }

  // 获取所有节点
  getAllNodes(): ModuleNode[] {
    return Array.from(this.nodes.values());
  }

  // 获取依赖
  getDependencies(id: string): Set<string> {
    return this.edges.get(id) || new Set();
  }
}
```

```typescript
class EnvironmentConfig {
  constructor(
    public name: "client" | "server" | "edge",
    public target: "web" | "node" | "edge" = name === "client"
      ? "web"
      : name === "edge"
      ? "edge"
      : "node",
    public runtime: Runtime = defaultRuntimes[name]
  ) {}

  // 是否应该包含此模块
  shouldIncludeModule(module: ModuleNode): boolean {
    return module.environments.has(this.name);
  }

  // 转换模块(针对特定环境)
  transformModule(module: ModuleNode): TransformedModule {
    // 根据环境应用不同的转换
    switch (this.name) {
      case "client":
        return this.transformForClient(module);
      case "server":
        return this.transformForServer(module);
      case "edge":
        return this.transformForEdge(module);
    }
  }

  private transformForClient(module: ModuleNode): TransformedModule {
    return {
      ...module,
      // 客户端特定转换
      // 例如: 移除 server-only 代码
      transformedSource: removeServerOnlyCode(module.source),
    };
  }

  private transformForServer(module: ModuleNode): TransformedModule {
    return {
      ...module,
      // 服务端特定转换
      // 例如: 保留所有代码
      transformedSource: module.source,
    };
  }

  private transformForEdge(module: ModuleNode): TransformedModule {
    return {
      ...module,
      // Edge 运行时特定转换
      // 例如: 移除 Node.js 特定 API
      transformedSource: removeNodeAPIs(module.source),
    };
  }
}
```

```typescript
    console.log(`处理入口点: ${entry.path}`);

    // 创建入口节点
    const entryNode = await this.createNode(entry.path);
    entryNode.environments.add(entry.environment);

    // 递归处理依赖
    await this.processDependencies(entryNode);

  // 处理模块依赖
  private async processDependencies(node: ModuleNode) {
    if (this.visited.has(node.id)) {
      return;
    }

    this.visited.add(node.id);

    // 1. 解析依赖
    const dependencies = await this.parseDependencies(node);

    // 2. 处理每个依赖
    for (const depPath of dependencies) {
      // 创建或获取依赖节点
      const depNode = await this.getOrCreateNode(depPath);

      // 添加边
      this.graph.addEdge(node.id, depNode.id);

      // 继承环境标记
      node.environments.forEach((env) => {
        depNode.environments.add(env);
      });

      // 递归处理
      await this.processDependencies(depNode);
    }
  }

  // 创建节点
  private async createNode(path: string): Promise<ModuleNode> {
    const source = await readFile(path);
    const hash = computeHash(source);

    const node: ModuleNode = {
      id: path,
      path,
      type: this.inferType(path),
      source,
      dependencies: new Set(),
      dependents: new Set(),
      environments: new Set(),
      metadata: {
        hash,
        size: source.length,
        lastModified: Date.now(),
      },
      cacheKey: hash,
      cached: false,
    };

    this.graph.addNode(node);
    return node;
  }

  // 解析模块依赖
  private async parseDependencies(node: ModuleNode): Promise<string[]> {
    const dependencies: string[] = [];

    // 根据模块类型解析依赖
    switch (node.type) {
      case "javascript":
      case "typescript":
        dependencies.push(...this.parseJSDependencies(node.source));
        break;
      case "css":
        dependencies.push(...this.parseCssDependencies(node.source));
        break;
    }

    return dependencies;
  }

  // 解析 JavaScript/TypeScript 依赖
  private parseJSDependencies(source: string): string[] {
    const dependencies: string[] = [];

    // 匹配 import 语句
    const importRegex = /import\s+.*?\s+from\s+['"]([^'"]+)['"]/g;
    let match;

    while ((match = importRegex.exec(source)) !== null) {
      dependencies.push(match[1]);
    }

    // 匹配 require 语句
    const requireRegex = /require\(['"]([^'"]+)['"]\)/g;
    while ((match = requireRegex.exec(source)) !== null) {
      dependencies.push(match[1]);
    }

    // 匹配 dynamic import
    const dynamicImportRegex = /import\(['"]([^'"]+)['"]\)/g;
    while ((match = dynamicImportRegex.exec(source)) !== null) {
      dependencies.push(match[1]);
    }

    return dependencies;
  }

  // 标记环境
  private markEnvironments() {
    // 'use client' 指令的模块及其依赖标记为 client
    // 'use server' 指令的模块标记为 server
    // Edge Runtime 配置的模块标记为 edge

    this.graph.getAllNodes().forEach((node) => {
      if (node.source.includes("use client")) {
        this.markAsClient(node);
      }
      if (node.source.includes("use server")) {
        this.markAsServer(node);
      }
    });
  }

  private markAsClient(node: ModuleNode) {
    node.environments.add("client");

    // 传播到依赖
    node.dependencies.forEach((depId) => {
      const dep = this.graph.getNode(depId);
      if (dep && !dep.environments.has("client")) {
        this.markAsClient(dep);
      }
    });
  }

  private markAsServer(node: ModuleNode) {
    node.environments.add("server");
  }

  // 优化图
```

### 1.3 图的遍历算法

**深度优先遍历**

```typescript
// DFS 遍历统一图

class GraphTraversal {
  // 深度优先遍历
  depthFirstSearch(
    graph: UnifiedDependencyGraph,
    startNode: string,
    visitor: (node: ModuleNode) => void
  ) {
    const visited = new Set<string>();

    const dfs = (nodeId: string) => {
      if (visited.has(nodeId)) return;

      visited.add(nodeId);
      const node = graph.getNode(nodeId);

      if (node) {
        // 访问当前节点
        visitor(node);

        // 递归访问依赖
        node.dependencies.forEach((depId) => {
          dfs(depId);
        });
      }
    };

    dfs(startNode);
  }

  // 广度优先遍历
  breadthFirstSearch(
    graph: UnifiedDependencyGraph,
    startNode: string,
    visitor: (node: ModuleNode) => void
  ) {
    const visited = new Set<string>();
    const queue = [startNode];

    while (queue.length > 0) {
      const nodeId = queue.shift()!;

      if (visited.has(nodeId)) continue;
      visited.add(nodeId);

      const node = graph.getNode(nodeId);
      if (node) {
        // 访问当前节点
        visitor(node);

        // 将依赖加入队列
        node.dependencies.forEach((depId) => {
          if (!visited.has(depId)) {
            queue.push(depId);
          }
        });
      }
    }
  }

  // 拓扑排序
  topologicalSort(graph: UnifiedDependencyGraph): string[] {
    const inDegree = new Map<string, number>();
    const result: string[] = [];

    // 计算入度
    graph.getAllNodes().forEach((node) => {
      inDegree.set(node.id, node.dependents.size);
    });

    // 找到所有入度为 0 的节点
    const queue: string[] = [];
    inDegree.forEach((degree, nodeId) => {
      if (degree === 0) {
        queue.push(nodeId);
      }
    });

    // 拓扑排序
    while (queue.length > 0) {
      const nodeId = queue.shift()!;
      result.push(nodeId);

      const node = graph.getNode(nodeId);
      if (node) {
        // 减少依赖节点的入度
        node.dependencies.forEach((depId) => {
          const degree = inDegree.get(depId)! - 1;
          inDegree.set(depId, degree);

          if (degree === 0) {
            queue.push(depId);
          }
        });
      }
    }

    // 检查是否有循环依赖
    if (result.length !== graph.getAllNodes().length) {
      throw new Error("检测到循环依赖");
    }

    return result;
  }

  // 查找循环依赖
  findCycles(graph: UnifiedDependencyGraph): string[][] {
    const cycles: string[][] = [];
    const visiting = new Set<string>();
    const visited = new Set<string>();

    const dfs = (nodeId: string, path: string[]) => {
      if (visiting.has(nodeId)) {
        // 找到循环
        const cycleStart = path.indexOf(nodeId);
        cycles.push(path.slice(cycleStart));
        return;
      }

      if (visited.has(nodeId)) return;

      visiting.add(nodeId);
      path.push(nodeId);

      const node = graph.getNode(nodeId);
      if (node) {
        node.dependencies.forEach((depId) => {
          dfs(depId, [...path]);
        });
      }

      visiting.delete(nodeId);
      visited.add(nodeId);
    };

    graph.getAllNodes().forEach((node) => {
      if (!visited.has(node.id)) {
        dfs(node.id, []);
      }
    });

    return cycles;
  }
}
```

```typescript
// 自动检测模块的目标环境

class EnvironmentDetector {
  // 检测模块环境
  detectEnvironment(node: ModuleNode): Set<"client" | "server" | "edge"> {
    const environments = new Set<"client" | "server" | "edge">();

    // 1. 检查指令
    if (this.hasUseClientDirective(node.source)) {
      environments.add("client");
    }

    if (this.hasUseServerDirective(node.source)) {
      environments.add("server");
    }

    // 2. 检查 Edge Runtime 配置
    if (this.hasEdgeRuntimeConfig(node)) {
      environments.add("edge");
    }

    // 3. 检查文件位置
    if (node.path.includes("/app/api/")) {
      environments.add("server");
    }

    // 4. 检查导入的API
    if (this.usesBrowserAPIs(node.source)) {
      environments.add("client");
    }

    if (this.usesNodeAPIs(node.source)) {
      environments.add("server");
    }

    // 5. 默认环境
    if (environments.size === 0) {
      // Next.js 默认为服务端组件
      environments.add("server");
    }

    return environments;
  }

  private hasUseClientDirective(source: string): boolean {
    return /['"]use client['"]/.test(source);
  }

  private hasUseServerDirective(source: string): boolean {
    return /['"]use server['"]/.test(source);
  }

  private hasEdgeRuntimeConfig(node: ModuleNode): boolean {
    return /export\s+const\s+runtime\s*=\s*['"]edge['"]/.test(node.source);
  }

  private usesBrowserAPIs(source: string): boolean {
    const browserAPIs = [
      "window",
      "document",
      "localStorage",
      "sessionStorage",
      "navigator",
      "location",
    ];

    return browserAPIs.some((api) => new RegExp(`\\b${api}\\b`).test(source));
  }

  private usesNodeAPIs(source: string): boolean {
    const nodeAPIs = ["process.env", "fs", "path", "http", "https", "crypto"];

    return nodeAPIs.some((api) => source.includes(api));
  }
}
```

```typescript
// 环境标记的传播规则

class EnvironmentPropagation {
  // 传播环境标记
  propagateEnvironments(graph: UnifiedDependencyGraph) {
    let changed = true;

    // 迭代直到稳定
    while (changed) {
      changed = false;

      graph.getAllNodes().forEach((node) => {
        const oldEnvs = new Set(node.environments);

        // 规则1: 客户端组件的依赖也是客户端
        if (node.environments.has("client")) {
          node.dependencies.forEach((depId) => {
            const dep = graph.getNode(depId);
            if (dep && !dep.environments.has("client")) {
              dep.environments.add("client");
              changed = true;
            }
          });
        }

        // 规则2: 服务端组件可以导入客户端组件
        // 但客户端组件不能导入服务端组件
        if (node.environments.has("client")) {
          node.dependencies.forEach((depId) => {
            const dep = graph.getNode(depId);
            if (
              dep &&
              dep.environments.has("server") &&
              !dep.environments.has("client")
            ) {
              // 错误: 客户端组件导入服务端组件
              throw new Error(
                `客户端组件 ${node.path} 不能导入服务端组件 ${dep.path}`
              );
            }
          });
        }

        // 规则3: Edge Runtime 限制
        if (node.environments.has("edge")) {
          // 检查是否使用了不支持的 API
          this.validateEdgeRuntime(node);
        }
      });
    }
  }

  private validateEdgeRuntime(node: ModuleNode) {
    // Edge Runtime 不支持的 API
    const unsupportedAPIs = ["fs", "child_process", "dns", "net", "tls"];

    unsupportedAPIs.forEach((api) => {
      if (node.source.includes(api)) {
        throw new Error(
          `Edge Runtime 不支持 ${api} API (在 ${node.path} 中使用)`
        );
      }
    });
  }
}
```

```typescript
// 为不同环境生成独立的 bundle

class EnvironmentBundler {
  // 为特定环境生成 bundle
  async generateBundleForEnvironment(
    graph: UnifiedDependencyGraph,
    environment: 'client' | 'server' | 'edge',
    entryPoint: string
  ): Promise<Bundle> {
    console.log(`生成 ${environment} bundle...`);

    // 1. 过滤节点
    const relevantNodes = this.filterNodesForEnvironment(graph, environment);

    // 2. 拓扑排序
    const sortedNodes = this.topologicalSort(relevantNodes);

    // 3. 代码分割
    const chunks = this.splitIntoChunks(sortedNodes, environment);

    // 4. 生成代码
    const bundle = await this.generateCode(chunks, environment);

    console.log(`${environment} bundle 生成完成: ${bundle.size} bytes`);

    return bundle;
  }

  private filterNodesForEnvironment(
    graph: UnifiedDependencyGraph,
    environment: 'client' | 'server' | 'edge'
  ): ModuleNode[] {
    return graph.getAllNodes().filter(node =>
      node.environments.has(environment)
    );
  }

  private splitIntoChunks(
    nodes: ModuleNode[],
    environment: 'client' | 'server' | 'edge'
  ): Chunk[] {
    const chunks: Chunk[] = [];

    // 1. 主 chunk(入口点)
    const mainChunk = this.createMainChunk(nodes, environment);
    chunks.push(mainChunk);

    // 2. 第三方库 chunk
    const vendorChunk = this.createVendorChunk(nodes);
    if (vendorChunk.modules.length > 0) {
      chunks.push(vendorChunk);
    }

    // 3. 共享 chunk
    const sharedChunks = this.createSharedChunks(nodes);
    chunks.push(...sharedChunks);

    // 4. 动态导入 chunk
    const dynamicChunks = this.createDynamicChunks(nodes);
    chunks.push(...dynamicChunks);

    return chunks;
  }

  private createMainChunk(
    nodes: ModuleNode[],
    environment: 'client' | 'server' | 'edge'
  ): Chunk {
    return {
      id: `main-${environment}`,
      type: 'main',
      modules: nodes.filter(node =>
        !node.path.includes('node_modules') &&
        !this.isDynamicImport(node)
      ),
      environment,
    };
  }

  private createVendorChunk(nodes: ModuleNode[]): Chunk {
    return {
      id: 'vendor',
      type: 'vendor',
      modules: nodes.filter(node =>
        node.path.includes('node_modules')
      ),
      environment: 'client',
    };
  }

  private createSharedChunks(nodes: ModuleNode[]): Chunk[] {
    // 找到被多个 chunk 共享的模块
    const sharedModules = this.findSharedModules(nodes);

    if (sharedModules.length === 0) {
      return [];
    }

    return [{
      id: 'shared',
      type: 'shared',
      modules: sharedModules,
      environment: 'client',
    }];
  }

  private createDynamicChunks(nodes: ModuleNode[]): Chunk[] {
    const chunks: Chunk[] = [];
    const dynamicImports = this.findDynamicImports(nodes);

    dynamicImports.forEach((importPath, index) => {
      const modules = this.getModulesForDynamicImport(nodes, importPath);

      chunks.push({
        id: `dynamic-${index}`,
        type: 'dynamic',
        modules,
        environment: 'client',
      });
    });

    return chunks;
  }

  private findSharedModules(nodes: ModuleNode[]): ModuleNode[] {
    const usage = new Map<string, number>();

    // 统计每个模块的使用次数
    nodes.forEach(node => {
      node.dependencies.forEach(depId => {
        usage.set(depId, (usage.get(depId) || 0) + 1);
      });
    });

    // 被多次使用的模块
    return nodes.filter(node =>
      (usage.get(node.id) || 0) > 1
    );
  }

  private findDynamicImports(nodes: ModuleNode[]): string[] {
    const dynamicImports: string[] = [];

    nodes.forEach(node => {
      const regex = /import\(['"]([^'"]+)['"]\)/g;
      let match;

      while ((match = regex.exec(node.source)) !== null) {
        dynamicImports.push(match[1]);
      }
    });

    return dynamicImports;
  }

  private isDynamicImport(node: ModuleNode): boolean {
    // 检查此模块是否通过动态导入引入
    return node.dependents.size > 0 &&
      Array.from(node.dependents).some(depId => {
        const dep = /* get node */;
        return dep?.source.includes(`import('${node.path}')`);
      });
  }

  private getModulesForDynamicImport(
    nodes: ModuleNode[],
    importPath: string
  ): ModuleNode[] {
    // 获取动态导入模块及其所有依赖
    const modules: ModuleNode[] = [];
    const visited = new Set<string>();

    const collect = (path: string) => {
      if (visited.has(path)) return;
      visited.add(path);

      const node = nodes.find(n => n.path === path);
      if (node) {
        modules.push(node);
        node.dependencies.forEach(depId => collect(depId));
      }
    };

    collect(importPath);
    return modules;
  }
}
```

```typescript
// 合并小模块以减少请求数量

class ModuleMerger {
  // 合并小模块
  mergeSmallModules(
    graph: UnifiedDependencyGraph,
    threshold: number = 1024 // 1KB
  ): UnifiedDependencyGraph {
    const merged = new UnifiedDependencyGraph();
    const processed = new Set<string>();

    graph.getAllNodes().forEach((node) => {
      if (processed.has(node.id)) return;

      if (node.metadata.size < threshold) {
        // 尝试与其他小模块合并
        const candidates = this.findMergeCandidates(graph, node, threshold);

        if (candidates.length > 0) {
          const mergedNode = this.mergeNodes([node, ...candidates]);
          merged.addNode(mergedNode);

          [node, ...candidates].forEach((n) => processed.add(n.id));
          return;
        }
      }

      // 不合并,直接添加
      merged.addNode(node);
      processed.add(node.id);
    });

    return merged;
  }

  private findMergeCandidates(
    graph: UnifiedDependencyGraph,
    node: ModuleNode,
    threshold: number
  ): ModuleNode[] {
    const candidates: ModuleNode[] = [];
    let totalSize = node.metadata.size;

    // 找到相关的小模块
    node.dependencies.forEach((depId) => {
      const dep = graph.getNode(depId);

      if (
        dep &&
        dep.metadata.size < threshold &&
        totalSize + dep.metadata.size < threshold * 5
      ) {
        candidates.push(dep);
        totalSize += dep.metadata.size;
      }
    });

    return candidates;
  }

  private mergeNodes(nodes: ModuleNode[]): ModuleNode {
    const mergedSource = nodes.map((n) => n.source).join("\n");

    return {
      id: `merged-${nodes.map((n) => n.id).join("-")}`,
      path: `merged-${nodes.length}-modules`,
      type: "javascript",
      source: mergedSource,
      dependencies: new Set(nodes.flatMap((n) => Array.from(n.dependencies))),
      dependents: new Set(nodes.flatMap((n) => Array.from(n.dependents))),
      environments: new Set(nodes.flatMap((n) => Array.from(n.environments))),
      metadata: {
        hash: computeHash(mergedSource),
        size: mergedSource.length,
        lastModified: Date.now(),
      },
      cacheKey: computeHash(mergedSource),
      cached: false,
    };
  }
}
```

```typescript
// 缓存整个依赖图

class GraphCache {
  private cache = new Map<string, CachedGraph>();

  // 缓存图
  async cacheGraph(graph: UnifiedDependencyGraph, key: string): Promise<void> {
    const serialized = this.serializeGraph(graph);

    await fs.writeFile(
      `.next/cache/graph-${key}.json`,
      JSON.stringify(serialized)
    );

    this.cache.set(key, {
      graph,
      timestamp: Date.now(),
    });
  }

  // 恢复图
  async restoreGraph(key: string): Promise<UnifiedDependencyGraph | null> {
    // 1. 检查内存缓存
    const cached = this.cache.get(key);
    if (cached && this.isValid(cached)) {
      return cached.graph;
    }

    // 2. 检查文件系统缓存
    try {
      const data = await fs.readFile(`.next/cache/graph-${key}.json`, "utf-8");
      const serialized = JSON.parse(data);

      if (this.isValidSerialized(serialized)) {
        const graph = this.deserializeGraph(serialized);
        return graph;
      }
    } catch (err) {
      // 缓存不存在或损坏
    }

    return null;
  }

  private serializeGraph(graph: UnifiedDependencyGraph): SerializedGraph {
    return {
      nodes: Array.from(graph.getAllNodes()).map((node) => ({
        ...node,
        dependencies: Array.from(node.dependencies),
        dependents: Array.from(node.dependents),
        environments: Array.from(node.environments),
      })),
      version: "1.0",
      timestamp: Date.now(),
    };
  }

  private deserializeGraph(data: SerializedGraph): UnifiedDependencyGraph {
    const graph = new UnifiedDependencyGraph();

    data.nodes.forEach((nodeData) => {
      const node: ModuleNode = {
        ...nodeData,
        dependencies: new Set(nodeData.dependencies),
        dependents: new Set(nodeData.dependents),
        environments: new Set(nodeData.environments),
      };

      graph.addNode(node);
    });

    return graph;
  }

  private isValid(cached: CachedGraph): boolean {
    // 检查缓存是否过期
    const age = Date.now() - cached.timestamp;
    const maxAge = 60 * 60 * 1000; // 1小时

    return age < maxAge;
  }

  private isValidSerialized(data: SerializedGraph): boolean {
    return data.version === "1.0" && Array.isArray(data.nodes);
  }
}
```

**A**: 主要优势包括:

1. **性能提升**: 共享模块只处理一次,避免重复
2. **内存优化**: 单一图结构减少内存占用
3. **一致性**: 所有环境使用相同的模块版本
4. **简化架构**: 单一遍历生成所有输出

### Q2: 如何处理环境特定的代码?

**A**: Turbopack 使用多种机制:

3. **配置**: runtime = 'edge'
4. **环境传播**: 自动传播环境标记

1. **单一数据结构**: 所有环境共享一个依赖图

5. **优化策略**: Tree Shaking、模块合并、缓存

### 架构优势

- **性能**: 避免重复处理,提高构建速度
- **一致性**: 确保所有环境使用相同的模块
- **简化**: 更简单的架构和逻辑
- **可扩展**: 支持大型复杂应用

掌握统一图概念是理解和使用 Turbopack 的关键。
